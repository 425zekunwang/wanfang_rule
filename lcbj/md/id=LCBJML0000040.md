# 肯尼迪病的治疗  
迄今为止，KD 尚缺乏特异性治疗方法。这是因为KD 患者数量很少，缺乏特异性生物标志物，政府和医药公司投入不足，因此难以开展大规模临床研究。但另一方面，与其他神经变性病特别是运动神经元病相比，KD 病因和发病机制相对明确，这为未来寻找有效的治疗手段提供了可能。而且，KD 进展缓慢，自然病程较长，这是不幸中的万幸。  
# 33. 肯尼迪病患者需保证营养均衡，避免重体力活动以及进行心理辅导  
KD 的一般治疗包括保证营养均衡，避免重体力活动以及心理辅导。  
饮食要注意糖类、蛋白质和脂肪的配比，增加维生素的摄入。通常来说，KD 患者没有特殊的饮食禁忌，但不应该饮酒，因为乙醇对神经系统具有一定损害。基础研究表明，姜黄素可改善KD 转基因鼠的临床症状。姜黄与生姜为同属植物，因此，适当服用生姜没有坏处，但不宜大量服用，因为生姜本身对胃肠道的刺激性较大。  
现有证据显示，无论是有氧运动还是无氧锻炼，都对KD 病情无明显缓解。KD 患者应避免重体力劳动及大运动量训练。KD患者运动神经元和肌肉存在原发损伤，过度运动有可能加重这种损伤，但也应避免长期卧床休息。KD 患者应在疾病不同阶段进行不同类型活动，如早期患者可骑车、散步或游泳。如果病程处于极早期，症状很轻微，可慢跑或打太极拳。晚期坐轮椅或卧床的患者应在床上进行肢体活动，避免废用性肌萎缩，保持关节活动度，防止关节脱位、褥疮等并发症。总之，在疾病不同阶段，患者可从事的运动种类具有一定差别。总体原则是，以活动后不觉得过度劳累为准。  
对于伴随焦虑抑郁状态的患者，应进行心理疏导。良好的家庭环境对于缓解患者不良情绪具有重要作用。必要时，患者应服用抗抑郁焦虑药物，并到精神专科辅以心理治疗。需要注意的是，苯二氮类药物如地西泮、劳拉西泮等，有可能会加重患者的无力症状，应避免使用。  
KD 患者往往对气温比较敏感，冬季时无力感加重。此时，应注意保暖，有条件的患者可经常泡热水澡或泡脚。  
#  抑制患者体内雄激素可以控制或延缓病情进展  
针对KD 发病机制的研究显示，突变AR 蛋白的毒性是其配体，也就是雄激素所依赖的。携带致病基因的女性，甚至是女性纯合子均不发病，可能与她们体内极低的雄激素水平有关。基础研究表明，雄激素会加重KD 雄性转基因鼠的临床症状和病理学改变、缩短其寿命，还可使不会发病的雌性转基因鼠出现肌无力、萎缩等症状。这些均提示，雄激素在发病过程中具有 关键作用。因此，理论上抑制患者体内雄激素就可以控制或延缓病情进展。  
依据此理论，对KD 转基因鼠给予药物或手术去势治疗后其肌无力症状可获得好转，体重增加，寿命恢复至正常，运动神经元内的核包涵体显著减少，达到临床治愈的效果。这一疗效是惊人的，人们似乎看到了控制甚至治愈KD 的希望（图11）。  
由于手术去势存在伦理方面的问题，药物去势治疗成为临床研究热点。醋酸亮丙瑞林（lerorelin acetate）是一种促黄体激素释放激素衍生物，可抑制垂体释放促性腺激素，抑制睾丸释放睾酮，达到药物去势的目的。其微球制剂为缓释剂型，常用剂量为$3.75\mathrm{m}/28\mathrm{d}$，皮下注射，可长期维持血中极低的睾酮水平。作为前列腺癌治疗药物，醋酸亮丙瑞林已上市多年，长期临床实践表明，其安全性和耐受性良好，主要不良反应为性能力的丧失。在使用早期，会出现人为更年期状态，包括烦躁、心慌、出汗、乏力、耳鸣等，但这些不适通常是一过性的，多数患者可以耐受。  
2009 年，Banno 等发表了第一个醋酸亮丙瑞林治疗KD 的Ⅱ期临床研究。研究采用随机双盲安慰剂对照设计：50 例患者被随机分为两组，分别接受醋酸亮丙瑞林皮下注射或安慰剂治疗，  
疗程共48 周。在治疗结束并揭盲后，不管治疗组还是对照组，都可选择继续使用醋酸亮丙瑞林96 周。研究结果显示，与对照组相比，治疗组患者阴囊皮肤核内包涵体显著减少，患者环咽 肌开放时间延长。持续接受醋酸亮丙瑞林治疗144 周的患者，其功能评分及吞咽指标均好于对照组。此研究表明，使用醋酸亮丙瑞林可通过抑制KD 的病理过程来延缓患者运动功能恶化（图12）。  
治疗组与对照组相比：A 血清睾酮水平显著降低；B 两组ALSFRS-R 评分均较基线下降，比较无统计学差异；C ALSFRS-R 量表球部评分均较基线下降，比较无统计学差异； D 环咽肌开放时间，KD 组较基线好转，对照组恶化，比较有统计学差异；E 血清CK 水平，KD 组较基线下降，对照 组升高，比较无统计学差异；F 睾丸皮肤活检，KD 组核内包涵体减少，对照组无变化，比较有统计学差异  
在此基础上，2010 年， Katsuno 等发表了JASMITT 研究。这是首个醋酸亮丙瑞林治疗KD 的大样本多中心随机双盲安慰剂对照研究，共纳入199 例患者，随机分为治疗组和对照组。治疗组接受醋酸亮丙瑞林皮下注射$11.25\mathrm{m}\mathrm{g}/12$ 周，总疗程为48 周，首要评价指标是吞咽钡餐时的钡残留量，残留越少，代表吞咽功能越好。结果显示，治疗结束与开始时相比，治疗组患者钡残留量减少$5.1\%$，而对照组增加$0.2\%$，但两组间无统计学差异。亚组分析表明，病程小于10 年的患者，钡残留量的减少与对照组相比有显著差异。这一结果虽然从整体上未支持醋酸亮丙瑞林的疗效，但似乎说明越早使用醋酸亮丙瑞林，患者受益越多。  
2013 年，Yamamoto 等的一项小样本前瞻性开放式研究显示，长期使用醋酸亮丙瑞林（平均疗程3.5 年）并不能使患者下肢近端肌力得到改善。但其中基线时力量较好或病程$<6$ 年的患者，其肌力下降幅度较小。  
2013 年，我们也发表了醋酸亮丙瑞林治疗KD 的开放式研究。治疗组包括12 例KD 患者，最终完成研究并纳入统计的共10 例患者，在基础治疗情况下，接受醋酸亮丙瑞林$3.75\mathrm{mg}/$ 月皮下注射，共12 个月；对照组包括20 例KD 患者，仅接受基础治疗。比较治疗前后患者肌力、Appel 评分、运动单位估数、血清睾酮、 CK 、糖化血红蛋白及三酰甘油水平的变化。结果显 示，治疗组在观察期结束时血清睾酮和CK 水平较治疗前降低，其他各指标与治疗前相比无统计学差异。对照组在观察期结束时MUNE 降低、CK 水平增高，其他各指标与治疗前相比无统计学差异（表3）。在治疗结束时，除治疗组的血清睾酮水平较对照组降低外，其余各指标两组间无统计学差异（表4）。  
现有数据表明，醋酸亮丙瑞林在动物实验中对KD 有非常好的治疗效果，但人体研究似乎不如动物研究效果显著。即使如此，仍有证据显示其对于早期患者可延缓病情进展。我们认为，动物试验与人体研究差异的原因可能在于：$\textcircled{1}$人体内环境和动物内环境有所区别；$\textcircled{2}$动物研究和人体研究的绝对周期一般都是48 周，但相对周期人体要短得多；$\textcircled{3}$人体代偿能力很强，当出现无力等临床症状时，已有相当数量的运动神经元变性凋亡，这些神经元无法再生，而药物只能保护尚未凋亡的神经元，这也是为何用药越早效果越好的原因；$\textcircled{4}$JASMITT 研究的评价指标过于单一，只用钡餐试验评价患者吞咽功能，而KD 患者早期主要是肢体近端功能受累，晚期才影响吞咽，因此从实验设计上很难完全反映醋酸亮丙瑞林的整体疗效。总之，虽然结果不一，但迄今为止，醋酸亮丙瑞林仍是临床证据最多的治疗KD 的药物。未来，仍需要更大规模的、评价指标更全面的随机双盲对照研究来评价其疗效。  
目前，临床使用醋酸亮丙瑞林仍存在一定顾虑，这不仅仅是因为其疗效尚需进一步确定，还因为要考虑到患者经济承受能力以及对生活质量和生育的要求。好在醋酸亮丙瑞林对雄激素的抑制作用是一过性的，停药后体内激素水平可很快恢复至正常。  
睾酮虽然是男性生殖腺分泌的雄性激素，但并不是AR 的直接配体。AR 的直接配体是DHT。在$5\,\alpha$- 还原酶的作用下，睾酮  
转化为DHT。度他雄胺（dutasteride）是一种$5~\alpha$- 还原酶抑制药，可抑制睾酮转变为DHT。使用度他雄胺后，体内DHT 的浓度可降低$90\%$，临床用于治疗前列腺增生和男性脱发，主要不良反应是性欲减退，但与醋酸亮丙瑞林相比，不良反应较小。  
2011 年，Fernández-Rhodes 等发表了度他雄胺治疗KD 的单中心随机双盲安慰剂对照研究。此项研究共纳入50 例患者，最终完成44 例，随机分为治疗组和对照组，治疗组每天口服度他雄胺$0.5\mathrm{ms}$，治疗周期共24 个月。基本评价指标是定量肌肉评估（quantitative muscle assessment ，QMA）。结果显示，治疗结束时，对照组患者与治疗前相比QMA 下降$4.5\%$，而治疗组则增加$1.3\%$，但两组间无统计学差异。次级评价指标，包括CK、肌力、运动神经传导、日常生活能力等在内，治疗前后的改变在两组间均无统计学差异。本研究未显示度他雄胺对KD 具有治疗作用，其原因可能在于：$\textcircled{1}$样本量过小；$\textcircled{2}$治疗周期较短。  
#  应保护肯尼迪病患者线粒体功能  
在KD 发病机制中，线粒体功能障碍是目前研究热点。线粒体对细胞有氧呼吸及能量代谢具有重要作用。多项基础研究表明，KD 转基因细胞线粒体功能严重异常。辅酶Q10、艾地苯醌等药物对KD 转基因细胞具有保护作用，并可改善KD 转基因鼠的症状，延长生存期。由于这些药物价格低廉，不良反应很少，  
因此，虽然缺乏临床研究数据，也可尝试应用于KD 患者。辅酶Q10 的使用剂量为每天$300\mathrm{mg}$ 左右；艾地苯醌的使用剂量为每天$90\mathrm{ms}$ 左右。  
# 36.  克伦特罗对肯尼迪病的治疗效果尚需评估  
克伦特罗（clenbuterol）是一种人工合成的$eta_{2}-$ 肾上腺素受体激动药，曾用于治疗动物和人的阻塞性支气管哮喘。克伦特罗具有较强的促进肌肉合成和降解脂肪作用，长期较大剂量使用克伦特罗可增加蛋白质合成，减少脂肪储存，促进各种动物肌肉生长，增加瘦肉率，因此，也被俗称为“瘦肉精”。克伦特罗的不良反应很大，急性中毒可导致心动过速、肌肉搐颤、腹泻等，严重时可致昏迷和癫痫；慢性中毒可导致心肌损害、生殖 腺异常。  
2013 年，Querin 等发表了一项克伦特罗治疗KD 的小样本开放式研究。研究采用自身对照方式，共纳入16 例患者，给予患者口服克伦特罗$0.04\mathrm{mg/d}$，治疗周期为12 个月。结果显示，克伦特罗可提高患者3 个月和12 个月的6 分钟步行能力约$12\%$，但不能使肌力和ALSFRS 评分获得改善，还导致血中CK 水平增高。这是除醋酸亮丙瑞林之外另一项获得阳性结果的药物，但此研究可能存在如下问题：$\textcircled{1}$开放式设计，样本量小；$\textcircled{2}$理论上，克伦特罗对KD 发病机制并无影响，只是被动增加肌肉合成，因此难以延缓病情进展；$\textcircled{3}$ 步行试验受情绪等主观因素影响较大，相对客观的肌力评分和量表评分则显示治疗前后无差异；$\textcircled{4}$CK 的增高说明肌肉破坏程度可能加重。结合其不良反应较大，我们认为，克伦特罗对KD 的治疗效果尚需要更大规模的随机双盲对照研究进行评估。  
# 37. 有氧锻炼和力量训练对肯尼迪病患者没有明显疗效  
近期有两个研究分别使用有氧锻炼和力量训练对KD 患者进行治疗。有氧锻炼方案是原地脚踏车30 分钟，共12 周。第1周和第2 周每周2 次；第3 周和第4 周每周3 次；其余8 周每周5 次，逐渐增加运动量。主要终点是最大氧摄取（$\mathrm{VO}_{2}\mathrm{max}$）、最大作功能力（Wmax）和日常生活能力评分量表（ADL 评分）的变化；次要终点是肌活检时肌肉形态和柠檬酸合成酶（citrate synthase，CS）活性、肌肉比例、肌电图、肌力测定、肺功能、用力肺活量、用力呼气量等数据的变化。结果显示，除Wmax 增加$18\%$，CS 活性增加$35\%$ 外，其他指标无显著性差异。该研究认为有氧训练对KD 无明显疗效。  
第二个研究对50 例患者进行力量训练。患者随机分为两组：训练组24 例使用各种方法训练肌肉力量；对照组26 例仅进行拉伸练习。主要终点是成人肌肉病评估工具（AMAT 量表）；次要终点包括平衡能力、肌力、蹲起时间、CK、生活质量量表等。在所有主要和次要终点方面，两组患者均未显示出明显差异。  
#  应按常规治疗控制肯尼迪病合并症  
对合并糖尿病的KD 患者，应按常规控制血糖，因为糖尿病本身也会导致神经损伤。  
对合并脂质代谢异常的KD 患者，无证据显示积极控制血脂会导致病情加重或进展加速，因此，应按照常规予以治疗。  
# 39.  对基因确诊但临床未出现症状者可给予线粒 体功能保护药物  
对于基因检测明确携带致病基因但临床尚未出现肌无力的男性患者，无证据显示任何治疗可延缓肌无力出现的时间。但根据发病机制，对于无禁忌证的患者，可尝试给予保护线粒体功能的药物。  
#  对症处理中晚期肯尼迪病并发症  
对于中晚期行动不便患者，应积极预防可能威胁患者生命的各种并发症，如吸入性肺炎、褥疮等。对于肺功能明显减退患者，应使用无创呼吸机；对进食困难患者，应适时进行胃造瘘手术以保障营养供应，同时避免误吸；对卧床患者，应勤翻身并使 用防褥疮气垫以避免发生褥疮。这些处理原则同运动神经元病的处理原则相同。  
一旦出现严重并发症如肺部感染和呼吸衰竭，应使用敏感的抗生素，同时促进排痰，必要时使用有创呼吸机。  
#  避免服用含雄激素的中药饮片  
虽然没有基础或临床研究证实中药对KD 治疗有效，但KD属于中医的痿证。中医认为，痿证以虚为本，或虚实错杂，临床虽以肺热津伤、湿热浸淫、脾胃虚弱、肝肾亏损、瘀阻络脉等证 型常见，但各种证型之间常相互关联，治疗时要结合标本虚实转变。扶正主要是调养脏腑，补益气血，调和阴阳；祛邪重在清利湿热与解温热毒邪。在治疗过程中还要兼顾运行气血，以通利经脉，濡养筋脉。  
需要特别强调的是，中药中某些动物成分的饮片，特别是动物的鞭类，可能含有雄激素，应避免使用。部分患者存在一种误解：性生活可以消耗雄激素，从而缓解病情进展。实际上，性生活不但不能消耗雄激素，反而会刺激睾丸分泌更多的睾酮。而且痿证的病因之一就是房劳过度。因此，应适当节制性生活。
