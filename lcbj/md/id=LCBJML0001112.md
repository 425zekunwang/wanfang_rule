# 子宫颈癌淋巴结的处理  
# 26.  不应常规对子宫颈癌患者做盆腔淋巴结清扫术  
多年来，对子宫颈癌手术中的盆腔淋巴结清扫术存在争议。  
（1）不应常规做盆腔淋巴清扫术的理由有：$\textcircled{1}$阴道广泛子宫切除术术者认为早期子宫颈癌不做盆腔淋巴结清扫术，治愈率与经腹广泛子宫切除术及盆腔淋巴结清扫术者相同。$\textcircled{2}$认为盆腔淋巴清扫术是不彻底的手术，要清除所有盆腔淋巴结在技术上是不可能的。$\textcircled{3}$在盆腔淋巴结癌侵犯病例中，也有部分病例腹主动脉旁淋巴结已有癌转移，而高位腹主动脉旁淋巴结是不可能完全被清除的。$@\mathrm{~}80\%\sim90\%$早中期子宫颈癌患者无淋巴转移，不需清扫盆腔淋巴结，清扫正常的淋巴系统对患者有害无益。  
（2）赞成需要清扫淋巴的理由：$\textcircled{1}$盆腔淋巴清扫术有助于评估预后，并确定患者术后是否需要加辅助治疗。$\textcircled{2}$手术时如发现髂总淋巴结有转移，应进一步做腹主动脉旁淋巴结清扫。多数学者报道，子宫颈癌Ⅱb 期、Ⅲb 期等淋巴转移预后差，腹主动脉阳性预后极差，这些结果均是将淋巴清扫后又补充放化疗的结 果，说明淋巴阳性再放化疗并不能改善预后，甚至可能造成更坏的结果。  
长期以来，由于没有更好的诊断治疗前淋巴是否受累的方法，子宫颈癌的淋巴系统处理策略处于一种不得不盲目过度治疗的尴尬之中，医生们对手术清除的多数淋巴结为阴性也就无动于衷了。殊不知，这些被清扫的正常淋巴结对患者有多大的保护作用，而且常常还可能因为这种不需要的清扫手术带来诸多的后遗症。其实，做与不做淋巴清扫手术，对患者的治疗和生存并没有多大的影响，所以，笔者认为常规一律清扫淋巴的方法需要改变，建议还要致力寻找更为方便、准确、经济的淋巴转移的诊断方法。另外，很多医生没有深刻了解淋巴系统对免疫功能的重要性。笔者认为，淋巴转移，特别是腹主动脉旁淋巴转移，绝不能被视为只是淋巴受累的局部问题，而是整个机体免疫力下降的重要问题，这时，任何进一步损伤免疫功能的处理，包括扩大手术范围、放化疗，不但不能起到有效的治疗作用，反而加速患者走向死亡。因此，在淋巴系统已经有转移的情况下，必须考虑在不损伤免疫功能的基础上给予治疗，即使不能起到理想的治疗效果，也不要增加患者的痛苦和负担。另外，应该考虑仅对淋巴系统施行淋巴化疗的可行性，做到既能治疗淋巴结转移，又能保护 没有受累的淋巴结，从而保持淋巴系统的完整性，保护患者的免疫功能，提高患者的生存率和生活质量。  
# 27. 子宫颈癌Ⅰ期、Ⅱ期淋巴处理多数是过度治疗，Ⅰ期子宫颈癌可不做盆腔淋巴清扫  
对子宫颈癌的治疗不论是手术或放疗，其分为两个部分。一是宫颈局部和附着的骶韧带、主韧带及宫颈旁组织、阴道及阴道旁组织的广泛切除，另一部分是盆腔淋巴结的系统清扫或放射治疗，部分病例甚至要求到高位腹主动脉旁，淋巴对所有病例只要是Ⅰa2 期或Ⅰb1 期以上一律要求常规清扫。  
（1）盆腔淋巴清扫手术范围：双侧髂总淋巴结、髂外和髂内淋巴结、深腹股沟淋巴结、闭孔深组和浅组淋巴结。如髂总淋巴 结可疑，冰冻阳性，再探查腹主动脉旁淋巴结，如腹主动脉旁淋巴结阳性则停止淋巴清扫手术，阴性则行腹主动脉旁淋巴结清扫手术。从肠系膜下动脉平面开始向下，如髂总淋巴结阴性，则行盆腔淋巴清扫手术即可。  
（2）盆腔淋巴结清扫术有两种手术方法：$\textcircled{1}$切开腹壁进入腹腔：打开盆腔腹膜暴露腹膜后区域，然后采用逆行切除方法，即从打开骨盆漏斗韧带，从上向下依次暴露髂总、髂内、髂外血 管和输尿管等，并剥离其周围脂肪及淋巴组织，自外周向内整块切除以上各组淋巴结。$\textcircled{2}$腹膜外盆腔淋巴结清扫术：同样切开腹壁，暴露腹膜，但不切开腹膜，而是将腹直肌筋膜与腹膜分开，然后将腹膜用手掌轻轻向中央推开，在膀胱侧方间隙显露出腹膜外盆腔，找到该侧圆韧带腹膜外部分切断、缝扎，暴露髂血管，用手指将腹膜向内侧分离，然后与经腹腔内盆腔淋巴结清扫手术 同样操作，以清除各组淋巴结。腹膜外盆腔淋巴结清扫手术的优点是手术时未切开腹膜，干扰腹腔内脏器较少、时间较短，手术后恢复快，其缺点是手术视野的暴露不如腹膜内，手术操作不方便。  
（3）对淋巴清扫的不同观点：子宫颈癌的主要扩散途径是宫旁转移（沿阴道和子宫转移），以及腹膜后淋巴结转移。子宫颈癌盆腔淋巴结转移的发生率随国际妇产科联盟分期的增加而上升。宫颈微小浸润癌（根据FIGO 临床分期的定义，Ⅰa1 期浸润深度不超过 $3\mathrm{mm}$ ，宽度不超过 $7\mathrm{mm}$ ； I a2  期浸润深度不超过 $5\mathrm{mm}$，宽度不超过$7\mathrm{mm}$）淋巴结转移率为$0.5\%\sim0.8\%$，其盆腔淋巴结受累的风险约为$3.4\%$。而$^\mathrm{~I~b~}$、Ⅱa 和Ⅱb 期子宫颈癌患者，腹膜后淋巴转移率不同医院报道差异较大，因为即使进行了系统性淋巴结切除也不可能对切下来的每个淋巴结都进行病理检查，故存在一定的漏诊率。Kastner、Mitra 等报道盆腔淋巴没有侵犯的患者，5 年存活率高达$90\%$以上。Meigs 报道了子宫颈癌术后盆腔淋巴有侵犯的患者，其5 年存活率为$42\%$。  
对于子宫颈癌不分早晚，治疗中一律需要进行盆腔淋巴清扫或腹主动脉旁淋巴清扫，笔者提出有必要这样做吗？尽管多年来笔者也按照FIGO 的规范要求这样做，但笔者提出的这一问题一直没有得到很好的答复。因为对早期子宫颈癌淋巴系统受累的病例是非常少数的，多数的临床报道早期子宫颈癌的淋巴受累Ⅰ期是$8\%\sim11\%$，$\Pi$a 期是$10\%$，Ⅱb 期是$30\%$，那意味着$70\%\sim90\%$ 以上的正常淋巴系统被清扫切除，这对患者是过度治疗，对患者是绝对的有弊而无利。因为淋巴系统是盆腔第一线的免疫保护系统，如果少量的癌细胞侵犯淋巴系统，可以被阻挡甚至可局限在淋巴结内不扩散到远处。如果全部正常淋巴清扫 后，子宫颈癌病灶会在微血管、淋巴管内残留少量的癌细胞，在患者免疫功能下降的情况下，这些癌细胞将会毫无阻挡地扩散转移，在一些Ⅰb 早期病例手术后发生复发转移应该与此有关。因此，笔者认为不应该对早期子宫颈癌患者一律清扫盆腔淋巴，但目前的困难是在早期子宫颈癌中确诊有无淋巴转移的方法还不够准确，不能在治疗前了解盆腔淋巴是否受累，因此采取的是盲目的全部清扫淋巴，这也是不得已且不精准的治疗方法。笔者建议对Ⅰb1 期可以不常规做盆腔淋巴清扫术，可以在手术中探查淋巴结的情况再准确处理，也可以采用术前 MRI 或 PET-CT 影像学了解淋巴情况再决定淋巴的处理，或采用腹膜后置管淋巴化疗的方法，这样对淋巴结内即使有癌细胞转移也可起到治疗作用，对没有癌细胞转移的淋巴，不但保留了完整的淋巴系统，而且淋巴化疗可以刺激淋巴组织增生，提高机体免疫力。希望这是一种理想的淋巴处理方法。  
#  术后淋巴结阳性不再需要放射治疗  
在早期子宫颈癌术后淋巴清扫中发现有淋巴结阳性，常规还要进行盆腔或腹主动脉淋巴结再次放射治疗，笔者多次咨询国内外同行、学者，这是为什么？受累的阳性淋巴结不是已被清扫了吗？为什么还要在这些手术后的区域照射？照射的目的是什么？回答是认为淋巴没有被清扫彻底，为什么不清扫彻底？因为清扫不容易彻底，顶多去掉$70\%$ 的淋巴，那为什么还要进行淋巴清扫手术，而不直接放疗或化疗？没有回答，因为手术后的盆腔淋巴结，还有盆腔的血管、淋巴管已被去掉了周围保护的脂肪结缔组织，直接暴露在放射野，极易受到放射损伤，造成纤维化粘连，从而容易发生淋巴水肿，更主要是这种术后放射治疗并不能提高生存率，有弊无利，笔者认为术后发现淋巴结阳性不需要再做放射治疗。  
淋巴清扫的不良反应不仅在手术中，对于没有侵犯的淋巴结自然容易清扫，但是是不必要的，真正有转移的淋巴侵犯经常是形成团块或者包绕大血管特别是与静脉粘连侵犯，致使淋巴结切除非常困难。破损的淋巴结造成残留或者很容易损伤静脉发生大出血的合并症，有时甚至造成严重后果。另外，如果淋巴清扫术后淋巴管断端封闭不全或开放，可造成术后淋巴囊肿、淋巴漏和淋巴象皮腿，很难治疗，严重影响患者治疗后的生活质量。  
# 29. 淋巴清扫后放疗的主要后遗症为淋巴水肿和淋巴漏  
淋巴清扫后放疗的后遗症有淋巴囊肿、淋巴水肿、淋巴漏，而引起淋巴漏的主要原因是因为在盆腔淋巴结切除和腹主动脉旁淋巴结切除的时候没有很好地结扎、封闭淋巴管，淋巴管持续的开放导致大量的淋巴引流，最终形成淋巴囊肿或淋巴漏。  
（1）淋巴囊肿  
淋巴囊肿是进行妇科肿瘤手术尤其是子宫颈癌手术后最常见的合并症，应该说多数小的淋巴囊肿经过一段时间是可以被吸收的，而大的淋巴囊肿则会存在相当长的一段时间，有时淋巴囊肿甚至可以随着活动的增加而增大。术后提早下床活动也将会促使淋巴液蓄积在局部，导致淋巴囊肿的形成。术后一周时是腹膜愈合的关键时间，而此时淋巴管闭合的还不够坚固，所以增加活动常常会导致淋巴回流的增加、局部淋巴液的蓄积，从而造成淋巴囊肿。  
淋巴囊肿的形成主要是淋巴液在局部的蓄积，而淋巴液的主要来源有两个，一个是由远心端向近心端的淋巴回流，另外一个就是由近心端向远心端的淋巴反流或回漏，因为淋巴管的瓣膜并不是像血管那样完善，因此淋巴的反流也是比较常见的。淋巴液的去路也有很多途径，第一个途径是经过淋巴管回流，第二个途径则是流入腹腔，经腹膜吸收。而淋巴囊肿的形成则是来源没有被阻断，而去路又没有很好地形成，这样就导致了淋巴液在局部的蓄积，从而形成淋巴囊肿。  
淋巴囊肿的处理：$\textcircled{1}$对于小的、没有症状的不需要处理，只需要观察，定期B 超检查其大小即可。$\textcircled{2}$对于大的、有症状的，特别是对重要器官构成影响的淋巴囊肿，可采用B 超引导下穿刺，抽出囊肿内的淋巴液，同时局部注射硬化剂，随后局部压迫 并且减少活动，穿刺之后局部加压、减少下肢的活动也是治疗淋巴囊肿的重要措施。  
预防淋巴囊肿的基本原则是阻断来源和疏通去路。阻断淋巴液的来源是指在手术当中将远心端的淋巴管彻底闭合，可以采用结扎的方式，也可以采用电凝闭合，这样就可以有效地阻断淋巴液的来源；另一个有效阻断淋巴液来源的方式就是将淋巴管的近心端也进行有效的闭合或者结扎，这样可以阻断由于淋巴液的反流导致局部淋巴液的蓄积。疏通淋巴液的去路也是减少淋巴囊肿 的措施之一，正常情况下淋巴液是通过淋巴管经过胸导管回流入大循环，但由于淋巴结切除，使得淋巴回流的通路被彻底阻断，而淋巴液如果能够流入腹腔是可以经过腹膜吸收的。因此，关闭后腹膜是不利于淋巴液的引流和吸收的，因为脏腹膜可以吸收淋巴液，因此不缝合后腹膜，这样给淋巴液进入腹腔提供了方便，也是增加淋巴液去路的主要措施之一。此外，如果能够有效地防止粘连或减少局部的粘连， 收，同时有利于淋巴囊肿的预防。当然，术后的引流极其重要，它不仅可以将盆腔渗出的血液引出，也可以将局部组织渗出液和淋巴液引流出来，所以，是否放置引流管会直接关系到术后淋巴囊肿的形成。  
# 2 ）淋巴漏  
淋巴漏的形成主要是切除盆腔淋巴结，特别是腹主动脉旁淋巴结时没有封闭淋巴管造成大量淋巴液流出，形成淋巴漏。因此，较大的淋巴管的近心端和远心端都必须结扎或封闭，轻度淋巴漏可通过加强引流、补充营养、减少活动等自行愈合。但重度的淋巴漏形成后，很难自行愈合，长期的淋巴液流出实际上是蛋白和脂肪的流出，这会导致患者发生严重的营养不良。本院早年一例患者术后出现严重淋巴漏，达2 个多月之久，最终该患者衰竭而亡，所以对不能自行愈合的淋巴漏，可在经腹腔镜下仔细找到淋巴管漏口并给予缝扎，这样可以彻底治疗。  
（3）淋巴水肿  
淋巴水肿是子宫颈癌术后常见的合并症，术后再辅以放疗，就更增加了患者发生淋巴水肿的机会。淋巴水肿除了大量组织间液的外渗之外，更重要的是伴有大量高渗的蛋白质渗出，使得局部组织间的胶体渗透压明显增高，让渗出的组织间液很难吸收，因此淋巴水肿很难有治愈的可能。  
子宫颈癌术后的淋巴水肿主要表现为腹部的水肿、下肢的水肿，甚至可以出现凹陷性水肿。随着水肿的逐渐加重，患者水肿部位的皮肤也逐渐变粗糙，形成象皮腿，这是淋巴水肿导致的最严重结果。但如果不进行淋巴结切除，就不会发生淋巴水肿，因此，淋巴结切除是淋巴水肿的必要条件。再有就是放疗之后会增加淋巴水肿的发生率，特别是手术后再放疗更容易发生淋巴水 肿，目前没有很好的办法来治疗淋巴水肿，最好的办法就是预防。患者术后可以进行适当的锻炼，但是一定要掌握度，而且锻炼时最好穿上弹力袜，不要采取如暴走、登山、跳广场舞等锻炼 方式，而散步、瑜伽、太极、游泳、温泉都是比较适合的。  
以下分享一个笔者最近治疗的一例子宫颈癌术后淋巴水肿4期患者的治疗病例。  
黎某，女，46 岁，2009 年在当地医院接受子宫颈癌手术以及术后放疗，2010 年出现左下肢淋巴水肿，随后病情逐渐加重，曾在琼海、海口、三亚等地进行中药、针灸等治疗均未果。2017年8 月来到本院，患者自诉下肢肿胀、疼痛、麻木，膝关节无法正常弯曲，肢体运动受碍，肢体外观呈象皮肿、皮肤发紫，触摸肢体张力大、褶皱严重无法打开，经诊断为子宫颈癌术后 $^+$  放疗 出现并发症（左下肢淋巴水肿4 期）。  
住院后给予CDT- 综合消肿疗法，具体操作：第1 天给予局部皮肤护理，第2 天手法$^+$ 设备淋巴引流，第3 天外周淋巴交通支的疏导，然后给予低弹力绷带加压治疗，后期给予腿部功能锻炼，经十多天治疗后，患者水肿明显改善（表 9 ） ，膝关节正常 弯曲、行动自如（图19），当然，这种病例治疗效果较好是少数，关键还是预防淋巴水肿的发生。  
# 30. 前哨淋巴结在子宫颈癌治疗中的作用效果 不明显  
前哨淋巴结（sentinel lymph node，SLN）的概念最早是在1977 年被cabansa 提出的，当时cabansa 在阴茎背侧进行淋巴造影时发现一种特殊的淋巴结，该淋巴结最先接受肿瘤部位的淋巴引流，为发生肿瘤转移的“第一站”淋巴结，所以命名为“前哨”淋巴结，并因此提出术中如能以可靠方法识别SLN，如SLN 活  
检阴性，则可以不做系统的淋巴清扫，减少手术带来的损伤。  
近年来，子宫颈癌SLN 活检在各国先后开展，SLN 的主要识别方法可归纳为以下3 种：$\textcircled{1}$生物活性染料示踪法：以亚甲蓝、专利蓝等生物活性染料为标记物；$\textcircled{2}$放射性核素示踪法：以放射性核素锝-99 为标记物；$\textcircled{3}$生物活性染料- 放射性核素联合示踪法。结果显示，SLN 识别率为$100\%$。腹腔镜下亦可同时运用染料- 核素联合示踪法进行SLN 识别，识别率为$92\%\sim100\%$。在国内外多项研究中，SLN 主要分布在髂内、外及闭孔区，而很少分布在宫旁淋巴结。在肿瘤$\leqslant4\mathrm{cm}$ 时有$73\%$能找到SLN，而在肿瘤$>4\mathrm{cm}$ 时仅$20\%$ 能找到SLN，可见SLN活检适于早期患者，原因是其淋巴结转移灶大，妨碍了淋巴引流。在体内识别SLN 的研究中，假阴性结果占一定比率，假阴性结果可导致对病情错误的评估和不正确的治疗。当前国际上一些专家建议，在子宫颈癌手术时，首先做前哨淋巴结检测，再确定是否清扫淋巴及清扫范围，术中如果发现前哨淋巴阴性则不需做淋巴清扫手术，前哨淋巴阳性而髂总淋巴结阴性则进行盆腔淋 巴清扫手术。笔者认为，前哨淋巴结在子宫颈癌治疗中的作用效果不明显，前哨淋巴结测定的临床操作复杂且不够准确，能确定的前哨淋巴结仅有$76\%$，因为子宫颈癌的前哨淋巴结部位并不固定在哪些部位，而是在多个不同的部位。根据我们的理解，并不是第一站的淋巴结为前哨淋巴结，如果在好几个前哨淋巴结部位都切除后再冰冻切片，这就相当于做了一个淋巴清扫手术，也就没有必要区别哪些是前哨淋巴结，哪些不是了。此外，切下的标本送冰冻切片耗费时间，而且冰冻切片报告的准确性也只有$70\%$左右，并不准确，反而容易误导。多数临床专家认为，即便前哨淋巴结阳性，子宫颈癌广泛切除手术仍然可以进行，所以，前哨淋巴结在子宫颈癌治疗中的作用就不那么重要了，但必要时在手术前做PET-CT 检查可更加准确，也不影响手术的进行。实际上至今多数的子宫颈癌治疗医疗中心并没有采取这一种方法，目前尚未被广泛应用。
