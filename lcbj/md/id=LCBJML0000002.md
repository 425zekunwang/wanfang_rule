# 11.  半椎体畸形是最常见的先天性脊柱侧 凸畸形  
先天性脊柱侧凸畸形是指出生时因椎体发育异常而导 致的脊柱畸形，通常可合并其他系统的发育异常。据报道，此病在新生儿中的发病率约为1/1000。根据影像学表现，先天性脊柱侧凸大致可分为3 类：形成障碍、分节障碍及混合型。半椎体畸形作为形成障碍的一种，是最常见的先天性脊柱侧凸畸形。半椎体畸形通常表现为椎体异常发育 为三角形或者楔形，由于1 个或多个半椎体嵌于正常椎体  
间而导致局部成角畸形从而形成脊柱弯曲，随着病情进展可在头侧及尾侧形成继发性代偿弯。患者从而出现肩部不平，骨盆倾斜，躯干偏移等临床表现，部分患儿的心肺发育也会受到影响。该病的治疗十分棘手，佩戴支具或矫形支架的保守治疗仅对一小部分患者具有控制侧凸发展的作用，而绝大部分患者需手术治疗。先天性脊柱侧凸的诊治目标是早期诊断，筛查相关系统畸形（神经系统、心血管系统、泌尿生殖系统），预防/ 治疗临床上明显的脊柱畸形，同时，尽量保留其他节段脊柱及胸腔的生长潜能。因此，需要根据患者的年龄、脊柱畸形的严重程度、半椎体的位置及形态选择个体化的治疗方案。  
#  半椎体畸形的Burnei-Gavriliu 分型  
半椎体畸形的分类对临床诊疗具有重要意义，但目前尚无针对半椎体畸形的单独分类，其分类可涵括在先天性脊柱侧凸分类中。近期，Alexandra 等对该院历时14 年的先天性脊柱侧凸畸形病例进行回顾分析，结合以往Winter、Imagama 及Kawakami 等的历史分型，于2015 年提出了更为系统的Burnei-Gavriliu 分型。  
根据半椎体的不同位置可将其分为颈胸段半椎体、胸  
段半椎体、胸腰段半椎体、腰段半椎体和腰骶段半椎体。  
（1）单发半椎体畸形可分为完全分节、部分分节及未分节3 类（图1）  
完全分节半椎体： 完全分节的半椎体是最常见的半椎 体畸形，其上、下缘具有完整的软骨终板及椎间盘，有较大的生长潜能。因生长方向异常，通常会导致侧凸快速进展，需手术干预。  
部分分节半椎体： 介于完全分节半椎体和未分节半椎 体之间。与邻近椎体骨性相连的一半相对稳定，而具有椎间盘的另一半仍具备生长潜能。因此，多数病例仍需手术干预。

#  半椎体畸形的影像学诊断最为重要  
半椎体畸形的诊断主要依赖病史采集、体格检查及影像学检查，其中影像学检查最为重要。半椎体畸形可通过X 线、CT、3D-CT 及MRI 等影像学检查确诊。X 线检查简单方便，应用广泛，可初步了解半椎体的位置、类型及个数。同时，可获得其他相关参数，如局部弯、主弯、代偿弯、躯干偏移、骨盆倾斜、脊柱柔韧性、矢状位前凸或 后凸、Risser 征等。但由于X 线片是二维显像，畸形椎体可能相互重叠而导致局部细节显示不清，3D-CT 可以克服此缺陷，局部融合椎体的细节、各个维度上的侧弯角度等信息均可获得，其高清及多维的优势还有助于进行术前手术设计。然而，辐射剂量相对较大限制了它的广泛应用。Humbert 等近年开创了一种应用二维显影重建三维图像的技术（图4），这种技术较X 线有更好的细节显示，虽然分

#  半椎体畸形的早期诊断极为重要  
早期及时的干预对于恢复脊柱的正常生长、防止畸形发展具有重要意义。近年来，半椎体畸形的产前检查获得了越来越多的重视，为优生优育及早期防治先天性脊柱侧 凸畸形提供了新的希望。超声具有的灵活、方便、安全等优势使其在脊柱畸形的产前诊断中得到了广泛应用。目前，可在妊娠$20\sim28$ 周时诊断并发现半椎体畸形（图5）。近年来，三维超声在临床上获得了进一步的应用，其透明成像模式可直观地显示椎体的大小、形态、椎间隙以及脊柱的连续性和生理弯曲，并清晰地表明脊柱与胸廓、颅底、骨盆的空间结构，在诊断脊柱畸形等方面有很大的优势。三维超声还可同时展现出多个截面的情况，以不同角度观  
察需关注的节段，甚至可直观地表现出类似于 CT  及 MRI 的图像效果，在多个及混合半椎体畸形的早期诊断方面有着独特的优势。  
尽管产前超声诊断有着重要的地位，但由于胎方位、羊水过少及骨骼遮挡等限制，MRI 更适合评估椎管内容物，对中枢神经系统及脊髓异常的诊断有着更高的敏感度。虽然会受运动伪影干扰，但MRI 可直接显示脊髓，是对超声检查怀疑脊柱畸形的一种补充，能更好地检测出相关的中枢神经系统畸形。这两种技术的结合使用更有利于产前咨询、产前决策及半椎体畸形治疗的选择。  
# 15. 半椎体畸形选择非手术治疗时需定期复查  
半椎体畸形导致的先天性脊柱侧凸随着患儿年龄的增 长，局部弯及代偿弯也进行性发展，甚至造成脊柱僵硬，从而增加手术难度，降低矫形效果。因此，早期干预具有重要的意义。然而，患儿心肺功能正处于发育过程中，如何在矫形的同时，尽可能地保留脊柱及胸廓的生长能力，是手术医师需要考虑的问题。  
如果半椎体畸形不会导致侧凸快速进展，可尝试非手术治疗，如“嵌顿半椎体”。还有相互代偿的多发半椎体畸  
形，也可暂不手术，通过佩戴支具等保守治疗来控制代偿弯的发展。然而，需定期复查以避免畸形突然加重。也有侧凸未经任何干预自发性好转的病例报道，但这种情况非常少见。  
# 16. 半椎体畸形的早期手术治疗具有较大意义  
对于导致严重侧凸畸形的半椎体畸形往往需手术治疗。手术方法分为两种：矫正畸形的手术（如半椎体切除/骨骺阻滞）及控制侧凸进展的手术（如原位内固定/ 融合术）。。半椎体导致的主弯会随着年龄增长而发展，之前未累及的邻近节段必然也会形成代偿弯，甚至僵硬，发展成为结构弯。因此，早期手术治疗具有较大意义。  
#  原位融合术目前已很少使用  
原位融合术是指不使用内固定器械进行矫形的脊柱融合术，包括后路原位融合术和前后路联合原位融合术。融合术可控制侧弯的进展，曾被认为是治疗先天性脊柱侧凸的经典方法。该术式简单、安全，但术中需要融合整个侧  
凸范围内的双侧椎板，融合节段长，植骨量大，并限制了患儿脊柱的正常生长发育，假关节和曲轴现象的发生率也较高。对于严重的脊柱侧凸或腰骶部的半椎体畸形，后路原位融合往往无法控制侧凸进展和躯干偏移。因此，该术式目前已很少使用。  
# 18. 凸侧骨骺阻滞术通常适用于主弯较小且脊柱具有一定生长潜能的患儿，但无法控制凹侧的生长潜能  
凸侧骨骺阻滞术是指凸侧行骨骺阻滞，凹侧继续保持 一定的生长潜能，从而使脊柱随生长发育进行自我矫形的治疗方法。凸侧骨骺阻滞术通常适用于主弯较小且脊柱具有一定生长潜能的患儿。有学者曾建议，年龄$<\,5$岁、主弯$<40^{\circ}$°的患儿可应用该术式。其不足之处在于它虽然可以通过抑制凸侧生长而阻断侧凸进展，却无法控制凹侧的生长潜能。对于长节段的脊柱侧凸，或凹侧合并有未分化骨桥时，该方法矫形效果有限。此外，患儿还需较长时间的外固定，给患儿带来诸多不便。这些不足限制了凸侧骨骺阻滞术的进一步应用。  
# 19.  半椎体切除内固定融合术适用范围广泛  
根据近10 年的文献复习，显示半椎体切除内固定融合术已逐渐成为目前半椎体畸形手术治疗的主流。该术式的矫形率为$64\%\sim81\%$。研究表明，半椎体切除术的矫正率明显高于原位融合术或内固定融合术，但其相关并发症如感染、神经损伤、内固定失败等也不容忽视。该术式的适应范围更加广泛，对于腰骶段半椎体、对侧骨桥形成及长节段严重畸形的患者，后路半椎体切除短节段融合也获得 了满意的矫正效果。  
# 20.  单纯后路一期半椎体切除短节段内固 定融合术是目前半椎体畸形手术治疗的主流  
目前半椎体畸形手术治疗较为主流的术式为单纯后路一期半椎体切除短节段内固定融合术，该术式固定融合较 短节段即可获得满意的矫形效果及牢固的稳定性。患儿可早期活动，术后无需长时间佩戴支具。较前后路半椎体切除创伤小、手术时间短、术后恢复快。不足之处是术野暴露范围有限，需要更多的牵拉硬膜或者神经根，导致神经损伤风险增高。但其手术效果及并发症的发生率与术者手术学习曲线有一定关系。单纯后路切除术后平均恢复周期也较联合切除术短。因此，单纯后路半椎体切除较前后路半椎体切除术具有一定优势。手术过程主要包括内固定置入、后路半椎体切除（包括椎板，上、下关节面及横突，若半椎体位于胸段，还需切除肋骨头及近端多余的肋骨）、凸侧加压、节段融合等步骤。术后局部侧弯可得到即刻改善，同时，头侧及尾侧的代偿弯也可以得到一定的矫正。对于半椎体合并对侧骨桥形成的患者，也可获得满意的主弯矫正。然而，螺钉脱出、断钉、断棒等相关并发症成为该术式需要改进的问题（图6）。  
适应证：年幼，半椎体位于端椎，且原发畸形不严重的患者最适宜行该项手术。腰骶段的半椎体通常会导致严重的畸形，引起明显的躯干偏移及巨大的代偿弯。因此，建议早期行半椎体切除。
