#  脂肪移植隆乳的原则  
（1）适应证：小乳症；乳房萎缩；Poland 综合征、先天性胸廓畸形； 乳腺癌切除术后的乳房重建； 外伤性、医源性 乳房局部组织缺损；假体隆乳术后局部组织覆盖不足。  
（2）禁忌证：有严重的器质性疾病，如严重心脑血管疾病、血液系统疾病、恶性肿瘤等；存在精神疾病或精神异常；对手术效果有不切实际的预期；身体某一部位存在炎症或感染病灶。  
（3）供区：脂肪来源供区通常包括上腹部、下腹部、大腿内侧、大腿外侧、大腿后侧、髂腰部、背部及臀部等部位。这些区域有大量的脂肪组织，脂肪容易获取，并且减少这些部位的脂肪组织能使局部形态得到改善，能够达到瘦身与丰胸一举两得的目的。  
（4）受区：理想的脂肪移植受区应具备2 个条件：足够的空间和充足的血浆营养。在乳房内，腺体组织与胸大肌之间存在腺体后脂肪，这一区域被称为乳腺后间隙，可作

#  自体脂肪移植隆乳术后处理与并发症  
自体脂肪移植技术在过去30 年里被广泛地应用于整形美容外科手术中，术后需注意的是：供区吸脂部位术后穿弹力衣加压1 个月，以促进供区皮肤的弹性恢复及外形平整；脂肪注射受区不需要特殊处理，2 周至1 个月内避免术区按摩。另外术后除了感染、出血、切口瘢痕、麻醉意外和心脑血管意外等手术风险以外，与脂肪移植隆乳术相关的并发症还包括以下几个方面：  
（1）吸收：移植物吸收是脂肪移植最常见的并发症。脂肪移植早期的水肿消失并不是真正的脂肪吸收过程。脂肪吸收主要是由于机械性损伤、受区局部血浆营养不良等情况所导致的脂肪细胞坏死。脂肪吸收的比例可高达$ 70\%$ ，因此通常需要不止一次的脂肪移植手术才能达到满意的效果。为减少脂肪吸收，应尽可能在脂肪抽吸、处理及注射的各个环节中进行仔细规范的操作，以保证移植到受区的脂肪细胞具有良好的存活力。

（1）乳房体积的变化：三维测量技术为评估脂肪移植隆乳术后的体积变化提供了良好手段。移植的脂肪组织通常在术后两个月内出现部分吸收现象，尤其是第一个月内的脂肪吸收非常明显，此后乳房体积的变化相对较小。自体脂肪移植隆乳术后，移植的脂肪必然会有一定程度的吸收，因此，乳房体积在一个月以后会比刚做完手术时小。而脂肪吸收的比例每个人不同，手术效果难以进行精确预测。但同时，自体脂肪移植后又有一定数量的脂肪存活，因此乳房的体积或多或少都会有一定程度的改善。

（2）乳房影像学检查：MRI、CT、乳房X 线及超声等影像学检查，可观察移植脂肪组织在乳房内的存活情况以及并发症的情况。如果移植脂肪发生液化、囊肿、钙化等并发症，通常可在术后$ 6\sim18$ 个月被发现。而依靠这些现代的影像学检查手段，可以比较容易地区分脂肪移植后的肿块与乳腺来源的肿瘤。  
（3）乳房内肿块的处理：自体脂肪移植后出现的肿块、硬结等如果比较小且量不多，可以注意观察而无须手术取出，如果出现较大硬块，则需要通过手术取出，可能会引起局部组织凹陷和切口瘢痕。
