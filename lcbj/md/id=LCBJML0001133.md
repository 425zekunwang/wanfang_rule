# 干眼的流行病学发展在曲折中前进  
# 8. 干眼的流行病学发展史  
早在1978 年德国的Rrecht 等就采用调查问卷的形式，在社区进行干眼的流行病学调查，调查问卷中设计了“你的眼睛经常有磨砂感吗？”结果显示，在$ 55\sim59$  岁人群中，$ 22.8\%$  女性和$ 9.9\%$  男性有磨砂感，在$ 45\sim54$  岁的人群中，约$ 20\%$  女性和$ 15\%$  男性有干眼的症状。也有学者采用自我评价干眼症状的问卷，结合干眼的客观检查进行流行病学研究：Shimmura 等在日本东京随机抽取普通人群中的2500 个体（平均年龄35.2 岁，男性97 人，女性403 人，$ 86\%$  的人在$ 20\sim49$  岁工作年龄段）发放包括干眼自我评价、有关干眼的知识、眼药水的使用及生活方式等30 个问题的调查问卷，调查结果发现$ 33\%$  的人自认为患有干眼，$ 29\%$  男性和$ 35\%$  女性患者为干眼症状阳性，其中最多见的症状是眼红、干涩感。  
随着干眼定义、诊断和检查方法的变化，干眼的流行病学研 究一直在变化。国外对干眼流行病学的研究也是近十年才开展较多，2007 年国际干眼工作小组发布DEWS Ⅰ报告，根据现有流行病学研究总结发布了干眼流行病学报告。该报告特别针对干眼流行病学数据、危险因素和干眼调查问卷三大方面进行了分析总结。2017 年国际干眼工作小组再次发布DEWS Ⅱ流行病学报告，目的是评价和总结在干眼流行病学领域现有的证据及更新对未来需要进行的研究的建议，使干眼流行病学的研究更加科学规范。  
#  干眼发病的危险因素  
自20 世纪90 年代中期开始，国内外许多学者围绕干眼患者的生活方式、饮食、日常行为等个体因素展开了大量的研究，并取得了一系列成果。  
目前，公认的干眼危险因素，包括女性、衰老、围绝经期雌激素治疗、omega-3 脂肪酸缺乏或高omega-6/omega-3 结构饮食、屈光手术、维生素A 缺乏、眼部放射治疗、骨髓移植、丙肝、一些特殊药物或手术治疗（抗组胺药物和减肥手术等）。其他确定的干眼危险因素，包括糖尿病、HIV 感染、自身免疫系统疾病、全身应用药物化疗等。饮酒、吸烟、咖啡因及女性经期在干眼发病中的作用仍有争议。口服避孕药、怀孕及人种差异是否是干眼的易感因素仍需要进一步研究。  
1 ）年龄  
近期多个研究证实衰老是干眼的危险因素之一，可能与随着年龄增长泪腺分泌功能减退有关，少数研究则认为年龄与干眼发病无相关性。目前，干眼的发病呈现年轻化的趋势。  
（2）性别  
大多数研究认为女性是干眼的易感因素之一，仅有少部分学者并未发现两者相关性。  
（3）睑板腺功能障碍  
睑板腺功能障碍（meibomian gland dysfunction，MGD）传统上曾被当作干眼的一个亚型——蒸发过强型干眼，但越来越多的研究倾向把MGD 和干眼分开研究。MGD 流行病学研究目前仍缺乏统一的定义及分级标准，2011 年IOVS 增刊对MGD 诊断有了初步的标准，但由于诊断过程较为烦琐，实际在流行病学研究中难以应用。值得注意的是，干眼和MGD 有些易感因素是共通的，比如衰老和女性。MGD 和干眼的关系不仅局限于临床表现的相似，也是干眼的危险因素之一。  
# （4）配戴角膜接触镜  
临床上长期配戴角膜接触镜（contact lens，CL）的患者常有眼干的主诉，这也是部分患者无法长期坚持配戴CL 的原因。CL配戴患者得干眼的概率是正常人的4 倍以上。2013 年国际泪膜及眼表学会的报告显示，干眼是CL 配戴患者眼部不适感的重要原因。CL 配戴后可以导致眼表微环境的改变，从而促使干眼的发生发展。  
# （5）骨髓移植  
随着近年来白血病诊疗技术的不断提高，骨髓移植（ bone  marrow transplantation，BMT）患者在临床上也越来越常见。接受BMT 治疗的患者多数有不同程度的移植物抗宿主反应（graft-versus-host disease，GVHD），其中，眼部表现由于严重影响患者生活质量，越来越受到眼科医生的重视。已有多个横断面研究表明BMT 术后的眼部GVHD 是干眼的重要危险因素。其中，受体与供体性别不同、免疫移植治疗方案不到位、病毒感染及伴有全身其他GVHD 是加重BMT 患者干眼的危险因素。GVHD 可以破坏患者的泪腺组织，因此造成的干眼病情往往较为严重，几乎所有GVHD 患者的干眼均需要接受泪道栓治疗。  
# 6）环境因素  
在干眼发病中环境因素的作用很早就受到学者们的重视。一般认为空气污染、强风、干燥气候及高纬度均是干眼的危险因素。部分研究显示，相对农村，城市干眼患病率更高。另外，干燥的室内工作环境也会促使干眼的发生发展。  
# 7）视频终端的使用  
大规模的横断面研究结果显示视频终端使用者（尤其是年轻人），发生干眼的概率非常高。有学者推测这主要是因为视屏终端使用者眨眼次数减少，从而加快泪液蒸发导致干眼症状。近年来，由于智能手机和电脑平板用户的大量增加，视频终端使用的人数越来越多，视屏终端使用诱发的干眼问题日益受到重视。由于这些用户的年龄越来越低，预计未来年轻人的干眼问题会越来越突出。  
（8）维生素缺乏  
维生素A 缺乏是一个重要的社会健康问题。因为维生素A缺乏可以导致眼表上皮形态和功能的异常，从而使患者对干眼易感。同时，近年来出现的减肥手术（即胃大部位切除来控制患者体重），由于使患者吸收维生素$ \mathrm{B_{6}}$     等能力下降，可能与干眼的患病存在一定联系。  
# （9）饮食  
饮食与干眼的相关性一直是学者们感兴趣的话题，不仅是因为可以借此揭示干眼的发病机制，更重要的是可以从患者的日常行为中加以干预达到控制，甚至预防干眼的目的。目前这方面的研究多数集中在必需脂肪酸的构成方面，特别是omega-3和omega-6 脂肪酸的平衡，其主要原因是必需脂肪酸具有抗炎作用，并且是泪液脂质层的重要组成部分。  
# 10 ）屈光手术  
原位激光角膜磨镶术（laser-assisted in situ keratomileusis，LASIK）是目前常见的屈光矫正手术。LASIK 术后的干眼既可以出现在早期，也可出现在晚期。一般认为LASIK 相关的干眼主要与术中切断角膜神经末梢有关。其他跟LASIK 相关干眼有关的危险因素包括LASIK 术中上皮瓣的位置及大小、屈光矫正的程度和激光切削深度。  
# 11）糖尿病  
糖尿病患者患干眼的概率是正常人的2 倍以上。糖尿病病程长和伴有糖尿病性视网膜病变使患者出现干眼的可能性加大。由于糖尿病患者角膜神经敏感性下降，因此，其流行病学研究不推荐单纯使用患者的自觉症状问卷或量表进行评价。  
# 12）精神因素  
在干眼发病中精神因素的作用在过去的5 年得到了深入的研究。尽管精神异常与干眼发病的因果关系尚未明确，但其作为干眼的危险因素逐渐在学界内达成共识。在未来的研究中需要加以鉴别的是，精神疾病本身使患者对干眼易感还是所使用的精神类药物使患者患病的概率加大。抑郁、焦虑及紧张均是干眼的高危 因素。  
# 13 ）睡眠缺乏  
睡眠缺乏可导致多种疾病，包括糖尿病、高血压、冠心病等。目前也有研究采用睡眠剥夺小鼠模型观察小鼠的眼表情况，发现睡眠缺乏可导致干眼。临床研究也发现缺乏睡眠与干眼相关，但其发病机制尚不清。  
# 14）遗传因素  
目前关于干眼致病或者易患基因的研究非常罕见。英国的双胞胎研究表明，仅有$ 30\%$  的干眼患者可能与遗传有关。多数学者推测干眼可能是一类多因素多基因的疾病。  
#  三种主流的患病率研究方案  
患病率是指单位时间内在某一人群中某一疾病的患病人群所占比例。目前主流的患病率研究方案有3 种：基于症状的患病率研究、基于体征的患病率研究、基于症状与体征的患病率研究。  
（1）基于症状的患病率研究  
以不同患者自觉症状为诊断标准的干眼流行病学研究所选样本的年龄、性别存在较大差异，所采用的诊断标准也有所不同，有些研究使用干眼症状（异物感、眼干、眼部烧灼）的出现频率，有些基于患者自诉的多个干眼症状，有些则将眼表疾病指数（ocular surface disease index，OSDI）$ >12$  分作为诊断标准。东南亚的干眼流行病学研究显示，以患者自觉症状为诊断标准的干眼患病率为$ 20.0\%\sim52.4\%$ ；新加坡的2 个研究显示，干眼患病率为$ 6.5\%$  和$ 12.3\%$ ；西班牙和美国使用类似的诊断标准发现，干眼的患病率分别为$ 18.4\%$  和$ 14.5\%$ ；英国和韩国的症状性干眼，患病率约为$ 20\%$ ；法国和伊朗使用OSDI 评分$ >23$  分作为症状性干眼的诊断标准，患病率分别为$ 39.0\%$  和$ 18.0\%$ 。  
大多数以患者自觉症状为诊断标准的干眼流行病学研究中女性患者均多于男性，女性患病率比男性高约1.5 倍。需要注意的是，大部分流行病学研究入组样本均要求在40 岁以上，因此干眼患病率的结果并不一定适用于年轻人。部分研究显示年轻人的患病率稍低于老年人。  
# （2）基于体征的患病率研究  
以患者体征为干眼诊断标准的流行病学研究结果与症状性干眼有所不同。这类研究所采用的诊断标准通常为单一或者多个阳性体征，包括泪膜稳定性、泪液分泌实验和角膜染色结果。如果以泪膜破裂时间低于10 秒为诊断标准，体征性干眼的患病率为$ 15.6\%\sim85.6\%$ ；以Schirmer 试验结果低于$ 5\mathrm{mm}$     为诊断标准，体征性干眼的患病率为$ 19.9\%\sim37.0\%$ ；以角膜染色评分为诊断标准，患病率为$ 5.8\%\sim77.0\%$ 。体征性干眼在不同人群中的患病率差异比较大，其原因包括使用不同体征指标、不同诊断截断值、入组样本的年龄、性别及人种差异等。  
多个研究都表明年龄越大，干眼体征阳性率越高。另外，人种差异对体征性干眼流行病学研究结果有一定影响，黄色人种的泪膜稳定性和眼表上皮完整性稍差于白种人。学者们对于性别与干眼体征的相关性仍然有不同看法，多数研究结果支持男、女干眼患者之间临床体征是没有区别的。  
# 3）基于症状与体征的患病率研究  
不同国家之间干眼的患病率存在较大差异，其潜在原因在于不同研究之间存在纬度差异、地区差异。目前多数干眼流行病学研究均是基于成人的统计数据，仅有部分亚洲地区的研究包含少量青少年数据。  
#  干眼的发病率研究  
相对于患病率，干眼的发病率相关的流行病学研究较少。发病率是指单位时间内在某一人群中某一疾病新发病例所占比例。Beaver Dam 眼科研究的结果表明，40 岁以上正常人5 年内干眼发病率为$ 13.3\%$ ，10 年内发病率为$ 21.6\%$ ；女性发病率（$ (25.0\%)$ ）高于男性（$ 17.3\%,$ ），且年龄越大新发干眼的可能性越大。英国的数据显示，$ 9.0\%$  的$ 20\sim87$  岁女性在过去的2 年内有过干眼症状，同期干眼发病率为$ 3.8\%$ 。干眼已成为全球流行性疾病，其发病率有逐渐升高的趋势。由于定义不同、调查时原始人群不同及使用的诊断标准不同，现有的流行病学资料难以比较，但也可以看出干眼发病率研究仍有待加强。  
#  干眼流行病学研究的新方向  
目前干眼的发病机制尚不完全清楚，其诊断尚无统一的金标准，因此其发病率的研究存在多种困难，完善相关的发病率研究是未来发展方向之一。此外，干眼引起的不适症状、视功能损害及心理情绪障碍等不同程度地损害了干眼患者的生活质量。  
近年来，国外逐渐展开对干眼患者生活质量的研究。Mertzani 等应用SF-36 通用型生活质量量表，从生理功能、生理职能、躯体疼痛、总体健康、活力、社会功能、心理健康、情感职能8 个领域对比研究130 例非干燥综合征干眼患者与32 例健  
康者和2474 例普通人群生活质量的差异。结果显示，与普通人群相比，非干燥综合征干眼患者在生理职能、躯体疼痛、活力 3 个领域的得分（量表分值为$ 0\sim100$  分，越高代表健康状态越好）低于普通人群。其中，轻度干眼患者主要表现为日常活动受限、躯体轻度不适、活力下降；中度干眼患者除了以上的不适，总体健康亦受损害；重度干眼患者除了心理健康这一领域以外，其余7 个领域的得分均低于普通人群。关于干眼相关的生活质量研究目前更多局限在其对生活质量带来的直接影响，间接引起的社会及经济负担尚没有研究结论，是干眼流行病学研究的新方向之一。  
随着交通方式和信息技术的发展，干眼的流行病学研究也可进一步加快步伐。患者可以更好地管理症状，寻找治疗方式。智能型手机的应用也可用来共享数据，方便流行病学的调查。  
流行病学是一门研究与应用并重的科学。它不仅研究干眼疾病的特征，而且可以将流行病学研究成果转化为控制措施来防治干眼，以达到有效地控制或预防干眼的目的。目前干眼的发病机制尚不完全清楚，更应强调采用流行病学研究方法为干眼临床及基础研究提供客观依据，帮助阐明一些尚未解决的问题。
