# 近视防控：需要政府主导，医教协同，全社会共同努力  
近视防控工作已经上升到国家发展战略层面，各级领导和政府高度重视，习近平总书记近期已就儿童、青少年近视问题做了重要指示和批示，教育部、国家卫生健康委员会等八部委印发的《综合防控儿童、青少年近视实施方案》（以下简称“实施方案”）文件提出了近视防控的总要求和具体目标：减少儿童、青少年新发近视率、近视进展率，并降低高度近视发生率。近视防控任重而道远，必须在政府主导下，教育和医疗等相关部门协同努力。要做到全社会了解和领会“实施方案”的精神，我们建议需做好以下工作。  
# 33. 建立保障机制，通过各级政府和各相关系统联动，有效推进近视防控工作  
1）各省、市、自治区组建政府近视防控协同领导小组，从  
制度建设、政策制定、基础投入和财政支持等方面保障近视防控工作开展。在政府主导下，成立儿童、青少年近视防控工作指导中心，联合各部门、医疗机构、学校等，及时分析儿童、青少年视觉健康问题，为近视防控提供科学有效的解决方案，并具体指导和推动全省各地的儿童、青少年近视防控工作。各地市、县区结合实际，组建相应的近视防控指导机构，行使相应职能。  
（2）成立儿童、青少年近视防控专家指导委员会，制定切实可行的近视防控总体规划、技术标准、实施方案等，为推进工作提供专业咨询和技术指导。  
（3）设立儿童、青少年近视防控工作实验区，选择一些典型区域先行先试。政府主导、部门联动、社会参与，开展近视防控试点和前导调研工作，积极探索推进青少年近视防控工作的有益做法，结合城乡、区域实际制定工作规范，汇总解决实际工作中发现的关键问题，积累经验，并发挥典型示范作用，引领近视防控。  
# 34. 贯彻落实好“实施方案”，真正做到近视 防控科学有效  
（1）开展近视普查，建立近视防控本底信息系统。近视基础数据的采集和统计结果关系到未来五年各项防控方案和绩效考核标准的制定。在普查基础上，建立儿童、青少年视功能和近视大数据信息库，科学分析，形成准确的、覆盖各年龄段的儿童、青少年视觉健康和近视的详细信息。  
（2）推进健康育人工程，发挥学校的近视防控主体作用。重视儿童、青少年健康体魄培育，推进实施“健康育人工程”，发挥学校在近视防控工作中的主体作用，切实做到“一增一减”，即增加户外活动时间，减少持续近距离用眼时间和强度。采取落实课改，调整课内外教学方案，减少考试次数；保障体育类课程时数，保证每日课外或户外活动时间不少于2 小时；明确电子产品使用规范，调整电子产品授课比重，控制持续使用时间；防止幼儿园小学化等措施防范近视发生。  
（3）推进教医协同，强化学校、医疗机构共防机制建设。建立学校（教师、校医）和基层医疗机构、专科医院、三甲医院协同近视防控联系机制。加强教师对学生视觉健康的监督作用，做到早期发现和及时上报。提升校医利用验光仪等设备开展近视筛查，进行早期诊断和早期干预的能力。加强三甲医院眼视光部门在眼健康和近视诊疗中的引导作用，科学干预，延缓近视进展。推动基层规范化视光诊所建设，开展医学验光，科学配镜，发挥近视防控的指导作用。  
（4）强化家庭责任，构筑家长、教师、学生三重防控保障。发挥家庭、家长在儿童、青少年视觉健康教育中的关键作用，与学校、教师保持一致，加强“近视是一种疾病”的认识，培养儿童、青少年“自身视觉健康第一责任人”的意识，保证孩子用眼卫生、学习姿势和电子产品的合理使用，加强健康饮食、睡眠管理、体育锻炼和户外活动，引导孩子养成自觉保护视力的学习、生活习惯，遇到视觉健康问题，能够第一时间告知家长，及时得到帮助和诊治。  
（5）规范行业标准，保证有利于近视防控的教学环境建设。推动“健康教室建设工程”，制定科学的教学环境标准和行业规范，加强与视觉健康和近视防控密切联系的环境、器具标准的研究和落实。增加各级教学督导对教学环境、器具标准的监督职能，实现标准检查常态化。推进实施“健康教室照明”计划，适当上调各项技术标准，保证不同天气、时段，桌面、黑板等器具表面的健康光照度。  
（6）加强医疗知识技能培训，形成近视防控的长效机制。加强校长、教师、校医的儿童、青少年视觉健康、近视防控意识和知识培训。从学校管理层面将近视防控提升到学生身心健康培养的关键位置；从一线教学层面将视觉异常的早期发现作为教书育人的重要技能；从学校卫生保健层面将近视筛查和干预纳入到校医考核的基本内容。在校园中形成重视视觉健康，关注近视防控的良好氛围。  
（7）加强科技研发，推广近视防治的有效方法和手段。加强视觉健康、近视防控大队列建设和研究工作，开展儿童、青少年近视人群干预和临床研究，制定科学有效的儿童、青少年近视诊疗方案的标准化研究和推广，打击以近视治疗为噱头的伪科学。  
（8）加强近视防控资料编写和科学普及宣传。营造重视近视防控的全民氛围，建立完善的近视防控科普和宣传体系，建立权威、科学的近视防控知识、信息发布、传播和科普教育平台，引导公众正确认识、有效接收近视防控诊治的科学知识和信息。  
# 35. 基于现有的科学发现，分层次有重点进行 防控工作  
1）儿童- 青少年阶段，近视防控的重点人群  
儿童- 青少年近视是近视防控的重点。在小学$ 2\sim3$  年级会出现一个高发点。多数的儿童、青少年近视发病后，其近视程度呈现一个先快后慢的上升过程，直至20 岁左右达到初步稳定，变化趋势接近他们的生长发育曲线。  
针对儿童及生长发育期的青少年，最重要的矫正手段是建立完善的屈光发育档案，通过定期的随访复查，监测儿童、青少年的近视度数及眼轴长度的变化。在此基础上，验配合适的框架眼镜通常是该类患者的第一选择。尤其是8 岁以下的儿童，尚未形成良好的自我管理能力，宜使用简便的框架眼镜矫正近视；目前的临床研究显示，足矫的框架眼镜相对不戴镜、欠矫或过矫的眼镜对患者的近视进展具有更好的效果，并且近年来渐变镜、周边离焦设计的特殊镜片均在近视进展的控制方面展现了一定的作用。  
同时，屈光发育档案建立的意义还在于提示我们是否应选择更为特殊的控制手段，在合理验配框架眼镜的基础上，若儿童-青少年的近视进展仍呈现渐进加速的趋势，角膜塑形镜就成了更好的选择，特别是$ 8\sim18$  岁的青少年，生长发育较快，也是近视进展的高峰期，且具备一定的自我管理能力，是使用角膜塑形镜的“黄金期”，诸多临床研究也证实了其具有控制或延缓近视进展的功效。  
值得指出的是，以上年龄段的患者，近视度数多数处在一个动态的变化过程中，通常不建议行屈光手术治疗。  
针对部分病理性近视的青少年，凹透镜有缩小物像的作用，即框架眼镜往往会导致物像失真，此时选择隐形眼镜可以避免这个问题。特殊设计的周边离焦软镜也具有一定程度控制近视进展的作用，若近视进展的趋势及程度难以控制，甚至影响到最佳矫正视力，可以考虑行后巩膜加固术延缓近视的进展。  
2 ）中、青年人近视明显变化，要高度关注眼部疾患  
在25 岁以后，大部分人近视不会有大的进展，若此时短时间内出现近视增加，而且波动明显，要考虑病理性近视，或圆锥角膜等眼部疾患的可能。这些疾病均以进行性近视加深为共同特征，伴有不同的眼部体征变化。若成年仍出现近视度数的较大变化，一定要及时就医，部分眼部疾患拖延至后期甚至有致盲的可能。  
（3）老年人的视力下降或近视度数增加要关注眼的“调节器官”变化  
晶状体是人眼调节系统的一部分，类似于照相机的变焦镜头，通过它在眼内的“辗转腾挪”，人眼可以看清远近距离不同的各种物体，但是随着年龄的增长，调节系统也会随之老化，带来的第一个问题就是老花——视近不清，这会给许多老年人带来生活上的烦恼。然而，随着年龄的进一步增长，在50 岁以后，部分老年人会自觉近距离的视力提高，这是否是调节系统恢复的表现呢？显然并不是，这是随着晶状体的进一步老化，晶状体核的密度增高，对光的折射能力增强，变相地使老年人的近视度数升高，在使近视力升高的同时，造成远视力的下降，并且，晶状体的老化将会持续进行，最终导致晶状体混浊，各个距离的视力均下降。因此，老年人如果出现近视加深现象或者老花症状的缓解，要尤为警惕是否为白内障进展的表现。  
# 4 ）关注病理性近视诱发的眼底病变  
近视发展到一定阶段，会引发眼底（视网膜）的病变，包括视网膜劈裂、视网膜变性、视网膜脱离、脉络膜新生血管等。病。既往认 为，这些病变仅出现在高度近视的患者群体中，然而，最近的临床研究显示，近视本身就是部分并发症的独立致病因素，近视度数越高，出现并发症的概率越高。  
通常建议，高度近视的患者每6 个月需要散瞳检查眼底一次，及早发现细小的周边视网膜变性或裂孔，提前干预。对于近视无法控制的患者，在具有手术适应证的前提下建议行后巩膜加固术以控制近视进展。针对视网膜脱离、视网膜裂孔等并发症，目前临床仍以手术治疗为第一选择。  
#  以“变”的科学态度，看待近视矫正技术发展  
截至目前，近视矫正三大成熟方法（框架眼镜、隐形眼镜、角膜屈光手术）已经日趋成熟，并且针对不同年龄段患者的变化特点，有了更多不同的选择策略。  
8 岁以下的儿童，尚未形成良好的自我管理能力，宜使用简便的框架眼镜矫正近视。  
$ 8\sim18$  岁的青少年是使用角膜塑形镜的“黄金期”，框架眼镜和隐形眼镜可作为常规选择，但不宜进行屈光手术。  
$ 18\sim40$  岁的成年人，近视状态趋于稳定，这个年龄段是行屈光手术的最佳时期，可以免去戴眼镜的麻烦。框架眼镜和隐形眼镜仍可作为常规选择。角膜塑形镜虽然已经失去了延缓近视进展的优势，但希望在白天摘镜，又惧怕手术的患者也可使用。  
40 岁以上的老年患者，逐渐出现老视症状，且四肢灵活性下降。此阶段，渐变多焦点框架眼镜是不错的选择。  
（1）框架眼镜  
框架眼镜经过一百多年的发展，囊括了目前近视矫正的方方面面。单焦、双焦及渐变多焦点镜片基本满足了近视及复合型屈光不正的矫正需求。镜片设计上经历了从球面镜片、非球面镜 片、周边离焦设计镜片的变迁，材料的透光性与屈光指数也日趋上升，镀膜技术的发展为镜片的实用性与耐用性提供了保证。目前，框架眼镜仍是多数患者矫正近视的第一选择。  
# （2）隐形眼镜  
隐形眼镜则具有框架眼镜不具有的应用前景与优点：$ \textcircled{1}$    良好的视觉质量，尤其针对高度近视、不规则散光、屈光参差的患者，隐形眼镜对物像放大率影响不大的特点及对散光矫正的良好性能成为良好视觉质量的保障，这是普通框架眼镜无法以拟的。$ \textcircled{2}$    设计不同，隐形眼镜对近视的控制效果不同。目前多数临床研究证实了OK 镜、周边离焦设计软镜等隐形眼镜均具有控制或延缓近视进展的效果，并且优于特殊设计的框架眼镜，这是设计上的优势。同时，随着隐形眼镜材料的更新，镜片透氧性提高，安全系数大幅度提升。中国相关规定，8 岁以上近视的儿童均可在具有正规资质的验配机构验配使用OK 镜，其安全性能也已得到了保障，但值得注意的是这仍需要家长的高度配合，及时发现配戴过程中出现的问题，避免因儿童对问题反映的不及时而造成不良影响。  
# （3）屈光手术  
临床开展的屈光手术仍以角膜屈光手术为主。目前，角膜屈光手术基本已全面跨入了“无刀时代”，以激光切削为主的角膜屈光手术成了当前的主流，包括T-PRK、FS-LASIK、SMILE 手术等，中国及美国均已通过了相关征兵法令，允许激光手术后的近视患者在裸眼视力良好情况下可以参军入伍。激光手术术后长期的有效性及安全性均已得到了国际的认可，这里尤其建议不规则散光、屈光参差的患者在条件允许、具有摘镜需求的情况下， 选择激光手术矫正近视。  
4 ）药物治疗  
针对近视的药物治疗研究，以长期应用不同浓度的阿托品眼用制剂为主要的研究方向。目前已证实了阿托品的浓度越高，控制近视的效果越好，然而，不良反应也越大。现今的研究热点聚焦在使用超低浓度的阿托品控制近视的临床研究，这有待研究的进一步完善。  
需要注意的是，视物模糊的变化不一定是近视发生了变化。当使用屈光矫正手段仍不能达到较满意的视力时，应警惕眼部疾病的发生，尤其应关注眼底的改变，因为这里的病变大多会引起不可逆的视觉损害。
