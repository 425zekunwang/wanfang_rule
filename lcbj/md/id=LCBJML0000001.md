# 青少年特发性脊柱侧凸的手术分型  
1. 青少年特发性脊柱侧凸是发生在青少年阶段的一种原因不明的最常见的脊柱畸形  
特发性脊柱侧凸（idiopathic scoliosis，IS）是一种原因不明的最常见的脊柱畸形。根据其发病年龄又分婴儿型（$0\sim3$ 岁）、少儿型（$4\sim10$ 岁）及青少年型（10 岁后）。其中青少年特发性脊柱侧凸（adolecent idiopathic scoliosis，AIS）最常见，普查显示占青少年人数的$2\%\sim3\%$，占整个脊柱侧凸发病率的$80\%$，严重危害着青少年的身心健康。AIS 发病机制及发展过程非常复杂，表现形式多种多样。轻度的AIS，一般只表现为脊柱的畸形，除引起脊柱的不

# 2. 青少年特发性脊柱侧凸的治疗正处于一个不断进步的阶段  
随着材料科学、生物力学、内固定系统等研究的迅速发展及手术操作的不断改进，新的治疗方法、治疗技术不断涌现，AIS 的治疗正处于一个不断进步的阶段。青少年期正是脊柱的生长发育期，如果不对脊柱侧凸累及的椎体进行手术内固定及融合，脊柱侧凸将会迅速进展，难以控 制。然而，若对脊柱侧凸的累及椎体都进行手术内固定及融合，则会形成一个僵硬及生长停滞的脊柱。因此，对于AIS 的手术治疗，必须在尽可能保留脊柱生理功能的前提下，进行脊柱矫形。所以，如何减少融合固定节段，同时避免失代偿的并发症，一直是广大学者所追寻的目标和争 论的热点。  
#  青少年特发性脊柱侧凸是三维畸形  
国际脊柱侧凸研究学会（Scoliosis Research Society，SRS）对脊柱侧凸的定义如下：应用 Cobb 法测量站立位正位（冠状面） X 线影像中的脊柱侧方弯曲度，如果角度＞ $10^{\circ}$，则定义为脊柱侧凸。但脊柱侧凸并不只是冠状面上的简单弯曲，还有矢状面的异常及椎体轴位面上的旋转，胸椎的侧凸还会引起胸廓的畸形。因此，脊柱侧凸是复杂的三维畸形，脊柱侧凸的类型及表现形式复杂多样。手术治疗中不能仅仅满足于冠状面畸形的矫正，应同时重视矢 状面及旋转畸形的矫正，特别是矢状面的矫形，尽可能使三维畸形得到三维矫正，重建和稳定脊柱的平衡，避免脊柱侧凸的进展及失代偿的发生。  
# 4. 制定科学合理的手术分型系统对青少年特发性脊柱侧凸手术的规范化治疗至关重要  
AIS 具有多种不同的表现类型，而每个类型的侧凸均有不同的特点，手术治疗的方法和融合范围也完全不同。因此，矫形手术复杂，尤其是在决定手术入路、决定融合范围时非常困难。如果手术融合范围不够，术后就很可能发生失代偿，畸形进一步加重，甚至必须再次手术；若延长融合节段，手术范围过大，除了创伤大外，术后可能导致脊柱丧失较多的活动功能及生长潜力；有时矫形破坏了平衡，导致术后畸形加重，躯干偏斜；有时矫正了冠状面畸形却产生了矢状面畸形，术后发生曲轴现象、近端或远 端的交界性弯曲或后凸……因此，如何使AIS 的手术治疗规范化、系统化，使手术有章可循，确定正确的手术入路和融合范围，对手术治疗非常关键，直接关系到治疗的效果和患者的满意度。而解决这一问题的方法就是制定科学合理的手术分型来指导手术。一个完善的、可信度和可重复性高的、临床上有可操作性的、并为大家所接受的分型系统对AIS 手术的规范化治疗至关重要。  
# 5.  当今公认的能指导青少年特发性脊柱 侧凸手术的分型系统主要有3 个：King 分型、Lenke 分型及协和（PUMC）分型  
长期以来，为了便于分类、指导手术及学术交流，不少国内外学者对AIS 进行了分型研究，如今仍一直是脊柱外科研究的难点与热点。1950 年，Ponseti 和Friedman 将脊柱侧凸分为腰弯、胸腰弯、胸弯、胸腰双弯及颈胸弯。1992 年，Winter 和Lonstein 根据形态学将其分为7 种类型。1998 年，Coonard 等根据顶椎位置和侧凸方向将IS 分为9个类型 11 个亚型。但是，所有这些早期的分型主要是基于脊柱侧凸的形态学分型，临床应用价值不大。随着手术技 术及内固定器械的发展，用于指导手术的分型也逐渐发展起来。当今公认的能指导AIS 手术的分型系统主要有3 个：King 分型、Lenke 分型及协和（PUMC）分型（表1）。三种分型的特点还会在下文作详细介绍。  
6. King 分型开创了脊柱侧凸分型指导手术的先河，但其不能正确反映IS 的三维畸形，且分型不全面、观察者间及观察者内的一致性差异较大  
1983 年，King 从405 例脊柱侧凸手术中总结出来的King 分型，首次将分型与融合范围的选择结合起来。King分型主要以胸弯为主进行分型，共分Ⅴ型。Ⅰ型：胸弯和腰弯，腰弯Cobb 角度数大于胸弯，腰弯柔韧性小，一般融合胸弯和腰弯直至尾端稳定椎，但一般不低于$\mathrm{L}_{4}$；Ⅱ型： 胸弯和腰弯，胸弯 Cobb  角度数大于腰弯，腰弯较柔 韧，可行选择性胸弯融合至稳定椎；Ⅲ型：单胸弯，仅需融合胸弯至稳定椎；Ⅳ型：长胸弯，$\mathrm{L}_{4}$ 倾斜并位于主弯内，需融合胸弯至稳定椎； Ⅴ型： 结构性双胸弯， $\mathrm{T}_{1}$  倾斜于上 胸弯的凹侧，上胸弯常为结构性，需融合上下两个胸弯， 下端至稳定椎。但是，King 分型是一种冠状面分型，是从Harrington 内固定系统冠状面矫正脊柱侧凸的经验中总结出来的，没有考虑矢状面和水平面的畸形，不能正确反映IS 的三维畸形。因此，并不完全适用于CD 三维矫形内固定系统及其衍生出来的节段性脊柱三维内固定系统。许多King Ⅱ型脊柱侧凸在选择性胸弯融合术后，出现了失代偿。此外，King 分型不全面，有些侧凸类型如单腰弯、  
单胸腰弯及三弯未包括在内。根据Lenke 等的研究显示，King 分型观察者间及观察者内的一致性差异较大，不是一种足够客观的分型方法。但King 分型开创了脊柱侧凸分型指导手术的先河，具有里程碑的意义。King 分型应用了30多年，至今仍有人应用，根本原因是其比较简单、实用、便于记忆和理解。  
# 7. Lenke 分型系统更为全面，但结构性弯的定义争议较大  
2001 年，Lenke 基于对315 例三维矫形的IS 手术病例的分析，提出了Lenke 分型系统。该分型由3 部分构成：弯曲类型（分为$1\sim6$ 型）、根据腰弯顶椎与骶骨中线（CSVL）的位置关系制定的腰弯修正型（分为A、B、C 型）及胸椎矢状面修正型（分为－、$_\mathrm{N}$、$^+$型），3 个参数组合起来共计42 种类型。Lenke 分型系统的弯曲类型共分6 型，1  型： 单个主胸弯； 2  型： 双胸弯，下胸弯为主弯； 3  型： 双 弯（胸弯和胸腰弯/ 腰弯），胸弯为主弯；4 型：三弯，上胸弯、下胸弯和胸腰弯 /  腰弯，下胸弯和胸腰弯 /  腰弯均 可以为主弯；5 型：单个胸腰弯/ 腰弯；6 型：双弯（胸弯和胸腰弯/ 腰弯），胸腰弯/ 腰弯为主弯。腰弯修正型分为3型，腰弯A 型：骶正中线通过腰弯顶椎的凸凹侧椎弓根之

8. 协和（PUMC）分型系统是以脊柱侧凸的侧弯数目进行分型的青少年特发性脊柱侧 凸三维手术分型系统  
自1983 年开始，我们在国家卫生和计划生育委员会（原卫生部）重点项目的资助下，在总结近20 年间北京协和医院骨科治疗的大量AIS 病例的基础上，根据427 例三维矫形手术分析，结合脊柱侧凸冠状面、矢状面和轴状面的三维畸形特点，于2003 年提出了协和（PUMC）分型系统，发表在国内《中华骨科杂志》上。2005 年发表在国际脊柱外科权威杂志《 Spine 》上。 PUMC  分型共 3  大类 型13 个亚型，在之后的推广及临床应用过程中，取得了满意的疗效。PUMC 分型中的所有定义（如侧凸、顶点）均遵循国际脊柱侧凸研究学会（Scoliosis Research Society，SRS）的标准，明确严格，易于国际学术交流，没有结构性弯等有争议的概念。  
PUMC 分型的最大特点就是在国际上首次提出以脊柱侧凸的侧弯数目进行分型，而不是像既往的所有分型那样根据侧凸的部位来分型。PUMC 分型根据脊柱侧凸的弯曲数目（即顶点个数）分Ⅰ、Ⅱ、Ⅲ   三大型，十分简单，容 易理解和记忆。而且侧弯数最多不会超过3 个，所以分型全面，几乎包括了所有类型的侧凸。Ⅰ型是1个顶点1个弯，

# 9. 协和（PUMC）分型系统改良后，对Ⅱ型的选择性融合标准进一步优化，临床操作性进一步加强

# 10. 协和（PUMC）分型系统分型全面而简单、指导意义大、可信度和可重复性令人满意  
PUMC 分型是手术分型，每型有相应手术入路及融合范围，指导意义大。并且是三维分型系统，适用于当今的三维器械矫形。因此，根据PUMC 分型进行的矫形融合手术，其失代偿率极低。通过对西班牙巴塞罗那Sant Joan de Deu 医院145 例AIS 的回顾性研究表明，符合PUMC 分型进行手术的失代偿发生率明显低于King 和Lenke 分型。
