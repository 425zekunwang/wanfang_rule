# 突发性聋的临床治疗手段和方法  
由于突发性聋发病急剧，听力下降严重，因此应作为耳科的急症来对待。但是，目前由于缺少对所有突发性聋患者均有效的统一标准的治疗方案，因此突发性聋的治疗要强调个体化治疗的原则，治疗时机力求早期治疗、尽早抢救听力，治疗手段和方法要综合考虑分别实施，达到以最好的方法在最短时间内提高患者听力的目的。

# 14. 降低纤维蛋白原的药物——巴曲酶治疗突发性聋比较安全  
临床上，已有大量试验对突发性聋患者的血流动力学进行了检查，发现其纤维蛋白原水平和红细胞集聚、血浆黏稠度较正常对照组显著升高，表明血浆纤维蛋白原增高在突发性聋发病中占有重要地位。因此，降低血浆黏稠度和纤维蛋白原浓度就可以改善血液流动特性，以利血液流动向内耳供血，恢复细胞和组织供血量和氧量，来恢复听

# 17.  临床应避免应用那些对突发性聋治疗 无效的方法  
突发性聋的主要病因是内耳缺血及血液循环障碍，因此，主要治疗手段和方法是改善血液循环，纠正供血、供氧，以最大限度保护细胞和组织免受进一步损害。但是，也有一些方法已在临床上应用，其结果并不令人满意，在此归纳一下以供临床工作者参考。  
高压氧治疗突发性聋已有较多报道，一般认为，高压氧可以减轻内耳水肿和缺血缺氧损害，临床试验用高压氧治疗突发性聋的有效率达$70\%$ 以上。但是，有学者认为高压氧会造成内耳组织氧自由基增加，反而加重病情，而不推荐突发性聋患者首先或积极做高压氧治疗。已有试验证

行星状神经节阻滞手术治疗突发性聋的也有临床报道，56 例突发性聋患者行星状神经节阻滞手术的结果是$70\%$ 的听力明显改善，认为星状神经节是由交感神经干的颈下神经节和胸上神经节组成，星状神经节阻滞后，同侧的椎动脉和颈总动脉血流的速度、血流量和血管横截面积均增加，从而提高耳蜗血供，耳蜗血流得以改善。但是，这一手段属于有创治疗，患者难以接受，其结果和效果无大样本佐证，很难使人认可此种治疗手段。目前，在国内几乎无人采用此法来治疗突发性聋。

# 18. 长期以来突发性聋的治疗较为混乱，根据听力曲线进行分型治疗十分必要

（6）对于治疗效果不佳者，听力稳定后，可根据听力损失程度，选用助听器或人工耳蜗等听觉辅助装置。  
# 20. 突发性聋诊断和治疗指南（2015）中对突发性聋分型治疗建议  
低频下降型的治疗原则：由于可能存在膜迷路积水，故需要限盐，输液量不宜过大，最好不用生理盐水。若低频下降型平均听力损失$<30\mathrm{dB}$，自愈率较高，可口服给药，包括糖皮质激素、甲磺酸培他司汀，以及改善静脉回流的药物（如马栗种子提取物）等。也可考虑鼓室内或耳 后注射糖皮质激素，如甲泼尼龙、地塞米松或复方倍他米松等。听力损失$>30\mathrm{dB}$ 者，可用银杏叶提取物注射液加糖皮质激素静脉用药。小部分患者采用上述方案治疗无效或耳闷加重时，应给予降低纤维蛋白原药物，如巴曲酶，以及其他改善静脉回流的药物进行治疗。  
高频下降型的治疗原则：首先要改善微循环，如应用银杏叶提取物等药物和糖皮质激素；离子通道阻滞剂，如利多卡因等，对于减轻高调耳鸣效果较好；可考虑使用营养神经类药物，如甲钴胺、神经营养因子等。  
全频下降型的治疗原则：包括平坦下降型和全聋型，主要应用降低纤维蛋白原的药物，如巴曲酶、糖皮质激素；改善内耳微循环相关药物，如银杏叶提取物等。  
突发性聋的临床治疗手段和方法 049  
中国突发性聋多中心临床研究数据表明，改善内耳微，合理的联 合用药比单一用药效果要好，其中还显示低频听力下降型疗效最好，平坦听力下降型次之，高频听力下降型和全聋 型疗效不佳。因此，根据听力曲线的分型对突发性聋的治疗和预后的判断具有重要的指导意义，临床工作中要按此指南操作，才能做到对突发性聋科学合理有价值的救治。
