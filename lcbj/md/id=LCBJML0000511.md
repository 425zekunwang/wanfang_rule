# 【病例点评】  
对于乳腺癌患者来说，疗前基线检查非常重要，特别是那些高危的早期患者（如T3 或T4，淋巴结转移较多），我们需要充分地评估全身状态，必要时可行PET-CT 明确有无全身转移。  
目前乳腺癌分子分型指导我们对不同患者进行分类治疗，但同一亚型的乳腺癌患者仍可能预后不一，对药物的反应性也不尽相同，原发或继发耐药现象仍然存在，这提示我们需要继续探索精准医学，去找到那些“例外”的患者，为其选择更加合适的治疗方案。虽然目前的精准医学研究结果尚不能改变我们的临床实践，但该患者治疗的成功提示对于那些标准治疗疗效欠佳的患者，精准医学提供了新的治疗机会。
