# 从中医的角度探讨慢性阻塞性 肺疾病防控的新思路  
自1972 年起全国各地陆续开展了对慢性阻塞性肺疾病的中西医结合诊断分型研究，提出本病可分为脾阳虚、肾阳虚和肺肾阴虚。从中医辨证分型的角度来看，慢性支气 管炎—肺气肿—肺心病的发展过程，相当于中医的肺气虚—脾阳虚—肾阳虚—阴阳俱虚的演变过程。中医学认为，脾与消化、吸收功能十分密切。当慢性阻塞性肺疾病进展到脾阳虚及其以后的阶段，机体则会出现明显的消化功能紊乱和营养代谢障碍。因此，弄清慢性阻塞性肺疾病患者脾阳虚的本质及其对机体的影响，有助于慢性阻塞性肺疾病的有效治疗。我们从1985 年以来先后发表了一系列关于中西医结合诊治慢性阻塞性肺疾病的文章。下面将分别介绍这些 文章的要点。  
# 61. 认识肺气虚的实质对慢性阻塞性肺疾病的发生发展过程研究有重要意义  
中医历来十分注意机体的抗病能力，认为“邪之所凑，其气必虚”“正气存内，邪不可干”“肺为气之本”“诸气者，皆属于肺”，可见气与机体抗病能力的关系十分密切，其中以肺气尤为重要。多年来国内许多学者先后从多方面对肺气虚的本质进行了一系列研究，初步证实了肺气虚时体内在许多方面确实发生了一定变化，这为我们进一步深入认识肺气虚的实质和慢性阻塞性肺疾病的发生发展过程提供了更多的依据。  
中医对肺气虚的诊断标准包括肺虚和气虚两个方面，我们做了一些肺气虚客观指标的研究，近年来人们已经逐渐认识到肺不仅是一个呼吸器官，还是一个重要的内分泌代谢器官，现代肺功能概念包括呼吸、代谢和防御三个方面，因此，要想真正认清肺气虚的本质就必须从多方面进行研究，现摘列如下：  
（1）免疫功能：肺气虚患者的细胞免疫功能较正常人降低，体液免疫功能也低于正常人，表现为患者痰中IgA  
含量降低，痰中溶菌酶浓度降低，提示呼吸道局部体液免 疫功能下降，与中医所说的肺气虚者卫外功能低下相符。  
（2）环核苷酸水平测定结果显示：环磷腺苷（cAMP）的含量低于正常。  
3）$ \alpha_{\ 1}–\mathrm{AT}$     水平降低。  
4）血清蛋白结合碘水平低于健康对照组。  
5）自主神经功能紊乱：以副交感神经功能亢进为主。  
（6）胸部X 线片检查发现肺气虚患者肺纹理增多增粗，部分表现为肺气肿、心脏增大、右下肺动脉横径增宽。  
（7）肺功能检查：绝大多数肺气虚患者肺功能损害已达到中重度通气功能损害的标准，包括$ 50\%\mathrm{VC}$     时的最大呼气流速、$ 25\%\mathrm{VC}$     时的最大呼气流速明显降低。肺气虚组患者的气道阻力明显高于正常人，大约$ 70\%$  肺气虚患者出现中、重度肺气肿。  
（ 8 ）血流动力学测定结果显示肺动脉血流量减少，肺 循环阻力增加。  
（9）血液流变学测定结果显示肺气虚患者全血比黏度、血浆比黏度明显异常，血细胞比容、血浆渗透压和红细胞 沉降率出现不同程度改变。  
（10）胃肠功能：部分肺气虚患者出现小肠排空时间延长。  
综上所述，目前的研究结果显示肺气虚的主要变化包括体细胞免疫和体液免疫功能降低、血浆cAMP 水平降低、血清$ \alpha_{\ 1}{\cdot}\mathrm{AT}$     水平降低、副交感神经功能亢进、肺通气功能降低、气道阻力增加、不同程度的肺气肿，肺血管弹性降低、肺血流量减少、肺循环阻力增加、血压流变学异常等。  
62. 弄清慢性阻塞性肺疾病患者脾阳虚的本质及其对机体的影响，有助于理解慢性阻塞性肺疾病患者营养障碍的发生机制，有效地改善营养状态  
1）慢性阻塞性肺疾病患者脾阳虚本质的研究情况。  
$ \textcircled{1}$    胃肠道运动功能变化：胃肠道形态和运动功能研究表明，脾阳虚患者胃蠕动功能减弱，张力减低，排空速度减慢，空腹潴留胃液增多，肠道蠕动加快、运动亢进、排 空速度加快，影响食物的消化和吸收。  
$ \textcircled{2}$    胃液变化：脾阳虚患者基础胃酸和最大胃酸分泌量均显著高于正常人。  
$ \textcircled{3}$     唾液淀粉酶的变化： 脾阳虚患者唾液腺分泌功能障 碍，负荷能力降低，对酸刺激反应降低。  
$ \textcircled{4}$    小肠吸收功能降低。  
$ \mathcal{S}$    大便形态和成分的变化：研究显示脾阳虚患者粪便中可见到较多未被消化的食物残渣和脂类物质及淀粉颗粒。  
$ \textcircled{6}$     糖代谢变化： 脾阳虚患者血清乳酸水平明显升高， 柠檬酸水平降低，乳酸 /  柠檬酸比值升高，提示体内有氧 代谢率降低。  
$ \textcircled{\scriptsize{2}}$    能量代谢变化：脾阳虚患者红细胞内ATP 含量减少。  
$ \circledast$ 水盐代谢变化：研究结果提示脾阳虚患者电解质失衡，可能与肾上腺皮质功能不足有关。  
$ \textcircled{9}$    内分泌系统变化：研究结果显示脾阳虚患者肾上腺皮质功能低下，表现为血浆皮质醇低于正常人，尿17- 羟和17- 酮降低，甲状腺功能低下。  
$ \circledcirc$ 自主神经功能紊乱：大多数脾阳虚患者表现为交感神经功能低下，副交感神经功能亢进。  
（2）中医学对脾阳虚的治疗有一套比较完整的理论、治则和方药。现已证明健脾治疗可以调整胃肠道功能，改善消化吸收功能，促进物质代谢，提高代谢水平，调整已失衡的自主神经功能，提高免疫力。目前认为健脾治疗具有以下作用：  
$ \textcircled{1}$    改善胃肠道运动功能，促进胃蠕动，抑制肠蠕动。  
$ \textcircled{2}$    提高胰腺外分泌功能，使尿淀粉酶水平升高。  
$ \textcircled{3}$    提高小肠吸收功能。  
$ \textcircled{4}$    提高血浆蛋白，尤其是白蛋白的水平。  
$ \mathcal{S}$    改善糖代谢过程。  
$ \textcircled{6}$    提高能量代谢水平。提高线粒体氧化磷酸化能力。  
$ \textcircled{\scriptsize{2}}$    提高基础代谢水平。  
$ \circledast$ 调整内分泌功能，使血浆皮质醇、尿17- 羟和17-酮的含量升高。  
$ \textcircled{9}$     调整失衡的自主神经功能，降低血中胆碱酯酶的含 量，提高血浆中cAMP 的含量。  
目前健脾治疗中应用较多的复方制剂为四君子汤、黄芪建中汤、补中益气汤、参苓白术散、附子理中丸、香砂 六君子汤、健脾粉、开胃冲剂等。  
（3）慢性阻塞性肺疾病病营养障碍的原因：多年来国内外对这一问题虽然做过一些研究，认为慢性阻塞性肺疾病营养障碍是由多方面因素造成的，如机体能量消耗增多，药物影响食欲，呼吸困难妨碍进食和咀嚼，高碳酸血症使 胃肠道淤血， 疡等，但是这些研究都还不够系统和深入。而中医对慢性阻塞性肺疾病患者脾阳虚原因的研究包括了胃肠道的形态、运动功能、胃酸分泌、唾液腺淀粉酶分泌、小肠吸收功能、营养物质的转运功能、糖代谢、能量代谢、内分泌功能、自主神经功能等方面，这样不仅使我们对慢性阻塞性肺疾病营养障碍的原因、机制有了更全面的认识，而且对今后如何更全面地改善慢性阻塞性肺疾病患者的营养状态提供

更多的途径。  
（4）慢性阻塞性肺疾病患者营养障碍测定指标：目前国外评价慢性阻塞性肺疾病患者的营养状态的指标主要是体重、肱三头肌皮折厚度，上臂中点周径和上臂中点肌肉 周径、总铁结合力、铁蛋白、肌酐、身高指数等，这些指标中，有些过于简单、粗糙，有些不尽合理，而国内研究脾虚的客观指标比较全面，涉及消化系统功能、营养物质代谢和能量代谢功能、自主神经功能及内分泌系统功能、免疫功能。  
（5）营养障碍对机体的影响：国外对慢性阻塞性肺疾病患者营养障碍对机体的影响方面的研究比较细致和深入，主要涉及：$ \textcircled{1}$    营养不良可导致肺结构破坏，影响肺通气和换气过程。$ \textcircled{2}$    影响呼吸肌的结构和功能。$ \textcircled{3}$    降低防御系统功能，干扰机体抗氧化损伤功能。  
国内对这一问题也做了一系列研究，结果表明慢性阻塞性肺疾病患者发展到脾阳虚阶段之后病变已经不再局限于呼吸系统，可以累及消化、内分泌、神经、网状内皮系统等，这表明慢性阻塞性肺疾病患者的脾阳虚症候群不仅包括了肺本身，还累及肺外多个系统，而这些系统的变化反过来又进一步加重了呼吸系统结构和功能的变化。即慢性阻塞性肺疾病可以导致脾阳虚，而后者又会使慢性阻塞  
性肺疾病病情进一步加重，两个方面互为因果关系，从而形成恶性循环。  
（6）乎是通过静脉、肠道给患者补充必要的营养物质，但是计算摄入量繁琐费时，很难持久。另外，仅仅通过静脉、肠道补充一些必要的营养物质并不能从根本上解决这类患者的胃肠道运动功能减退、消化酶分泌功能异常、肠道吸收障碍、物质及能量代谢障碍等一系列问题。此外，慢性阻塞性肺疾病患者营养不良除了因为各种营养物质消化吸收不良以外，机体消耗增多也是一个重要原因，所以仅仅通过静脉、肠道补充各种营养物质并不能从根本上解决问题。  
中医认为慢性阻塞性肺疾病其标在肺，而本在脾肾。对于慢性阻塞性肺疾病患者只有采取“急则治其标，缓则治其本”的方法才能收到良好的效果。所谓治本则是强调在疾病的缓解期重视对脾虚的治疗。中医许多健脾方药都具有较好的效果，这些药物不仅可以改善患者的食欲，还有调整胃肠功能、促进消化酶的分泌、改善消化吸收功能、促进物质和能量代谢、调整自主神经功能等作用，特别值得提出的是中医健脾治疗不仅注重调整机体已经失衡的各种功能，而且十分注重整体效应，即注意调整机体各个系统之间的关系。  
综上所述，健脾治疗慢性阻塞性肺疾病在某些方面具有独到之处，如能将中西医结合起来有可能会使我们在慢性阻塞性肺疾病的研究和治疗方面有较大的改观。  
# 63.  从中医脾虚角度探讨慢性阻塞性肺疾 病呼吸肌疲劳的发生机制对于防治呼吸衰竭是十分重要的  
呼吸肌是呼吸泵的重要组成部分，是完成外呼吸的动力。呼吸肌收缩的能量来源于营养物质代谢产生的ATP。因此，呼吸肌的肌力主要决定于机体营养状况及代谢水平。研究证明呼吸肌疲劳是呼吸衰竭的重要潜在原因。不少研究表明慢性阻塞性肺疾病患者存在持续的呼吸肌疲劳，这 是引起呼吸衰竭的重要因素。因此，深入研究呼吸肌疲劳的发生机制，早期诊断和有效治疗对于防治呼吸衰竭是十分重要的。  
（1）国外关于呼吸肌疲劳的发生原因和机制有以下一些方面：$ \textcircled{1}$    长期低氧血症，有氧代谢障碍，呼吸肌能量供给减少。$ \textcircled{2}$    长期全身营养不良，除热量供应不足之外，尚有蛋白质、氨基酸、维生素及微量元素缺乏，久之则造成肌肉萎缩、收缩力下降。$ \textcircled{3}$    慢性阻塞性肺疾病时发生酸中毒，使呼吸肌肌力减低。$ \textcircled{4}$    肺气肿时功能残气容积增加，吸气肌长期处于收缩状态，得不到充分舒张和休息，久之使肌力减退。$ \mathcal{S}$    气道阻力增加，以及肺气肿时肺泡弹性回缩力降低，使呼吸肌负荷增加，耗能增多。  
呼吸肌疲劳的早期诊断：目前国外主要是通过测定呼吸肌的功能发现呼吸肌疲劳，主要测定指标包括，呼吸肌肌力、呼吸肌耐力、最大跨膈压和膈肌耐力、膈肌张力时间指数、膈肌肌电图、呼吸频率和节律以及呼吸类型的变化。  
呼吸肌疲劳的治疗：$ \textcircled{1}$    呼吸衰竭时适时采取机械辅助通气，减轻呼吸肌负荷，以利于呼吸肌疲劳的恢复。$ \textcircled{2}$    应用各种支气管舒张剂降低气道阻力减轻呼吸肌负荷。 $ \textcircled{3}$     应 用茶碱等强心药物增加膈肌收缩力。$ \textcircled{4}$    建立合理的饮食习惯和健康食谱，以保证足够的热量供应，必要时通过静脉短期补充营养。尽管上述措施可以不同程度地改善呼吸肌疲劳，但是并不能从根本上解除呼吸肌疲劳的病因。  
（2）中医学有关脾虚与肌肉关系的研究：中医学历来重视脾胃功能，认为“脾主身之肌肉”“脾虚则肌肉削”，由于脾主运化，可将五谷精微和水湿输部到全身，故脾健则肌肉强健有力，如果脾虚其运化能力不足，不能把足够营养物质运送到身体各处，久则脾气虚营血亏，肌肉失养 则疲弱无力，所以从现代医学观点来看，脾主肌肉是可以通过脾主运化这一功能得到很好解释的。

（3）健脾治疗对肌肉收缩功能的影响：许多研究表明，健脾治疗可以增强肌肉的收缩功能，据报道，补中益气汤、益气健脾汤、附子理中丸、健脾益气糖浆等均有较好的健 脾作用。  
（4）呼吸肌疲劳治疗的综合设想：根据中医学的“脾主身之肌肉”的理论，以及异病同治的原则，可以推测在慢性阻塞性肺疾病患者呼吸肌疲劳的发生过程中脾虚是一个重要环节，故可以通过健脾益气的途径对呼吸肌疲劳进行治疗。现已证明健脾治疗可以有效地改善消化吸收功能，促进物质和能量代谢，由此可以预料健脾益气治疗可能通过改善营养状态和代谢水平增强呼吸肌收缩能力。  
中医认为慢性阻塞性肺疾病其标在肺，而本在脾肾。根据急则治其标，缓则治其本的治则，初步设想慢性阻塞性肺疾病的呼吸肌疲劳的治疗方案如下：  
# 1）急性期：  
$ \textcircled{1}$    短期应用机械辅助通气，减轻呼吸肌负荷，为以后治疗争取时间创造条件。  
$ \textcircled{2}$     尽早纠正低氧血症，提高有氧氧化水平和能量代谢 水平，纠正酸中毒，增加肌肉收缩力，解除低氧血症引起的血管痉挛，降低肺动脉压，减轻右心负荷，改善心功能和外周血液循环。  
$ \textcircled{3}$    使用支气管舒张剂降低气道阻力，减轻呼吸肌负荷，改善通气，提高$ \mathrm{PaO}_{2}$    ，降低$ \mathrm{PaCO}_{2}$    ，纠正酸中毒。  
$ \textcircled{4}$    经过肠道或静脉补充营养物质改善营养状态。  
2）缓解期：应用健脾益气药物改善消化吸收功能，提高代谢水平，改善呼吸肌的结构和功能，减少呼吸衰竭的复发。  
# 64.  从脾虚角度探讨慢性阻塞性肺疾病患 者免疫功能降低  
国内外许多研究结果均提示慢性阻塞性肺疾病病情进展的原因除了病毒、细菌等外界因素的不断侵袭外，更重要的是患者机体免疫功能降低。国内一些学者提出从中医扶正固本的方法入手，可以提高患者的免疫力，这可能是一个很有前途的抗复发途径。  
（1）目前西医认为慢性阻塞性肺疾病患者免疫功能降低包括细胞免疫和体液免疫两大方面，此外还包括某些非特异性免疫功能。  
近年来人们已经逐步认识到慢性阻塞性肺疾病患者可能出现不同程度的营养不良，中重度营养不良对免疫系统的影响主要是干扰细胞免疫功能，蛋白质缺乏、维生素缺乏也可以使细胞免疫功能降低。营养不良时特异性抗体合成障碍，也可能是细胞免疫功能低下的原因。此外，营养

不良对于非特异性免疫功能也有一定影响，包括多种血清补体降低、转铁蛋白减少。  
（2）从脾虚的角度探讨慢性阻塞性肺疾病患者免疫功能的变化。中医学认为“正气内存，邪不可干”。而正气与肺脾肾三脏关系密切，其中脾肾最为重要。脾虚患者细胞的免疫功能、体液免疫功能、非特异性免疫功能降低，均可通过营养不良，特别是蛋白质缺乏和能量代谢障碍得到解释。一些学者认为中医学所说的脾实际上可能是以肠、肝、脾组成的综合功能单位，可以认为消化系统本身就是一个很重要的免疫器官。胃肠道本身就是一个具有免疫功能的淋巴网状组织，肠道淋巴组织具有控制体液免疫的功能，肠道黏膜固有层淋巴细胞大多能发育成可以合成和分泌各种免疫蛋白的浆细胞，这是体内合成IgA 的主要来源之一。分泌型IgA 是人体防御系统中最重要的防线之一，肠道中的溶菌酶和备解素可作为第二道防线。慢性支气管炎患者一旦发展到脾虚阶段，其胃肠功能和结构均会出现一系列变化，这些结构和功能变化可以导致肠道局部免疫功能的降低。  
（3）从脾虚的角度探讨慢性阻塞性肺疾病患者免疫功能低下的治疗。慢性阻塞性肺疾病患者其标在肺，其本常在脾肾，因此对于慢性阻塞性肺疾病患者不但应在急性发作期积  
极治疗各种临床症状，还应该在缓解期扶正固本，提高免疫功能，防治急性发作。近年来，越来越多的人认识到提高体液免疫功能是治疗慢性阻塞性肺疾病的重要措施。许多单味补肺健脾中药可以改善免疫功能，如人参、党参、黄芪、白术、茯苓、苡仁、山药等。目前可用于提高免疫力的复方有四君子汤、参苓白术散、补中益气汤和玉屏风散等。这些药物均可增强细胞免疫功能，促进网状内皮细胞的吞噬能力，增强非特异性免疫功能，促进炎症吸收。
