# 乳腺癌化疗基本地位与优化  
乳腺癌的标准治疗中，无论在早期还是复发转移阶段，化疗都占有重要地位。几十年来，基于大型临床试验的结果及治疗理念的进步，乳腺癌的化疗方案不断更新优化，化疗的获益人群也进一步明确。从分子分型治疗到精准医学时代的今天，我们需要更加精细的化疗。选择正确的患者、采取合适的化疗方案，既可避免过度化疗，也可避免不 足化疗。以下将就乳腺癌化疗的若干问题进行回顾与探讨。  
# 16. 辅助化疗在不同类别患者中的应有地位  
随着技术手段进步和研究的深入，对乳腺癌的认知已  
经从简单的同一疾病，到区分为不同的分子亚型的一类疾病，基于乳腺癌分子分型进行分类治疗是当前乳腺癌治疗 的基本原则。  
在大量临床研究数据积累的基础上，无论是最新版NCCN  指南、欧洲肿瘤内科学会（ European Society for  Medical Oncology ， ESMO ）早期乳腺癌临床实践指南（以 下简称ESMO 指南）、St.Gallen 专家共识，还是我国乳腺癌相关的指南均指出乳腺癌术后辅助全身治疗的选择应首 先基于肿瘤的病理分子分型及对不同治疗方案的反应性。  
对于早期三阴性乳腺癌、HER-2 阳性的乳腺癌进行术后全身辅助化疗的推荐基本一致。  
然而，Luminal A 型和Luminal B（HER-2 阴性）的乳腺癌患者是否需要辅助化疗，各个指南推荐有不一致和模糊之处。原因在于：$\textcircled{1}$普遍认为Luminal A 型对化疗敏感性欠佳。$\textcircled{2}$此类肿瘤预后相对好，TEXT 研究中绝经前“Luminal 型”的低危患者（无化疗亚组患者：$16\%<40$ 岁，$19\%$   肿瘤 $>2\mathrm{cm}$ ， $21\%$   淋巴结阳性）仅行内分泌治疗的 5 年无乳腺癌复发率可达$94.6\%$ 以上，不化疗预后也很好。$\textcircled{3}$患者预后好，使明确化疗获益的临床研究难以开展。  
对TEXT 研究中890 例选择化疗的Luminal 型患者相关因素进行了分析，结果显示淋巴结状态是选择化疗与否

# Ki-67 数值在化疗决策中的作用  
Ki-67 作为乳腺癌细胞增殖指数标志物已经有30 多年的研究历史，大量的临床研究已经证实Ki-67 是乳腺癌的独立预后和预测因素。普遍认为同一分子亚型的乳腺癌，Ki-67 与患者的预后呈反比，与化疗敏感性呈正比。  
但尽管如此，Ki-67 的最佳临界值仍存在争议。2009年St.Gallen 专家共识建议将$\mathrm{Ki-}67\leqslant15\%$ 定义为低值，$\mathrm{Si-}67>30\%$ 定义为高值；2011 年St.Gallen 专家共识建议将Ki-67 为$14\%$ 作为Luminal A 和Luminal B 的临界值；2013 年，这一临界值被重新定义为$20\%$，并提到各单位也可以设定本单位检测的临界值，如Ki-67 临界值为$20\%$，那么低于$10\%$ 属于低值，高于$30\%$ 属于高值。  
Ki-67 作为一项反应肿瘤中增殖细胞百分比数值，应为一个连续变量，临床实践中由于肿瘤的时空异质性和Ki-67 结果判读的主观性，精确判定Ki-67 数值存在较大难度，因而Ki-67 介于临界值附近数值的患者预后未必有很

# 18. 年龄在辅助化疗决策中的地位及作用  
需要明确的是，年龄从来不是决定患者是否需要辅助化疗的主要因素。年龄对乳腺癌辅助化疗决策的影响，主要体现在其反映的女性生理功能差异，从而影响患者接受标准的辅助治疗的可能性。  
对于年轻患者而言，一方面其脏器功能代偿良好，能够耐受治疗毒性；另一方面年轻女性卵巢生理功能旺盛。对于激素受体阳性年轻患者，高水平性激素可能是其不良预后因素之一。此外年轻患者相对于老年患者，期望生存期更长。  
在E-3193 和INT-0142 研究中，绝经前腋窝淋巴结无转移的HR 阳性乳腺癌患者，即使不化疗10 年生存率也高达$95\%$。在TEXT 和SOFT 研究中，分别有$21\%$ 和$8\%$ 的患者未接受辅助化疗，其5 年的无乳腺癌相关事件率分别为$97\%$ 和$96\%$。因而年龄是患者能够接受辅助化疗的重要因素，但年龄并非预后或预测因素。对年轻患者的辅助化疗治疗决策应该基于肿瘤生物学特征和肿瘤分级，而非单一参考年龄。  
对于老年患者，应基于患者肿瘤的生物学行为决策是否需要辅助化疗，再评估其接受标准化疗的可能性。由于辅助化疗带来的生存获益是远期且未知的，但毒性却是明确的，因而对于老年患者，应该基于多因素综合制定正确的治疗方案。这些因素不仅包括并发症、药物、脏器功能、营养方面，而且还包括生理状态和家庭社会条件。

# 19.  临床病理危险度与基因检测结果不一 致时辅助化疗如何选择  
随着精准医学和分子检测技术的不断进步，在传统临床病理特征的基础上，越来越多的生物标志物被用于全身辅助治疗决策。近期一篇文献对生物标志物在临床应用中的价值进行了评价。最终该文献认为，有充足证据表明Oncotype DX 、 EndoPredict 、 PAM50 、尿激酶型纤溶酶原 激活因子、纤溶酶原激活物抑制因子1 等对特殊亚群的乳腺癌有临床应用价值；但除了ER、PR 和HER-2 之外，没有找到可以预测具体治疗方案的生物标志物。

# 21.  三阴性乳腺癌辅助化疗方案的选择  
由于三阴性乳腺癌预后相对差，没有明确的可治疗性靶点，缺少内分泌或靶向治疗机会，因此辅助化疗更为重要。NSABP B-15 和CALGB 9344 研究验证了AC 或TC 对如淋巴结阴性、肿瘤小于$2\mathrm{cm}$、未见脉管癌栓等低危的三阴性患者有效性，奠定了 4  周期 AC  或 TC  方案对这类患 者作用。BCIRG005 研究验证了AC-T 方案对淋巴结阳性的三阴性患者的有效性，但是在选择紫杉类周疗或三周疗上仍存在争议。

因此，对于三阴性乳腺癌患者，如若不存在高危因素，可以考虑短程化疗；而如存在高危因素，建议AC-T方案。综合疗效获益与不良反应预防而言，可以考虑使用含紫杉和蒽环的剂量密度方案。推荐在第一周期进行标准剂量的 3  周疗法，如患者耐受程度佳，下一周期可考虑改 为剂量密集方案。  
# 22. 卡培他滨在三阴性乳腺癌辅助化疗中的 地位  
对大部分三阴性乳腺癌辅助治疗而言，AC-T 的方案仍然是现行的标准治疗，剂量密集化疗可能会进一步提高患者的总生存。在此基础上，卡培他滨的地位仍需要更多的证据支持。在FinXX 研究中，多西他赛、表柔比星和环磷酰胺方案中加入卡培他滨并未提高整体人群的无复发生存率。但亚组分析的结果提示，三阴性乳腺癌患者能从卡培他滨的治疗中获益。另一项来自亚洲的研究同样提示了卡培他滨在辅助治疗中的地位，试验纳入910 例接受新辅助治疗后未达到pCR 的HER-2 阴性的患者，一组患者接受标准辅助治疗，另一组患者在标准治疗基础上联合卡培他滨的治疗，初步的随访结果提示，在标准治疗基础上增加卡培他滨能够改善HER-2 阴性、非pCR 患者的DFS（$74.1\%$ vs$67.7\%$$P<0.005$）和5年OS（$89.2\%$ vs $83.9\%$$P<0.01$）。这提示了，对于新辅助治疗后非pCR 的HER-2 阴性患者卡培他滨可能在TNBC 的辅助治疗中能占有一席之地。  
因此综合考虑，我们认为在整体人群中应用卡培他滨的辅助治疗证据仍不够充分。但新辅助化疗后非pCR 的三阴性乳腺癌患者，可考虑卡培他滨的治疗，理由如下：$\textcircled{1}$三阴性乳腺癌患者新辅助化疗后没有辅助治疗。$\textcircled{2}$三阴性乳腺癌患者复发高峰在术后的3 年内，加用化疗药物可能获益。 $\textcircled{3}$ 三阴性乳腺癌对化疗敏感。 $\textcircled{4}$ 卡培他滨治疗的安 全性可控、口服剂型方便、患者依从性好。  
#  铂类药物在三阴性乳腺癌中的地位  
新辅助治疗领域，在蒽环、紫杉类药物联合或序贯的治疗方案基础上加入铂类药物可以进一步提高TNBC 患者的pCR。来自德国的GeparSixto Ⅱ期临床研究和CALGB 40603 研究均验证了在标准治疗的基础上增加卡铂显著改善了患者的pCR。此外，一些小样本的临床研究结果显示，BRCA1/2  突变携带者对单药铂类新辅助治疗表现出令人难 以置信的pCR，为$61\%\sim90\%$。  
在辅助治疗领域，AC-T 的方案仍然是大部分三阴性乳腺癌患者的标准治疗，而铂类药物在TNBC 中的地位如何？目前正在进行的NRG BR003 研究在标准化疗AC-T 的基础上，加入卡铂的治疗，可能会解答这一问题，目前仍没有证据。  
解救治疗领域，铂类可能是三阴性乳腺癌患者的有效选择，特别是BRCA1/2 突变的患者。TNT 研究表明在三  
阴性乳腺癌中卡铂的有效率、PFS 及OS 与多西他赛相似。但在BRCA1/2 突变携带者中，卡铂显示出更高的有效率，分别为$68\%$ 和$33.3\%$；更长的中位PFS，分别为6.8 个月和3.1 个月。在联合治疗方面，由国内学者牵头的CBCSG 006 研究，对比了吉西他滨联合顺铂（GP）和吉西他滨联合紫杉醇（GT）方案一线治疗晚期三阴性乳腺癌的疗效与安全性，结果显示含铂类药物的方案在三阴性人群均具有更好的疗效，PFS 由6.47 个月提高到7.73 个月。此外，通过总结2005—2013 年军事医学科学院附属医院乳腺肿瘤科经治186 例HER-2 阴性MBC 患者接受吉西他滨联合顺铂（GP）和长春瑞滨联合顺铂（NP）方案的安全性和有效性，结果显示GP 和NP 在HER-2 阴性乳腺癌患者中疗效相当，不良反应均可耐受。提示NP 也可能是TNBC 晚期一线治疗的可选方案。  
总结来说，化疗仍是TNBC 患者的主要治疗手段，在解救治疗领域含铂类药物的方案可成为TNBC 患者的首选治疗，特别是那些BRCA1/2 突变的患者；在（新）辅助治疗领域，蒽环、紫杉联合方案仍是标准，鼓励更多含铂类药物的（新）辅助治疗临床研究的开展，来解答这一问题。  
# 24.  蒽环类药物在HER-2 阳性乳腺癌  辅助化疗中的地位  
NSABP B-31 研究和HERA 研究奠定了化疗联合1 年曲妥珠单抗在早期乳腺癌中的地位。按照既往多个指南的推荐，HER-2 阳性患者目前常用的化疗联合靶向治疗方案有AC-TH 和TCbH。既往相对低危或具有心脏风险的患者更倾向于选择 TCbH  方案，而高危患者更倾向于选择 AC-TH 方案。近期公布的BCIRG 006 的10 年随访结果，再次将AC-TH 方案的临床价值以及蒽环类药物在HER-2阳性乳腺癌中的治疗价值推向讨论热点。  
最新研究结果TCbH 组相对于AC-TH 组DFS 事件数仅多10 例，AC-TH 组没有比TCbH 组更好的趋势。包括在淋巴结阳性亚组，甚至是淋巴结≥4 个阳性亚组都没有看到AC-TH 在复发及生存方面更好的优势。同时在不良事件方面，AC-TH 组的充血性心力衰竭事件数约为TCbH 组的5 倍（21 例 vs 4 例），并且AC-TH 组的白血病事件、持续性左心室射血分数（LVEF）下降$>10\%$ 的事件更多。  
因此，对于HER-2 阳性患者，AC-TH 以及TCbH 都是可选方案，AC-TH 方案更具临床实际的可操作性。但对于需考虑心脏毒性风险的患者而言，TCbH 方案可作为首选。  
# 25. 复发转移性乳腺癌解救化疗的时机  
晚期乳腺癌的治疗目的在于延长生存、提高生活质量，因而治疗过程中平衡治疗获益和毒性尤为重要。细胞毒类化疗药物对肿瘤疗效明确，但毒性较大，因而需要合理掌握用药时机。  
对于新确诊为复发转移性乳腺癌，需要评估患者的病变范围，激素受体和HER-2 状态以及对化疗的耐受程度等。对于 HR  阳性、 HER-2  阴性、无症状或肿瘤负荷不 大的内脏转移患者，首选内分泌治疗。HER-2 阳性晚期乳腺癌患者一般首选曲妥珠单抗联合化疗。激素受体阴性[hormone receptor-negative，HR（—）] 需快速缓解肿瘤负荷、有症状的内脏转移或激素受体阳性但辅助内分泌治疗短期内复发的患者考虑先行化疗。  
对于上一轮解救治疗失败的晚期乳腺癌患者，同样应评估患者疾病情况和耐受程度。若患者近期进展迅速，考虑改化疗。但若患者近期连续2 轮以上化疗失败，应考虑暂停化疗，给予内分泌治疗或靶向治疗，甚至对于某些 轻微进展患者可以考虑暂停治疗，休息1 个月后再进行治疗，使其能够从连续的化疗毒性中恢复。  
#  晚期乳腺癌的维持治疗  
对于复发转移后的乳腺癌患者，我们的治疗目的是延年益寿，争取更长的生存时间和更高的生活质量。此时的治疗应该兼顾患者一般情况、肿瘤负荷、分子分型、既往治疗情况、患者个人意愿、家庭社会因素等情况，为其制定“量体裁衣”式的个体化解救治疗方案。  
临床研究中为了考核某一药物或方案的最佳疗效持续时间和最大耐受剂量，往往要求治疗至疾病进展，或出现不能耐受的不良反应。但临床实践毕竟与临床研究的目的不同，实际治疗中往往会出现在疾病没有进展时，患者因为不能耐受联合化疗的药物的不良反应而提前中断化疗。这种辛苦的化疗方式对患者生活质量影响很大，因此有学者提出维持治疗的理念，即一线联合化疗进行诱导治疗，后续单药的“维持治疗”。现有的证据也表明延长化疗的获益，2011 年一项纳入11 项晚期乳腺癌一线随机对照临床试验的荟萃分析显示延长化疗的确改善了患者PFS 及OS，这也进一步证明了维持治疗有效性。  
在选择维持化疗的药物时，一般考虑：单药有效率高、不良反应相对较轻、使用方便。口服卡培他滨可作为优先选择， （TX/NX）化疗方案，$6\sim8$ 周期达到最佳疗效后考虑停用静脉化疗药物，保留单用口服药物的维持治疗。卡培他滨耐药后，单药的多西他赛、紫杉醇、长春瑞滨等药物可以 作为临床选择。此外依托泊苷单药也有一定的有效率，不失为维持治疗的选择。  
#  化疗管理：心脏保护  
乳腺癌化疗药物对心脏的影响主要来源于蒽环类药物，蒽环类药物导致的心脏毒性往往是进展性和不可逆的，首次使用蒽环类药物就能对心脏造成损伤。随着曲妥珠单抗在HER-2 阳性乳腺癌辅助治疗中地位的确立，二者联合应用对心脏毒性的影响也越来越受到关注，据一项纳入45537 例老年（$.67\sim94$ 岁）乳腺癌患者的回顾性研究显示，蒽环类药物辅助化疗基础上加用曲妥珠单抗，心力衰竭/ 心肌病风险增加，研究者认为，对老年乳腺癌患者，曲妥珠单抗联合蒽环类药物化疗后，增加了心脏风险，且超出临床试验数据。  
因此，对于早期低危乳腺癌患者，尤其是65 岁以上或存在心脏风险的患者，应尽量避免使用蒽环类药物，早期可用TC 方案辅助化疗。而对于需要应用蒽环类药物的患

#  化疗管理：骨髓保护  
骨髓功能抑制是化疗常见的非特异性毒性，也是影响化疗疗程及剂量的关键因素。大多患者在联合化疗方 案用药后 $1\sim2$  周出现白细胞下降， $10\sim14$  日达到最低 点，$3\sim4$ 周时恢复正常。不同的方案骨髓抑制发生的程度也不尽相同，TAC、AT、剂量密集型ddAC-ddT 等方案有高度风险引起粒细胞缺乏伴发热（FN），而AC、EC 等方案风险相对较低。化疗前应评估FN 发生风险，根据化疗方案、患者危险因素、治疗目的采取相应的防护 措施。  
聚乙二醇化重组人粒细胞刺激因子（PEG-rhG-CSF）是重组人粒细胞集落刺激因子（rhG-CSF）的长效剂型，已有临床研究证实，与安慰剂相比，预防性应用 PEG-rhG-CSF，可以显著降低患者FN 的发生率、缩短住院时间、减少静脉抗菌药物的使用。而应用PEG-rhG-CSF 治疗的患者出现Ⅲ、Ⅳ度中性粒细胞减少的平均持续时间与应用rhG-CSF 治疗者无显著性差异。  
因此，对于接受高FN 风险化疗方案的患者，无论治疗目的是治愈、延长生存期还是改善疾病相关症状，均建议其预防性使用PEG-rhG-CSF。对于低度FN 风险的患者，不予常规预防性使用PEG-rhG-CSF，但若在第一个化疗周期中患者发生FN 或剂量限制性中性粒细胞缺乏症，则下一个化疗周期可以考虑预防性使用PEG-rhG-CSF。剂量方面，PEG-rhG-CSF 每个周期推荐使用 6mg，并于化疗给药结束后一天给予。对于第一周期应用后粒细胞升高过于明显的患者，可在后续治疗过程中减量至$3\mathrm{mg}$。
