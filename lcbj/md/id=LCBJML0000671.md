# 高同型半胱氨酸血症造成大血管损害的人群研究  
多项研究已证明Hcy 水平升高与冠状动脉疾病、脑卒中和深静脉血栓密切相关。本部分主要基于我们前期的相关研究结果，从Hcy 水平升高与腹主动脉瘤（abdominal aortic aneurysm，AAA）和主动脉夹层（aortic dissection，AD）两种大血管疾病关系的人群研究入手，探讨Hcy 水平升高对大血管造成的损害及潜在的作用机制。  
# 18. 高同型半胱氨酸血症与腹主动脉瘤关系的 人群研究  
在年龄$ >65$  岁的男性中，AAA 发病率为$ 4.3\%\sim8.8\%$ ，而其破裂导致的死亡占此年龄男性总死亡的$ 1\%\sim2\%$ 。流行病学研究已经发现了一些和AAA 相关的危险因素，包括年龄增加、吸烟、男性和AAA 家族史等。除此之外，多项研究证实Hcy 水平升高是AAA 的危险因素，Hcy 浓度升高在AAA 的发生、发展乃至破裂中扮演着重要角色。  
1）Hcy 水平升高同AAA 发生相关性的人群研究  
早期的临床研究主要关注AAA 患者Hcy 升高的现况。Brunelli 等早在2000 年就做了一项AAA 患者轻度高同型半胱氨酸血症发生率的研究，结果发现试验组有$ 48\%$  患者的Hcy 水平升高（> 15.5mol/L），而对照组的Hcy 升高比例仅为$ 5\%$ ，因此提出Hcy 可能是导致部分人发生AAA 的可能原因。2001 年Caldwell 等的研究证实了Brunelli 的研究结果，并发现两组在年龄、叶酸水平、维生素$ \mathrm{B}_{12}$     水平、肾功能等方面无统计学差异。2003 年Spark 等的研究显示$ 53\%$  的AAA 患者伴有Hcy 水平的升高（\langle \Omega > 16\, \mu\mathrm{mol/L} \rangle），而对照组仅为$ 7\%$ ，Spark 的研究中所有人的叶酸和维生素水平均在正常范围内。Warsi 等的研究发现高达$ 68\%$  的病例组伴有Hcy 水平升高（> 15.5\, \mathrm{\mu mol/L}）而对照组的发生率只有$ 6\%$ ，从而提出Hcy 水平升高同AAA 具有更高的关联性。2013 年的大样本研究进一步证实Hcy 水平升高是AAA的危险因素（$ O R{=}1.45$ ；$ 95\%C I$ ：$ 1.10\sim1.91$ ）。我们的相关研究亦证明，在中国汉族人群中，Hcy 升高与AAA 密切相关。然而Peeters、Lindqvist 等发表的研究提出：低维生素$ \mathrm{B_{6}}$     或维生素$ \mathrm{B}_{12}$    ，而并非Hcy 水平升高，是AAA 的危险因素。因而，Hcy 水平升高和AAA 的相关性尚需进一步研究和探讨。  
（2）Hcy 水平升高与AAA 发展相关性的临床研究  
大量研究证实Hcy 升高和AAA 发生密切相关。Brunelli 等的研究认为Hcy 升高亚组AAA 直径明显增大，Soﬁ等的研究也印证了Brunelli 等的观点；同样，最近的大样本资料显示Hcy升高的研究对象比Hcy 正常人群的平均主动脉直径大$ 0.41\mathrm{mm}
$ （$ 95\%C I$ ：$ 0.06\sim0.77)$ ），Hcy 每升高$ 5mu\mathrm{mol}/\mathrm{L}$    ，主动脉直径增加
$ 0.15\mathrm{mm}$    $ (95\%C I;\ 0.01\sim0.28)$ ）。然而，另外3 项较小样本量的研究没有发现Hcy 升高对AAA 直径的明显影响；同时2011 年Laughlin 等发表的大样本横断面队列研究，共调查了1926 名参与者，发现Hcy 水平和主动脉直径相关性未见统计学差异。我们基于CSPPT 研究人群的横断面研究表明，发现Hcy 水平和主动脉直径存在显著相关性。由此可见，Hcy 水平和AAA 及腹主动脉直径的关联，还需要更多的临床流行病学研究予以考察和证实。  
另外，高Hcy 浓度与AAA 扩张速率是否相关一直是未解决的问题。虽然Lindholt 研究中发现高Hcy 与AAA 扩张速率无关，但2007 年Halazun 等对影响AAA 生长的各项因素进行分析后发现Hcy 是影响AAA 生长的唯一因素，并且二者呈正相关（$ '_{r=0.28}$    ，$ {\it P}{=}0.003$ ）这项结果有助于我们更深入了解AAA 发病过程，以及在AAA 治疗中早期干预Hcy 升高的意义。2008 年Gottsater 等的研究观察了他汀类治疗AAA 患者Hcy 等的水平，发现他汀类治疗者中有更低的Hcy 水平，降低Hcy 水平可能是他汀类治  
疗AAA 的途径之一。Wald 等的研究证实通过每天摄入$ 0.8\mathrm{ms}$     的叶酸能够减低血清 Hcy  水平 $ 3mu\mathrm{mol}/\mathrm{L}$     ，从而使冠心病风险降低 $ 16\%$ ，深静脉血栓风险降低$ 25\%$ ，卒中风险降低$ 24\%$ 。CSPPT 研究亦证实摄入叶酸以降低血清Hcy 水平来显著降低卒中发生风险。由此可以设想降低Hcy 水平是否可以作为治疗AAA 的一种选择。对于降低AAA 患者的Hcy 水平能否使其在减少并发症、延长生存时间等方面获益，尚需要临床试验进行验证。  
3）Hcy 水平升高与AAA 发生发展因果关系的人群研究  
目前尚没有试验能提供足够的证据来证实Hcy 升高和AAA的因果关联，为了克服Hcy 升高与AAA 反向因果关系的问题，一些研究也评估了AAA 患者的MTHFR 基因型。Strauss 等确认了基因型的关联性，他的研究中，有$ 66\%$  的AAA 患者（$ \backslash n{=}63$ ）具有T 等位基因，而对照组仅有$ 35\%$ （$ n{=}75.$ ）；回归分析表明，杂合子突变的患者比C 等位基因纯合子患者患AAA 的风险增加4.4 倍。Peeters 等研究发现：在85 例AAA 患者中，有15 例是677C 到T 突变的纯合子，而对照组86 例中只有7 例，OR 值是2.1。另外两项Meta 分析也证实了AAA 患者具有更高的TT 等位基因频率。但是，另外2 项较大样本的研究并未显示基因C677T突变和AAA 之间存在独立的相关性。我们新近一项纳入10 123例研究对象的Meta 分析研究显示，在年轻患者、非吸烟患者与更大AAA 直径患者，MTHFR 基因突变与AAA 显著相关。因此Hcy 升高与AAA 的因果关系仍需更多研究予以揭示。  
# 19. 高同型半胱氨酸血症与主动脉夹层关系的人群研究  
AD 的发病机制仍然未明，相关流行病学研究发现了一些和主动脉夹层相关的危险因素，包括高血压、男性、滥用可卡因和主动脉壁结构异常等。除此之外，一些研究证实Hcy 升高与AD密切相关。  
Pezzini 等首次报道自发性颅内动脉夹层组患者Hcy 水平明显高于对照组，以高于$ 12mu\mathrm{mol}/\mathrm{L}$     作为定义Hcy 升高，两组Hcy发生率分别为$ 64.0\%$  和$ 13.9\%$ 。Benninger 等的研究也认为轻度Hcy 升高是自发性脑动脉夹层的危险因素。Giusti 等提出在Marfan 综合征患者中，Hcy 升高可能是主动脉夹层的危险因素。他们的研究将Marfan 综合征患者根据其心脏血管症状的严重程度分为3 组：A 组没有受累；B 组轻度受累；C 组主动脉扩张或主动脉夹层。C 组的Hcy 浓度高于B 组；在C 组中有主动脉夹层患者Hcy 浓度高于无夹层患者。多元回归分析显示：Marfan综合征患者中，严重的心血管症状尤其是主动脉夹层与Hcy 浓度有关。Hcy 升高是自发性颅内动脉夹层和Marfan 综合征患者主动脉夹层的危险因素，为Hcy 升高是否为主动脉夹层的危险因素提供了线索。  
2005 年Takagi 等提出假设：Hcy 升高是主动脉夹层的危险因素，同年的研究表明主动脉夹层合并主动脉瘤组血浆Hcy 水平较对照组增高，分别为（$ (19.75\pm12.77)$ ）$ mu\mathrm{mol}/\mathrm{L}$     和（$ \left(13.43\pm5.15\right)$ ）$ mu\mathrm{mol}/\mathrm{L}$      
$ \!\!\left(P<0.001\right)$ ），病例组高血浆同型半胱氨酸患者的比率高于对照组，分别为$ 67.4\%$  和$ 38.5\%$ （$ \left(P\mathrm{~<~}0.01\right)$ ）。Sbarouni 等的研究表明，急性主动脉夹层组的患者与慢性动脉瘤患者及健康对照组相比具有更高的Hcy 水平和更低的叶酸水平，3 组Hcy 水平分别为$ (22\pm14)\ \ {mu\mathrm{mol/L}},\ \ (17\pm5)\ \ {mu\mathrm{mol/L}},\ \ (14\pm4)\ \ {mu\mathrm{mol/L}}$     ；叶酸水平 分别为$ (6.1\pm3.3)~{mu\mathrm{mol/L}},(8.5\pm2.8)~{mu\mathrm{mol/L}},(8.0\pm2.7)~{mu\mathrm{mol/L}}_{\odot}$    。  
我们的研究分析了2011 年10 月至2013 年1 月符合纳入标准的111 例受试者。急性主动脉夹层组（A 组）45 例，平均年龄（$ (57.62\pm9.74)$ ）岁，其中男35 例，女10 例；非急性主动脉夹层组（B 组）16 例，平均年龄（$ (53.88\pm8.12)$ ）岁，其中男12 例，女4 例，对照组为50 例同期体检者（C 组），平均年龄（$ (55.62\pm4.07)$ ）岁，其中男41 例，女9 例。结果显示3 组血浆Hcy 水平分别为（$ (17.85\pm8.65)$ ）$ mu\mathrm{mol}/\mathrm{L}$    、（$ (22.15\pm10.45)$ ）$ mu\mathrm{mol}/\mathrm{L}$     和（$ 12.74\pm3.32)$ ）$ mu\mathrm{mol}/\mathrm{L}$    （$ \textstyle P<0.001.$ ）；高同型半胱氨酸血症发生率3 组分别为$ 57.8\%$ 、$ 68.8\%$  和$ 30.0\%$ （$ \textstyle(P<0.001]$ ）。多元Logistic 回归分析显示Hcy 升高较Hcy 正常人群患AD 的风险显著增加（$ O R{=}4.78$ ；$ 95\%C I$ ：$ 1.21\sim18.92$ ；$ P<0.05$ ）。不同于Sbarouni 等的研究结果，我们团队研究发现非急性夹层组Hcy 水平最高并且Hcy 升高是AD 的独立危险因素。随后我们的研究增大了样本量，未改变Hcy 升高是AD 独立危险因素的结论，提示降低Hcy 水平有降低主动脉夹层患病风险的可能性。另一项研究显示高血压合并主动脉夹层组 Hcy  水平显著高于高血压组和健康  
组，高血压合并主动脉夹层组和高血压组叶酸水平较健康组显著降低，高血压合并主动脉夹层组和单纯高血压组MTHFR CT 和TT  基因型频率显著高于健康组，提示 H  型高血压与主动脉夹层 的发生发展密切相关，需要更多研究来揭示H 型高血压和主动脉夹层的关系。  
# 20. 高同型半胱氨酸血症造成大血管损害的可能机制  
传统观点认为Hcy 升高造成大血管损害是因为Hcy 导致血管壁动脉粥样硬化作用，而近年的研究表明Hcy 升高患者可能通过以下4 个方面在大血管疾病的发生发展中产生作用。  
1）遗传易感性  
编码MTHFR 基因的普遍突变型（C677T）能使MTHFR 活性升高从而提高Hcy 浓度。虽然突变率存在着种群差异，但是普通人群中约$ 10\%$  的人是T 等位基因的纯合子，$ 43\%$  的人是T 等位基因杂合子。T 等位基因纯合子人群比C 等位纯合子人群Hcy水平平均增加约$ 2.5mu\mathrm{mol}/\mathrm{L}$    ，导致T 等位基因纯合子人群患心血管疾病的风险增加。多项研究表明，AAA 患者TT 等位基因频率更高。Warsi 等研究发现实验组（$ 'n{=}38$ ）4 个患者有AAA 家族史，对照组（$ 'n{=}36$ ）没有一个患者有AAA 家族史，因此提出AAA的家族遗传可能由于Hcy 升高的遗传性引起的观点。  
# 2）促动脉粥样硬化  
虽然现在认为AAA 以及AD 和动脉粥样硬化性疾病的发病机制不完全相同，但动脉粥样硬化过程可能是AAA 和AD 的发病过程之一。Hcy 升高可通过损伤血管内皮，促进脂蛋白氧化、促进脂质沉积于动脉壁，还可改变动脉壁糖蛋白分子纤维化结构，促进斑块钙化形成等作用促进动脉粥样硬化。2009 年Van Campenhout 等证实了Hcy 在AAA 和周围动脉疾病动脉钙化中的作用，同时其体外实验证实了高浓度Hcy 促进骨细胞分化，从而确认了Hcy 在动脉钙化中扮演重要角色。  
# （3）细胞外基质成分重塑  
Hcy 水平升高能够增强蛋白成分降解。研究表明Hcy 升高能引起动脉平滑肌细胞合成丝氨酸蛋白酶增加，通过细胞外基质退化和向平滑肌细胞释放弹力肽细胞趋化因子引起弹力蛋白降解。研究证实MMPs 参与了AAA 形成的病理过程，Hcy 升高在增强丝氨酸蛋白酶合成的同时激活MMPs，尤其是MMP-2 和MMP-9， 进一步引起AAA 中膜的弹力纤维和胶原纤维的降解。动物研究发现，高半胱氨酸血症可能通过激活MMPs 引起弹力纤维降解，产生主动脉壁明显的细胞外基质重塑。进一步研究发现，Hcy 可能会直接影响原纤维蛋白-1 或胶原蛋白通过二硫键交换分子间或分子内的二硫键，或与自由巯基基团结合，从而影响与硫酸根高亲和力的蛋白酶作用，如MMP-3。另外，Hcy 升高时血管内皮细胞也可能参与内皮下动脉弹性结构的退化，这种  
作用已经在Hcy 病理水平（$ \mathrm{\Omega}^{\prime}\mathrm{100}mu\mathrm{mol}/\mathrm{L}$     和$ 250mu\mathrm{mol}/\mathrm{L}$    ）时明确观察到。很有趣的现象是高 Hcy  实验浓度在高摩尔比［ \mathrm{[Hcy:MMP\ 为\ (103\sim 104):1]} ］时抑制 MMP  的活性。低 Hcy  浓度（同临 床上高同型半胱氨酸浓度相匹配的浓度）被发现增加MMP-2和MMP-9 的表达水平，而激活MMP-2 前体发生在低摩尔比时（$ \mathrm{(Hcy:MMP)}$     为$ 10:1$ ）。但是目前没有体内数据支持Hcy 水平同MMPs 组织活性相关。  
# （4）增强慢性炎症反应  
有研究认为Hcy 水平升高能够通过活化NADPH 氧化酶的途径，诱导ROS 的产生及分泌趋化因子，同时能引起多种炎症递质如白细胞介素、肿瘤坏死因子等的释放，引起单核细胞、中性粒细胞对内皮细胞的黏附并且进入到内皮下层。孔炜教授团队的研究首次证实Hcy 水平升高促进AAA 和AD 发生的机制，可能与Hcy 促进外膜成纤维细胞NOX4 表达上调，从而促进成纤维细胞向肌成纤维细胞转化及之后的IL-6 产生、MCP-1 产生，最终促进外膜炎症有关。但2009 年Arapoglou 等观察了89 例AAA患者瘤体组织标本，发现Hcy 水平同AAA 组织炎症程度没有相关性。上述结果表明，Hcy 升高同AAA 以及AD 的具体作用机制仍然存在争议，需要更多的研究予以揭示。  
小结与展望：目前研究普遍认为Hcy 水平升高与AAA 密切相关，但现有证据不足以说明Hcy 升高是AAA 的病因，同时Hcy 升高是否促进AAA 的发展乃至破裂具有较大的争议，尚需更多的相关研究进一步探讨和解析。  
多项研究已经证实降低Hcy 水平能显著降低心脑血管事件，尤其是脑卒中风险，但是目前国内外尚未见针对伴有Hcy 升高的主动脉疾病患者降低Hcy 水平能否在减少并发症、延长生存时间等方面获益的相关报道，因而是未来该领域值得大家关注和推进的一个重要研究方向。
