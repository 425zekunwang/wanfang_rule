# 硬核白内障  
#  年龄是影响晶状体核硬度的主要因素  
硬度是晶状体的一种生理特性，影响晶状体混浊及硬度的因素很多，这与晶状体纤维代谢特点有关。晶状体纤维终生不断 生长，老纤维逐渐被压缩进入晶状体中心形成成人核，随着年龄的增加，密集的晶状体纤维分子间二硫键发生交联，压缩形成$ \gamma-$  晶状体蛋白，使晶状体不溶性晶状体蛋白量逐渐增加。核的硬度随年龄的增大而增加，但增加的程度不均一，且同一年龄组中核硬度的变异程度较大。
