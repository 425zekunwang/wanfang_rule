# 婴儿期癫痫性痉挛的临床表现、  
检查及其诊断  
# 16.West 综合征典型表现为三联征  
2010 年国际抗癫痫联盟（International League Against Epilepsy，ILAE）分类工作汇报中，癫痫性痉挛已被明确为一种独立的发作类型。该类型发作表现为突发的、主要累及躯干中轴和双侧肢体近端肌肉的强直性收缩，历时$0.2\sim2.0$ 秒，突发突止，发作可成串并伴有短时意识丧失。在临床上，痉挛发作主要分为屈肌痉挛（多见）和伸肌痉挛。IS 为特定在婴儿期起病的痉挛发作，常表现为成串发作的点头拥抱样动作，也偶见单次发作。在所有癫痫性痉挛中，West 综合征最为常见，该病以痉挛发作、脑电图高度失律、精神运动发育迟滞的三联征为主要特点，其中精神运动发育迟滞在发病前不一定存在。IS 还包括其他的发作形式或者变异：$\textcircled{1}$单个痉挛变异型（infantile spasms single-spasm variant ， ISSV)  是 IS  比较少见的类型，其 痉挛发作一般单独出现（如果1 分钟内仅有1 次痉挛发作可视为单发痉挛）；$\textcircled{2}$不伴有痉挛发作的高度失律，仅有脑电图高度失律而没有临床发作；$\textcircled{3}$有癫痫性痉挛发作而没有脑电图高度失律。  
# 17. 脑电图在婴儿期癫痫性痉挛诊断中作用重大  
脑电图(electroencephalogram，EEG) 是诊断IS 和评估其预后的重要手段之一。IS 的典型脑电图表现为高度失律，但是痉挛发作前和发作期有所不同。  
（1）痉挛发作前脑电图：Endo 等研究了25 名症状性IS 婴儿在出现痉挛发作以前的脑电图，他们发现最早出现脑电图痫样放电的部位为枕叶（14/25），其次为多部位（7/25）和枕叶以外部位（4/25），其中枕叶患儿多是早产儿（12/14），且除1 例外均有侧脑室旁白质软化（periventricular leukomalacia，PVL）；而多部位者都是足月儿或伴有缺氧损伤的近足月儿。既往的研究也提示，早产儿特别是伴有PVL 而且发展为IS 的婴儿在其早期的脑电图中容易在顶枕部有明显的不规律棘慢波或多发的棘慢波。Suzuki 等回顾性分析了19 例有PVL 的婴儿，发现有后头部癫痫样放电改变的10 例婴儿中7 例发展为婴儿痉挛症，而且多数伴有PVL 的婴儿痉挛症的早产儿在纠正年龄 3  个月时脑电图就有顶枕 区占优势的痫样放电。随着近年来医学影像学技术的不断发展，更多的证据表明后头部皮质的功能异常和器质性损伤与婴儿痉挛症的发生密切相关。应用PET 发现，隐源性婴儿痉挛症婴儿顶枕颞区域的代谢明显减低。Koo 等分析了伴有脑局部结构损伤的婴儿痉挛症患儿临床资料后指出，有限局性枕部损伤的患儿出现强 直痉挛的时间要早于其他部位损伤的患儿。  
（2）痉挛发作间期脑电图：最具特征性的脑电图表现为典型的高度失律，高度失律定义为“随意高波幅慢波和棘波”，这种棘波在持续时间和部位上不固定，

$\textcircled{1}$半球间同步化增加的高度失律：多灶性棘波和尖波以及弥漫的不同步慢波活动被明显的半球间同步和对称的活动替代或混插，此形式不持续。同步化高度失律大脑两侧半球高度失律波型$50\%$以上同步出现。  
$\textcircled{2}$不对称型高度失律：指大脑半球两侧相对应脑区或两侧大脑半球之间，脑电波幅、频率或棘波数量差别超过$50\%$。也称为半球高度失律或单侧高度失律。不对称高度失律与潜在的脑部结构异常有关，既可出现于异常半球，也可出现于正常半球。  
$\textcircled{3}$ 伴持续局限异常的高度失律： 明显的局限性棘波 或尖波复合于典型的高度失律背景上，有时出现局限性电发作改变。 成功后仍持续存在。

$\mathcal{S}$伴有小棘波或尖波的高度失律：是一种少见的变异，为高波幅、不同步或同步性慢波活动伴有小棘波或尖波。  
（3）痉挛发作期脑电图：发作期脑电图改变形式更加多样，1979 年Kellaway 等首次明确了11 种发作期脑电图形式：$\textcircled{1}$全面性以额区为著的高波幅慢波继以波幅衰减； $\textcircled{2}$ 全面性尖慢复合波； $\textcircled{3}$ 全面性尖慢复合波继 以波幅衰减；$\textcircled{4}$仅为波幅衰减；$\mathcal{S}$仅为一过性全面性慢波；$\textcircled{6}$复合快节律的波幅衰减；$\textcircled{\scriptsize{2}}$一过性全面性慢波继以复合快节律的波幅衰减；$\circledast$伴有节律性慢活动的波幅衰减； $\textcircled{9}$ 仅为快活动； $\circledcirc$ 全面性尖慢复合波继以复合快 节律的波幅衰减；$@1\!)$复合快节律的波幅衰减继以节律性慢活动。此后，国内外许多学者通过对发作期视频脑电图的研究证实了上述表现，同时发现痉挛发作时以全导同步中至高幅快波节律中至长程出现，电压抑制以 及中央、顶区单个正性慢波成分最常见。在丛集性痉挛发作中，可见到多种形式脑电图改变，提示IS 是一种多灶性癫痫。  
近年来随着脑电图技术的发展，人们借助高科技手段对痉挛发作期的各种频率的脑电波形变化进行了更为细致的分析，同时进一步解释了痉挛发作的可能病理机制：$\textcircled{1}$高波幅慢波（HVSs）：对发作期HVSs 的研究显示，固定、不对称、不同步的HVSs 多见于症状性West 综合征患者，但在隐源性患者中也可见到，故HVSs 和病因学的关系并不确定。另外，对HVSs 的形态学研究显示，慢波相对负相主要位于后头部，正相位于额、顶或颞部，慢波前的尖波也多分布于枕区，认为其位相的分布变化与产生慢波的新皮质病理部位有关。发作期HVSs 的产生机制和痉挛的产生密切相关，痉挛的出现与皮质和皮质下结构复杂而异常的相互作用有关。目前多认为，棘波多产生于皮质，而高幅慢波的产生与皮质下结构有关。$\textcircled{2}$$\upbeta$ 节律活动：Panzica 等通过对23 例West 综合征患儿2 年的脑电图随访指出，在多数患者中可检测到一侧性与痉挛相关的β 节律活动，这种发作期的电活动与反复顽固性发作显著相关，是痫性发作不良结果的一个危险因素，但还不能说是恶性West 综合征的突出特点，因为在很多后来发作彻底缓解和临床预后较好的婴儿中也存在此类快活动。这种发作期β 节律活动与早期痉挛和晚发痫性发作相关，它在一侧持续存在被认为是那些对早期治疗反应差的婴儿存在局限性皮质功能不良的指征，这些患儿通常发展为严重的癫痫。$\textcircled{3}$γ 节律：数字化脑电图技术的发展使人们得以发现极快的快活动，这在以前的走纸脑电图上是无法发现的。频率$>40\mathrm{Hz}$的$\upgamma$ 节律最早在有局部起源的癫痫发作的患者颅内电极检测中记录到，最近在与痉挛相关的头皮脑电图中也被检测到。婴儿痉挛发作期脑电图显示 $\upbeta$  活动和继发的 HVS ， HVS 与临床发作相关，而发作期γ 节律与HVS 有密切的一过性关联。Inoue 等研究了15 例West 综合征患者发作期EEGγ 高峰的特征（高峰频率，强度，持续时间及成簇发作中每个部位的γ 高峰数量），指出在多数患者中可检出γ 高峰，而且在成簇发作的每次痉挛发作中γ 高峰形式相似，顶枕部γ 高峰较额—中央区强度强且时限长，发作期γ 高峰的频率高于发作终止时，而且与发作前出现的β 高峰到临床发作之间的潜伏期长

# 18. 一级或二级医疗等级单位的诊疗方案有待推广  
由于IS 表现为点头拥抱样动作，有些患儿早期发作幅度小，家长未予重视，加上一级或二级医疗单位专科医师经验有限，患儿往往得不到及时的诊治。早期干预IS 对改善患儿的预后有积极作用。随着科技发展，移动设备作为便携的拍摄终端，在一定程度上提高了IS 发作的早期诊断率，这有助于早发现、早干预、改善预后。  
EEG 检查对于诊断IS 至关重要，在实际工作中如可能，患儿应尽量行视频脑电图（VEEG）检查，EEG中应包括清醒期、睡眠期及觉醒过程中的脑电描记。然而，目前仍没有足够的数据能统计出不同病因应检查的EEG 的时长。24 小时 VEEG 最容易发现患儿的高度失律和痉挛发作。对于有相应设备的单位，VEEG 被推荐  
为最优医疗方案。有相应设备的医疗单位应对拟诊患儿行$3\sim12$ 小时VEEG 检查以帮助得到足够的数据明确诊断。 EEG  结合临床数据对于诊断有较高的价值。除非 临床及EEG 均不支持癫痫诊断，否则单纯的EEG 未描记到异常并不能作为癫痫的排除诊断标准。根据最新的推荐共识，所有怀疑IS 的患儿均应行头颅核磁共振成像（MRI）检查。头颅影像学检查对于诊断IS 不是必需的，但是有助于寻找病因以及指导下一步的检查，条件允许的医疗机构应尽可能在治疗开始前完善头颅MRI检查。对于首次头颅MRI 未发现异常或者其他检查没有发现病因的患儿必要时可重复头颅MRI 检查。  
很多代谢性病因所致癫痫也可表现为痉挛发作。常见的导致癫痫性痉挛的代谢性疾病包括生物素酰胺酶缺乏症、Menkes 病、有机酸尿症、氨基酸代谢病、线粒体呼吸链疾病。其中一些代谢性疾病如果早期发现并予以特定的干预，预后较好。如一些辅因子依赖性癫痫，补充辅因子如维生素$\mathrm{B_{6}}$、亚叶酸、生物素后也可产生较好的治疗效果。表2 列举了怀疑为代谢性疾病时应进行的检查，由于医疗条件所限，对于一级及二级医疗单位可行的检查包括血糖、血常规、肝功能、血氨、尿常规、 $\mathrm{pH}$ 、动脉血气分析、血浆电解质水平 $(\mathrm{Na}^{+},\ \mathrm{K}^{+},\ \mathrm{Cl}^{-}$、阴离子间隙）、脑脊液及血液乳酸分析、脑脊液葡萄糖( 与血中对比)。

# 19. 三级医疗等级单位的诊疗方案需要更加细化  
对高度怀疑基因缺陷或代谢障碍的患儿可行基因  
检查或代谢筛查。目前基因检查手段多种多样，主要根据怀疑的潜在病因不同选择不同的策略。例如，怀疑Dravet 综合征或者热性惊厥附加症的患儿可以采用SCN1A  基因的全基因测序； 多重配体依赖性探针扩增 （multiplex ligand-dependent probe amplification，MLPA)适用于怀疑Rett 综合征的患儿；染色体、寡核苷酸及单核苷酸多态性阵列分析适用于15q13.3 缺失综合征的患儿；对于怀疑21 三体综合征的患儿可以采用染色体核型分析；荧光原位杂交（ ﬂuorescence in situ  hybridization，FISH）适用于由染色体缺失引起的疾病如Miller-Dieker 综合征；甲基化分析适用于怀疑快乐木偶综合征的患儿；对于病因不明的婴幼儿可以选择全基因组二代测序。在精神运动发育方面，三级医疗单位应对患儿行发育评估检查。另外，早期的癫痫发作和孤独症行为及智力缺陷是有一定相关性的。因此，美国儿科学会也建议对于高危人群行孤独症行为筛查。
