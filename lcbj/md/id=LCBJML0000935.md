# 高度近视的眼底改变  
高度近视的眼底病变，包括视盘倾斜、近视弧、豹纹状眼底、视网膜色素上皮（retinal pigment epithelium，RPE）和脉络膜广泛或者局部的地图状萎缩、后巩膜葡萄肿及后期被称作Fuchs 斑的脉络膜新生血管（choroidal neo vascular iz ation，CNV）。高度近视眼也可合并漆裂纹、局限性视网膜下出血等。与正常眼比较，在同样视功能表现情况下，弥漫的眼底萎缩性改变在高度近视较常见。继发于长时间（大于5 年）脉络膜视网膜萎缩斑扩大的进行性视力下降可能是高度近视CNV 的特点。当然，一些伴有广泛萎缩和（或）后巩膜葡萄肿的高度近视也可因黄斑裂孔或者视网膜劈裂而影响视力。由于高度近视相关眼底并发症严重损害视力且治疗较正常眼更加困难、效果更差，早期发现和治疗可能发生的眼部病理损害在高度近视的治疗中尤为 重要。  
#  视盘表面及周围改变  
# （1）视盘  
典型高度近视眼视盘呈椭圆形，长轴垂直。高度近视眼底后极部向后突出，视神经斜入巩膜，视盘呈斜椭圆形。有研究表明，随着年龄的增长，视盘面积轻度增大后再轻度减小。  
（2）弧形斑  
高度近视经常出现弧形斑，或称为颞侧弧形斑，可于出生时即已存在，但至青春期更为明显。在视盘颞侧缘，有一白色边缘清晰的弧形斑，其中含脉络膜血管与色素。在这两个新月形弧形斑的边缘，还常有色素出现。近视弧形斑的产生可能与近视眼继续扩大的物理因素有关，在病程中RPE 与脉络膜未能到达视盘的颞侧缘。通常脉络膜比RPE 更接近视盘。近视弧形斑的宽窄变异较大，有时可达半个视盘直径。虽然多数弧形斑位于颞侧，但约有$10\%$ 的病例弧形斑环绕视盘呈环形，有的甚至侵入黄斑区。少见弧形斑位于视盘鼻侧，称为反常性近视弧形斑。更少见者为位于视盘下方合并眼底下部巩膜扩张，称为Fuchs 视盘下弧形斑。高度近视随着近视度数的增加，出现近视弧形斑的变化、出现眼底并发症已在临床常见，而超高度近视后巩膜葡萄肿的高度将明显影响近视弧形斑的变化（图4）。  
# 3 ）牵引性弧形斑  
由于视神经斜入巩膜，在视盘颞侧形成近视弧，而在鼻侧视网膜或视网膜脉络膜组织被牵拉，似长在视盘上，呈一处带红色或带棕色新月形区，使视盘鼻侧边界模糊，称牵引性弧形斑。其平面较颞侧近视弧为高。鼻侧视网膜血管绕弧形斑缘而出。  
# （4）视盘周围视网膜脱离  
在高度近视的情况下，被牵拉伸长的视网膜不足以适应局部扩张的巩膜（即巩膜葡萄肿），从而产生一个使视网膜感觉层与色素上皮分离的力，而后巩膜葡萄肿区域内视网膜脉络膜萎缩又可能削弱该两层组织之间的附着力，进而导致局限视网膜脱离。视网膜脱离一般局限于后巩膜葡萄肿区域内，脱离的视网膜长度确实比相应处与巩膜附着的 RPE  和内层脉络膜短。这提示视盘 周围视网膜脱离可能与视网膜脉络膜萎缩、漆裂纹、视网膜和视网膜下出血及CNV 一样，是后巩膜葡萄肿的另一并发症。  
# 10.  后巩膜葡萄肿  
高度近视，当眼球后部显著增长，后极部形成局限性巩膜扩张，即为后巩膜葡萄肿。其发病与眼球轴长和脉络膜视网膜萎缩密切相关。当病变累及黄斑时，常使中心视力减退。后巩膜葡萄肿可能早年已经存在，以后随年龄增大而逐渐加重。后巩膜葡萄肿的边缘可呈斜坡或陡峭。检眼镜下陡峭的边缘出现暗棕色的半月形线条，视网膜呈屈膝状爬出。有的后巩膜葡萄肿边缘似圆嵴，其上可见视网膜血管。有时后巩膜葡萄肿有几条暗线，提示眼球向后扩张呈阶梯状凹下。暗线最常见于距视盘鼻侧1PD 大小或更远，严重者可形成环形。后巩膜葡萄肿后凹的深度与大小最好用双目立体检眼镜观察，其边缘与凹地屈光差别明显。  
#  后极部视网膜和脉络膜改变  
# 1 ）豹纹样  
高度近视眼常呈豹纹样眼底。因RPE 层和脉络膜色素变薄，暴露出脉络膜血管和血管间的色素。大的脉络膜血管常在后极部暴露，脉络膜动脉呈钩状进入眼球，脉络膜静脉则呈虫样形态。 RPE  和脉络膜色素显著变薄后，脉络膜血流供应也相应减 低，脉络膜静脉的解剖分布也不正常，可见大的静脉汇集成涡静脉横越黄斑或围绕视盘。后巩膜葡萄肿者 60  岁以后脉络膜萎缩 增多，局部萎缩可表现为圆形或不规则形、小或广泛，孤立或多发的黄白色区，其边界分隔清晰并可有色素聚集。进行性萎缩区中可见黄白色脉络膜血管，有的带白鞘。荧光素眼底血管造影，萎缩区呈弱荧光，造影早期可见其上视网膜血管充盈大致正常，无渗漏。当血内染料再循环至萎缩区时，各萎缩区均有相似的进行性着染。在近视进程中，萎缩斑趋向融合，当病变伸展至黄斑，中心视力受损害。  
# 2 ）漆裂纹  
漆裂纹是变形性近视Bruch 膜最典型的病变。在后极部已有组织变性的近视发病最多。漆裂纹表现为很细的线性或星状，粗细不规则的黄白色条纹，多沿水平方向分布，单一或多条，常交叉分支或呈渔网状，它们位于视网膜的最深层，其边缘有细的色素颗粒。较大的漆裂纹上，可见脉络膜血管跨过，该处神经视网膜包括视网膜内层和血管大致正常。漆裂纹的形成，一般认为是Bruch 膜的破裂，其邻近脉络膜毛细血管和色素上皮也受波及。多数漆裂纹位于黄斑，$17\%$ 与颞侧弧相连，荧光素眼底血管造影能发现非常细微、常规检查忽略的漆裂纹。造影早期，通过部分萎缩的脉络膜毛细血管，透见漆裂纹为不规则、不连续的强荧光线条。在造影过程中荧光中等增强；晚期荧光减弱，可能因为脉络膜萎缩，巩膜着染，但强荧光并不侵入漆裂纹内。在无并发症的漆裂纹中，未见视网膜内或视网膜下渗漏。有时沿着漆裂纹可见脉络膜视网膜局灶萎缩区，并与后极部相连，最后形成大片脉络膜视网膜变形区。有漆裂纹的眼中心视力预后难测。因为可能发生视网膜下新生血管膜及沿途出现局部变性，使视力受到危 害。漆裂纹的发生与发展，可并发黄斑区视网膜下出血。患者常于青年时代突然视力下降，视物变形，出血吸收后视力可好转，但日后还可在同一部位出血复发。黄斑出血常为局灶性，以中心凹为中心，无视网膜脱离。近视有黄斑出血者多与漆裂纹有关$\left(96\%\right)$），且多沿着漆裂纹，很少位其邻近。出血浓密，可完全掩盖有关的漆裂纹，即使荧光素眼底血管造影也不易发现。  
# 3 ） Fuchs  斑  
高度近视眼底后极部出现任何黑斑均可称为Fuchs 斑。典型者位于黄斑或其附近，为一个$1/3\sim3/4\mathrm{PD}$ 大小、灰色或黑色、圆形或椭圆形稍隆起的斑块。其位置也可不在黄斑，甚或在视盘鼻侧。双侧近视眼者均可患病。最初Forster 报道为黄斑出血，其后经Fuchs 详细观察，认为黑斑是RPE 细胞的增殖而非出血。长期过程中Fuchs 斑附近常有出血，其他近视性眼底改变或大致正常眼底。Fuchs 斑本身亦有变化，有时扩大变为其他形态，甚至很不规则，或分解为散在的色素斑点，但不会完全消失。视力亦不恢复正常。荧光素眼底血管造影能够显示，新生血管从脉络 膜延伸至RPE 下间隙，此与Fuchs 斑有关。其原因可能受玻璃体后界膜破裂的影响，脉络膜毛细血管的牵扯亦可能为一相关因素。Gass 指出Fuchs 斑的形成可能与新生血管形成的两个不同时期有关：一是急性出血性 RPE  脱离；二是出血机化并有视网膜 色素上皮细胞增殖，其细胞增殖可围绕每一脉络膜血管。  
4 ）视网膜下或脉络膜新生血管膜  
可发生在多种变性及炎症性眼底病。常见于年龄相关性黄斑变性及近视等。近视眼脉络膜新生血管形成的患病率约为$5\%\sim10\%$，高度近视可高至$40.7\%$。有双侧Fuchs 斑的近视眼，有$12\%\sim41\%$ 有此并发症。黄斑出血和Fuchs 斑代表脉络膜新生血管形成的不同过程。当脉络膜新生血管向内生长，可诱发急性无痛性视力下降，常伴有视物变形。用双目立体检眼镜或裂隙 灯观察，CNV 为一个略反光的圆形或椭圆形黄斑病变，通常为小的局限性病变（图5）。$58\%\sim74\%$ 的脉络膜新生血管直接侵犯中心凹，其余亦距中心凹$100\sim300\upmu\mathrm{m}$ 以内。位于中心凹下的脉络膜新生血管膜是变性性近视的临床特点（图6）。患眼眼球轴长常大于正常人的$26.5\%$，还经常伴有广泛的后极部脉络膜视网膜萎缩。  
在自然病程中，视网膜下新生血管膜因病变的不同时期而有其相应的临床表现。早期在黄斑中心由于血的聚集，呈现一暗棕色斑点，但出血并不广泛，主要位于神经视网膜，视网膜脱离常很浅，很少有硬性渗出。以后有色素变性，成为一黄灰色圆形或椭圆形微隆起病灶。当急性期过去，因为出血与渗出的吸收及习惯用偏中心注视，视力有某些程度的好转。日久，在大片脉络膜视网膜变性区内，出现RPE 增殖，位于视网膜下新生血管膜上，  
即Fuchs 斑。其病理机制尚未明确。有些病变除黄斑色素增多外，尚有广泛纤维增殖与萎缩。中心纤维组织内含有黑色素、血铁蛋白变性产物及黄色素等，晚期可有机化膜形成。  
一些研究提示，漆裂纹与视网膜下新生血管膜相关。漆裂纹的发病在有CNV 的近视眼中高于无CNV 者。有漆裂纹的近视眼$82\%$ 发生CNV，无可见的漆裂纹者$43\%$ 发生CNV。Bruch 膜破裂或萎缩区可先于CNV 存在，但很难判断萎缩区内是否曾经有过CNV。  
在荧光素眼底血管造影时，青年近视中可出现很小的脉络膜新生血管丛，位于近中心凹区。活动期CNV 呈梭形、类圆形隆起或不规则的高反射反光团，瘢痕期CNV 近表面呈高反射而其下的反射明显减弱，萎缩期CNV 表面完全变扁平。造影早期呈强荧光，以后出现染料渗漏，表明视网膜下新生血管的存在，但在整个造影过程中荧光增强不多，并只限于CNV 的边缘。其原因可能与变性性近视眼脉络膜血流迟缓、脉络膜毛细血管严重循 环紊乱有关。  
此外，脉络膜基质丧失、广泛的脉络膜变薄及部分血管阻塞亦有影响。因为新生血管来自脉络膜，其渗漏取决于脉络膜血流动力学特征。在老年近视中，视网膜下新生血管膜较多渗漏，造影晚期染料超过早期显示的新生血管膜的边界。这种渗出扩大有可能与老年性黄斑变性有关。有研究者认为只有年龄小于55 岁者才考虑为单纯近视眼性视网膜下新生血管膜。  
# （5）黄斑劈裂  
高度近视黄斑劈裂多发生于合并后巩膜葡萄肿的病理性近视患者，由于黄斑中心凹区不完全玻璃体后脱离或合并存在黄斑前膜，导致对中心凹前后方向牵引，黄斑部视网膜神经上皮层间分离，表现为不同层次和不同范围的劈裂，多位于神经纤维层或外丛状层。当玻璃体牵拉持续存在时，可发生继发性黄斑裂孔。  
高度近视的眼底中，黄斑劈裂并不常见，由于近年来高分辨率 OCT  技术的发展，才加深了对病理性近视黄斑劈裂的认 识。文献报道，在合并后巩膜葡萄肿的病理性近视患者中，有$9\%\sim20\%$ 黄斑劈裂发生。  
在后巩膜葡萄肿形成过程中，由于后极部巩膜向后膨出，牵引其上脉络膜视网膜的伸展变薄，当视网膜向后伸展到一定程度，在黄斑区视网膜层间的粘合力小于玻璃体和后巩膜葡萄肿导致的牵拉力时，则发生黄斑区视网膜层间劈裂。有文献报道，视 网膜动脉的牵拉也可能是劈裂的因素。Bando 等人将近视性黄斑劈裂患者的内界膜做了电镜观察，并与黄斑孔患者内界膜比较，发现近视黄斑劈裂患者的内界膜细胞成分与胶原含量明显多于黄斑孔患者，提示这两种成分对近视黄斑劈裂形成可能有一定的作用，认为玻璃体手术时应当剥除内界膜。  
文献报道，近视性黄斑劈裂的视力在$0.05\sim0.8$，可多年保持稳定。当伴有视网膜前膜等黄斑表面病变时，视力下降危险性显著增加。一旦并发黄斑视网膜脱离，无论是否进行手术干预，多会出现黄斑孔。  
高度近视黄斑劈裂在眼底常规检查中不易发现，借助OCT检查可以发现黄斑劈裂的特征性表现，视网膜神经上皮层被分为内外两层，两层中间见柱状反射桥接，色素上皮层前可见细的中等反射附着。黄斑劈裂包括外层劈裂和内层劈裂两种类型，两者也可合并存在。劈裂区常伴有局限性的视网膜脱离和其他黄斑病变，如黄斑前膜、不完全玻璃体后脱离对黄斑的牵引、板层，甚至全层黄斑裂孔。  
近年来频域OCT 与眼底自发荧光技术的发展，为认识病理性近视黄斑劈裂提供了新视角。Sayanagi 等使用频域OCT 发现近视性黄斑劈裂患者光感受器内节与外节损伤是该病的特征性病变之一，并且与术后视力恢复有一定的相关性。一部分近视性黄斑劈裂患者眼底自发荧光很接近正常人，其意义在于，可用于与黄斑孔视网膜脱离相鉴别。黄斑孔视网膜脱离裂孔处强荧光，周围视网膜脱离处因视网膜下液遮挡，呈弱荧光，而视网膜劈裂处虽有视网膜间液体，但有时仍呈正常荧光。  
#  周边部视网膜改变  
病理性近视眼除黄斑病变外，周边部（赤道部附近）也是病变好发部位之一，同属眼轴延长的结果。其破坏性仅次于黄斑病变，只是由于不直接影响中心视力，故多不被早期发现。Karlin及 Curtin  检查 1437  名高度近视患者证实，随眼轴延长，周边病 变增多。出现的病变主要可分为4 型：格子样变性、色素变性、铺路石样变性及白色（无压力型）变性（图7）。  
# （1）格子样变性（lattice degeneration  
格子样变性区内，视网膜血管管壁增厚或闭塞，呈白色线条状交织，格子样变性因此得名。格子样变性是视网膜脱离眼中最常见的玻璃体视网膜病灶，其发生率与眼轴长或屈光度高度相关。好发部位主要在赤道部，多双眼对称发生，与视网膜脱离的发生率呈正相关。临床上，格子样变性多为椭圆或圆形纬线方向走行的病灶，边界清楚且与玻璃体粘连，病灶区内玻璃体液化。在不同眼，变性区的数量、范围、大小可有很大改变，数量可$1\sim20$ 个不等，$90\%$ 以上为椭圆形或带状，可以是多个排列。变性区内视网膜表面常有白线样或网样结构、蜗牛迹样改变、色素沉着及视网膜萎缩孔，萎缩孔多在变性区内或变性区边缘，可单发和多发。  
组织学研究表明，变性区内的视网膜血管硬化，玻璃样变性以及闭塞；白色条纹多为神经胶质增生或结缔组织增生，部分为闭塞的视网膜血管，视网膜组织变薄，神经节细胞消失，仅残存少量胶质细胞。  
# 2 ）色素变性（ pigmented degeneration  
周边部视网膜弥散性黑色外观，也可是局灶性色素沉着或色素团块堆积。变性的后缘可延伸到赤道后甚至赤道后数个视盘直径，前界多不清楚。色素样变性主要见于高度近视眼，有报道显示，眼轴长$>31\mathrm{mm}$ 患者，色素样变性的发生率为$75\%$，但也和年龄有关，40 岁以上人群，色素样变性明显增多，且多为双侧，无性别差异。色素样变性的病因目前还不清楚，认为和炎症、血液供应、毒性、玻璃体牵引、RPE 先天异常等因素有关。  
组织学观察显示，脉络膜毛细血管丧失、Bruch 膜增厚、RPE 变性、光感受器细胞减少，因此，有人认为色素样变性是脉络膜毛细血管和RPE 对眼球扩张和年龄增长的反应。由于这种改变主要发生在外层视网膜，所以与视网膜裂孔和脱离的关系不大。  
# 3 ）铺路石样变性（ paving stone degeneration  
病变区主要分布在赤道与锯齿缘之间，虽在正视眼甚至低度远视眼可见，但主要见于高度近视。表现为白色小圆形或椭圆形边界清晰的脉络膜视网膜萎缩灶，可伴有块状色素斑，病灶中脉络膜血管清晰可见。由于这种改变主要发生在视网膜外层，一般与视网膜脱离无关。  
# 4 ）无压白（ white without pressure  
无压迫状态下周边部视网膜变白的区域，主要见于年轻高度近视眼。Curtin 等报道，20 岁以下眼轴长大于$33\mathrm{mm}$ 的眼$100\%$有无压迫变白。多环形走行，后缘清晰，前缘模糊。以颞下象限最常见，但大小、形态、位置可随时间而改变。  
有关的组织学研究较少，认为和玻璃体视网膜粘连有关。这种变性，很少发生视网膜脱离。但双眼无压白患者一眼发生巨大视网膜裂孔后，对侧眼发生巨大裂孔的可能性很高，有人强调要进行预防性视网膜激光光凝。
