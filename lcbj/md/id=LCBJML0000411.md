# 血运重建治疗支架的空间支持  
作用  
干细胞、生长因子和适宜的支架环境被认为是引 导组织再生的三个关键因素。在根管内营建合适的支架，吸引根尖外的干细胞进入，并在不同生长因子作用下促进干细胞的生长分化是组织再生的基本过程。  
在牙髓再生治疗中，根管内支架起着重要的作用。作为牙髓再生的重要条件，支架一方面为生长因子和干细胞提供生长分化的空间场所，同时还为再生的过程提供营养并预防细菌感染。理想的支架材料具有以下特点： $\textcircled{1}$ 孔隙大小和数量应适合细胞种植和传播； $\textcircled{2}$ 具  
有良好的生物活性与物理机械强度，免疫原性低，具备支持血管的能力；$\textcircled{3}$可被周围组织吸收，生物降解性好，降解与新组织形成的速度相匹配而不留下任何有毒副产物。  
从来源角度，根管内支架可以分为内源性支架和外源性支架，前者如血运重建治疗中自身的血凝块，源于自身的富血小板血浆（platelet-rich plasma，PRP）和富血小板纤维蛋白（platelet-rich ﬁbrin，PRP），后者如放置入根管内的各种人工合成支架、可吸收生物膜甚至是明胶海绵。  
# 42. 内源性支架主要包括血凝块、富血小板血浆和富血小板纤维蛋白  
血运重建中血凝块、富血小板纤维蛋白和富血小板血浆都是来源于自身的内源性支架。与外源性支架相比，内源性支架和机体间不存在安全性和相容性的问题。

# （2）富血小板血浆为支架的牙髓血运重建技术  
既然根管消毒后从根尖区导入的血凝块具有促进 牙根继续发育的功能，有学者认为如果进一步提取血凝块的有效成分，不仅可以避免治疗中无法诱导血凝块产生的概率同时还可以有效提高治疗的成功率。富血小板血浆是新鲜血液于$4\sim6\mathrm{h}$内以$27.5\sim37.5$转/min 低速离心 $15\sim20\mathrm{min}$ （或 1220  转 /min  离心 5min ）得 到的血小板浓缩物，将采集的全血在室温下低速离心，使红细胞、白细胞基本下沉，由于血小板比重轻，大部分保留在上层血浆中，分离出上层血浆，可获得全血中$70\%$ 以上血小板，即为富血小板血浆，含有血小板衍生生长因子（PDGF）、转化生长因子（TGF）、类胰岛素生长因子、表皮生长因子和血管内皮细胞生长因子等。

PRP 被认为可以满足理想再生支架的大部分要求。它富含生长因子，可以在根管内形成一个三维的纤维蛋白基质结构，利于生长因子在其中发挥作用。PRP 具有较高浓度的血小板，达到正常血液中血小板含量的$338\%$。血小板可以分泌血小板衍生因子（PDGF），该因子能刺激血管平滑肌细胞、成纤维细胞、胶质细胞的分裂增生。血小板数量增加的同时，它们分泌的生长因子也大量增加，促进了干细胞的诱导增生和组织的再生愈合。  
PRP 也具有一定局限性，如生长因子释放过快，作用时间短暂；制备过程中需要加入异种凝血酶和抗凝血制品，可能会导致免疫排斥反应的发生；提取过程中血小板在体外脆弱、易破碎等。  
不同研究对PRP 为支架的牙髓血运重建技术效果评价并不一致。Jadhav认为与传统的血运重建治疗相比，该方法可以更有效地促进根尖周病症愈合，加速根尖封闭和促进牙本质壁增厚。 Priya  甚至还报道 1  例 11  岁 男孩外伤脱位超过8 小时中切牙PRP 牙髓血运重建成功的报道病例。但Zhang 和Bezgin 分别通过动物实验表明，两者治疗后效果没有统计学差异。尽管存在争  
议，该方法至少可以解决某些情况下血凝块无法解决的难题。  
# （3）富血小板纤维蛋白为支架的牙髓血运重建技术  
富小板纤维蛋白是在PRP 基础上进一步提取制备并应用于临床的第二代血小板浓缩制品。静脉血经 $2~500\sim3~000$ 转/min 高速离心$15\sim20\mathrm{min}$后分三层，上层淡黄色澄清物为去血小板血浆，下层红色疏松物为红细胞碎片，中间层为PRF 蛋白凝胶。PRF 完全取自自体血，制备方法简单，并富含血小板及各种细胞因子，具有良好的促进软、硬组织再生的能力。PRF所含成分与PRP 相似，但具有更加优良的生物学特性。$\textcircled{1}$ PRF 可机械滞纳各种生长因子，并可释放生长因子以抵抗内源性纤维蛋白酶的蛋白降解作用，延长生长因子的作用时间；$\textcircled{2}$ PRF 制备过程中无须添加抗凝试剂及凝血酶制品，减少了免疫排斥反应的发生率、避免了交叉感染危险；$\textcircled{3}$ PRF 的纤维蛋白网状结构与人类天然的组织结构相似，具有组织疏松、孔隙较大、弹性良好的特点，使得营养成分及氧气可以轻松地弥散至细胞周围，促进骨髓干细胞分化；$\textcircled{4}$ PRF 凝胶含有

外源性支架是目前研究的热点。从组成角度而言，

外源性支架可分为天然支架、半合成支架和合成支架；从理化角度而言，它还可以分为非刚性支架和刚性支架。天然生物材料具有良好的生物相容性，为干细胞提供营养支持，促进干细胞分化。它们的缺点在于来源有限，具有一定变异性和免疫原性，结构内孔径大小受限，材料的机械性能受到一定的限制。与天然生物材料不同的是，合成支架可以在设定条件下按照要求获得所需的形状，并具备促进细胞分化特性和机械性能、化学性质。在满足强度、孔隙特征和降解率符合再生治疗的条件下，合成支架可以获得无限量供应。合成生物材料的缺点是缺乏细胞黏附位点，需要通过化学修饰以提高细胞黏附能力与效率。  
天然支架包括胶原蛋白、壳聚糖、丝绸、纤维蛋白等；半合成支架包括微藻螺旋藻、细菌纳米纤维素、复合纳米天然纤维、纳米纤维支架等；合成支架包括聚乙醇酸交酯、聚乳酸（PLA）、聚羟基乙酸（PGA）、聚丙交酯– 乙交酯（PLGA）和聚己内酯（PCL）。其他合成材料包括无机磷酸钙材料如羟基磷灰石（HA）或$eta-$ 磷酸三钙（β–TCP）、硅酸盐和磷酸盐玻璃。其中，PLA、PGA、PLGA 和PCL 等聚合物通常用于形成多  
孔支架。  
# （1）非刚性/ 软生物材料  
可以作为外源性支架的软生物材料很多，除了在富血小板纤维蛋白为支架的牙髓血运重建技术之外，尚有明胶海绵、胶原蛋白、海藻酸钠等。明胶海绵是白色或微黄色质轻软而多孔的海绵状物，具有吸水性。有学者在动物血运重建模型研究中，待血凝块形成之后在根管内加入可吸收性明胶海绵。研究结果发现与单纯血凝块相比，加入明胶海绵的牙根尖区炎症恢复更好。明胶海绵具有价格便宜操作简便的优点，在血运重建治疗的应用有待验证。  
胶原蛋白可以从肌肉、肌腱、皮肤等来源获得，结构与牙体组织细胞外基质的结构蛋白相似，目前被广泛应用于牙髓再生的各项研究中。胶原蛋白具有良好的生物相容性和生物活性，可以促进细胞黏附、细胞迁移和细胞生长。胶原蛋白可通过胶原酶降解，降解时只有轻微的炎症反应。胶原蛋白具有很高的拉伸强度，因此可以被编织或扭曲成所需的形状。它的物理强度不高，但已足够用于牙髓再生。胶原蛋白支架对牙髓干细

海藻酸钠是一种无毒并具有良好生物相容性的天然多糖，具有促进细胞黏附、传播、增生和分化的功 能。其缺点是机械强度较低，体内降解率不稳定。改性海藻酸钠水凝胶可通过增加钙含量和交联密度来提高机械强度。在牙髓和牙周膜再生基础上，海藻酸钠水凝胶还可以促进生长因子如$\mathrm{TGF-}eta_{1}$的输送，诱导成牙本质细胞分化和牙本质基质分泌，提高牙髓的自然再生能力。

聚乙二醇是一种无毒的高分子化合物，具有良好的生物相容性、低免疫原性和在体内降解的能力。它对细胞和蛋白质的吸附作用降低了免疫系统对它的排斥。聚乙二醇水凝胶被广泛应用为组织再生的支架材料。聚乙二醇水凝胶的力学强度由其分子量、交联度和浓度所决定，它的弹性模量可以通过降低分子量或增加聚合物浓度加以提高。目前研究中，已有学者将牙髓干细胞和牙周膜干细胞加入到聚乙二醇纤维蛋白凝胶支架中进行血管组织的再生研究。  
壳聚糖是角质素的衍生物，同样具有良好的生物相容性和生物降解能力，并具有一定抗菌作用和促进组织愈合、诱导骨形成的能力。它可以促进生长因子、糖胺聚糖和 DNA 相互结合形成生物膜、凝胶、纳米纤维或支架。由于这些特性，壳聚糖凝胶剂被认为可以单独或与其他胶原膜相结合而应用于组织再生工程。  
丝绸具有良好的生物相容性，无毒，其结构易于干细胞的吸附。在已知的牙周和口腔颌面外科再生修复研究中，丝绸材料提供了柔软而耐久的三维支架。在一项研究中，将牙胚干细胞接种于六氟异丙醇（HFIP）为基础的丝绸支架中可以看到新的矿化组织的产生。$10\sim600~mu\mathrm{m}$大小的丝质材料被应用于构建纤维复合材料，促进细胞分化与骨组织修复再生。另一项体外研究中，碱性成纤维细胞生长因子被添加到丝绸支架，随后发现新生成组织中有牙髓血管的再生、新的基质沉积和牙本质样组织形成。  
# （2）刚性生物材料  
聚乳酸、聚羟基乙酸和聚丙交酯– 乙交酯等是一类具有良好生物相容性、可生物降解和降解时仅引起轻度炎症的合成化合物。它们的结构、黏度、孔隙度和降解率都可以通过调整聚合物比例、分子量和结晶度加以调节。它们降解产生乳酸和乙酸可以通过自然代谢途径排出体外。在体外研究中，PLA 和PGA 支架可以

陶瓷支架是指钙/ 磷酸盐材料、生物活性玻璃和微晶玻璃等，长期以来一直用于修补颌骨的缺损。陶瓷支架生物相容性好，免疫原性低。但是陶瓷材料难以成形、机械强度差、脆性大、降解速度慢和密度过高限制了它们作为支架材料在组织再生中的应用。  
复合支架是由两种不同的支架材料组成的复合支架材料，如合成高分子材料和无机材料，结合了各种材料的优点，并最大限度地避免了它们的缺点。高分子材料缺乏足够的刚度，玻璃和陶瓷等硬质材料的加入克服了聚合物的固有缺点，使其适合于牙齿组织再生。在一项对不同复合材料支架的评估中，PLGA/TCP 支架最适合牙本质牙髓再生。
