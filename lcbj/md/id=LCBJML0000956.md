# 颅颈交界区后路螺钉固定技术  
# 4.  熟识颅颈交界区解剖有助于选择适当的手术  方式  
颅颈交界区是指由枕骨、寰椎、枢椎以及其周围的神经、血管、韧带等组织构成的复合体。作为颅骨与颈椎之间的移行区，其解剖结构复杂。因此，熟识此区域独特的解剖结构有助于更好地理解各种颅颈交界区畸形，进而选择适当的手术方式进行治疗。其中与后路内固定技术相关的解剖结构包括寰椎（$\left(\mathbf{C}_{1}\right.$）侧块、椎弓根，枢椎$\left(\mathbf{C}_{2}\right)$）棘突、椎板、椎弓峡部以及椎弓（图5、图6）

# 5. 颅颈交界区后路螺钉内固定技术已广泛取代其他 内固定技术  
颅颈交界区疾病种类多样，常见的如颅底凹陷、寰枢椎脱位等。临床上应根据疾病的不同，选择合适的手术方式进行治疗。当存在颅颈交界区失稳时，就需要合理应用内固定技术维持其稳定性。目前常见的固定方式包括前路钉板系统、齿状突中空螺钉、经侧方关节螺钉固定以及后路螺钉内固定等技术。由于后路螺钉内固定技术具有良好的稳定性、合理的生物力学原理、术中复位方便以及可被接受的手术风险等特点，已广泛取代其他内固定技术。  
颅颈交界区后路螺钉内固定技术主要涉及枕骨、寰椎以及枢椎的置钉，根据病情需要，有时还需要进行$\mathrm{C}_{3}$ 置钉。下面将对这些置钉技术进行简要说明。  
1）枕骨置钉技术  
枕骨螺钉的作用是为了固定枕骨板。由于枕骨嵴处骨质较厚，因此通常将螺钉置于枕骨中线，以获得最大的把持力。置钉位置如图7。进钉方向垂直于骨面即可。除了中线螺钉，通常还配合使用中线旁螺钉，能有助于抵抗枕骨板的旋转力。此外，还需选择合适大小的枕骨板，以便能良好地贴合于枕骨，同时避免 后续步骤中不必要的塑棒。  
（2）寰椎（$[\mathrm{C}_{1}$）置钉技术  
寰椎置钉有两种常见的方法：$\mathrm{C}_{1}$ 侧块螺钉与$\mathrm{C_{1}}$ 椎弓螺钉。侧块螺钉进钉点位于侧块后方的中央，而椎弓螺钉进钉点为椎弓在$\mathrm{C}_{1}$ 后弓上的投影（图8）。  
寰椎侧块螺钉进钉方向在矢状面上与水平线成 $10^{\circ}\ \sim15^{\circ}$ ° 角，在轴位上与中线成$10~^{\circ}~\sim15~^{\circ}$°角。螺钉长度通常选择$22\sim28\mathrm{mm}$（螺钉位于侧块内部的长度为$14\sim18\mathrm{mm})$）。而椎弓螺钉的进钉方向则根据椎弓方向进行确定。螺钉长度同样为$22\sim28\mathrm{mm}$（图9）。  
注：A、B. 为侧块螺钉，C、D. 为椎弓螺钉  
# 3）枢椎（$\mathrm{C}_{2}$）置钉技术  
枢椎置钉技术包括椎弓根置钉和椎板置钉。枢椎椎弓根螺钉置入时，首先明确椎弓根与峡部连接处内上缘切线，然后根据该 切线与椎弓峡部后方的交点来确定进钉点（图10）。枢椎椎弓根螺钉实际沿峡部进入，经过椎弓根后进入椎体。  
而进钉角度需要根据椎弓根内上缘与峡部连接处切线方向来 确定（图11）。进钉长度通常为$20\sim24\mathrm{mm}$。  
枢椎椎板交叉螺钉进钉点分别为两侧棘突基底部枢椎椎板的上、下1/3 处。置钉方向沿椎板方向为准（图12）。螺钉长度通常选择$25\sim35\mathrm{mm}$。  
# （4）$\mathrm{C}_{3}$ 侧块螺钉置入技术  
$\mathrm{C}_{3}$ 侧块螺钉进钉点在侧块中点内上方$1\sim2\mathrm{mm}$。进钉方向稍向上倾斜，与上关节突关节面平行，向外倾斜$20^{\circ}~\sim30^{\circ}$°。螺钉贯穿前后骨皮质，进钉长度为 $10\sim16\mathrm{mm}$ ，尖端位于关节 突前面的上外侧（图13）。

#  术前完美并认真评估患者颅颈交界区影像学检查  
颅颈交界区解剖结构复杂，尤其对于颅颈交界区畸形的患者，解剖变异与个体差异十分显著，为手术的进行带来极大的风险与挑战。因此需要在术前完善并认真评估患者的影像学检查，充分了解每个病例的解剖特点，以便合理制定手术方案，顺利完成手术。  
颅颈交界区畸形，尤其是合并寰枕融合的患者，椎动脉走行异常较为常见，如走行在$\mathrm{C}_{1}$ 后弓下方、出现椎动脉开窗等，因此手术显露过程中需要特别注意，避免血管损伤。图14 所示患者可见双侧椎动脉走行异常。  
进行$\mathrm{C_{1}}$ 椎弓螺钉置钉前，一定要留意椎动脉与$\mathrm{C}_{1}$ 的解剖关系。一般情况下，椎动脉会走行在$\mathrm{C}_{1}$ 上方的椎动脉沟内。但在某些情况下，椎动脉沟形成“椎动脉管”（图15），可能会增加置钉风险，应特别注意。  
在枢椎（$\mathrm{C}_{2}$）椎弓根螺钉置入过程中，则需特别关注$\mathrm{C}_{2}$ 椎弓根的大小及其与血管的关系。图16 中由于椎动脉发育及走行不同，$\mathrm{C}_{2}$ 椎弓根及峡部的大小也不同，螺钉置入时应特别关注。图17 可见椎动脉在$\mathrm{C}_{2}$ 内走行位置较高，即椎动脉高跨。此时螺钉置入风险较高，或根本不适合螺钉置入。  
#  颅颈交界区后路螺钉固定方式  
1 ） $\mathrm{C_{1}-C_{2}}$  椎弓根螺钉内固定术  
# 适应证  
寰椎侧块解剖结构正常时，可首选$\mathrm{C}_{1}$ 椎弓/ 侧块$\mathrm{-C}_{2}$ 椎弓根螺钉进行固定。常用于治疗齿状突小骨、外伤横韧带撕裂及类风湿等引起的寰枢椎脱位。  
有时即使患者存在寰枕融合，但若因既往后颅凹减压术等原因造成枕骨不完整而无法进行枕颈固定时，也需在寰椎侧块- 枕髁复合体中置钉。在习惯上，我们仍称该部位的螺钉为${}^{\circ}\mathrm{C}_{1}$ 侧块螺钉”。在此异常结构中置钉前要仔细评估患者术前影像，避免破坏椎动脉管或舌下神经管，以免造成神经血管损伤。  
# 病例1  
女性，45 岁，主诉进行性四肢麻木、无力2 年。查体见四肢肌力Ⅴ级，腱反射亢进，双侧 Hoffman  征及双侧 Babinski  征阳 性。考虑诊断齿状突小骨引起的寰枢椎脱位。术前低头位X 线显示 $\mathrm{C_{1}-C_{2}}$  脱位，齿状突尖部与基底不连续（图 18A ） 。 MRI  显 示$\mathrm{C}_{1}\mathrm{-C}_{2}$ 区域脊髓内异常信号（图18B），CT 显示齿状突游离小骨（图18C，图18D）。手术采用$\mathrm{C}_{1}$ 椎弓$\mathrm{-C}_{2}$ 椎弓根螺钉进行内固定。术中X 线显示螺钉固定位置良好（图18E）。术后CT 可见寰枢椎对位良好，寰椎两侧椎弓螺钉及枢椎两侧椎弓根螺钉位置良好（图18F，图18G，图18H，图18I）。  
注： A.  术前低头位 X  线； B.  术前 MRI ； C 、 D.  术前 CT ； E.  术中 X  线； F 、 G 、 H 、 I.  术后 CT  
# 病例2  
女性，59 岁，四肢麻木、力弱进行性加重1 年入院。诊断为寰枢椎脱位。术前颈椎中立位 X  线检查可见寰齿间距显著增 加（图19A），正中矢状位CT 可见枢椎相对寰椎向后上方脱位（图19B），MRI 可见延髓显著受压（图19C）。手术复位后采用$\mathrm{C}_{1}$ 侧块$\mathrm{-C}_{2}$ 椎弓根螺钉进行固定。术后复查CT，正中矢状位片可见复位良好（图19D），两侧可见螺钉在位良好（图19E，图19F，图19G，图19H）。

病例3  
女性，34 岁，主因双上肢麻木进行性加重1 个月入院，诊断为先天性寰枢椎脱位、小脑扁桃体下疝畸形、寰枕融合。术前CT 可见寰齿间距增加，伴颅底凹陷（图20A）。术前MRI 显示小脑扁桃体下疝畸形，高位颈髓受压明显（图20B）。手术采用后路寰枢椎复位内固定术$^+$ 后颅凹减压术。尽管患者存在寰枕融合，融合的寰椎侧块—枕髁复合体置钉难度较高，但是由于患者同时需要行后颅凹减压术，不适合进行枕颈内固定术，所以固定方式依然选择为$\mathrm{C}_{1}$ 侧块$\mathrm{-C}_{2}$ 椎弓根螺钉。术后CT 见寰枢椎复位、后颅凹减压且双侧螺钉位置良好（图20C，图20D图20E）。

# （2）$\mathrm{C_{1}-C}_{2}$ 椎板螺钉内固定术  
# 适应证  
一侧或双侧$\mathrm{C}_{2}$ 椎弓根较细而不适合螺钉置入，或前次椎弓根螺钉置钉失败后不具备再次置钉条件时，可选择$\mathrm{C}_{2}$ 椎板行交叉螺钉置入。虽然此方法没有损伤椎动脉的风险，但也要注意避免螺钉置入椎管而造成脊髓损伤。  
# 病例  
女性，41 岁，主诉进行性双下肢无力6 个月，诊断为齿状突小骨引起的寰枢椎脱位。术前 CT  正中矢状位可见齿状突小 骨，寰枢椎脱位（图21A）。侧方关节层面矢状位CT 显示枕髁、寰椎侧块发育良好，但椎动脉在枢椎内的位置较高，椎弓根细小，不适合椎弓根螺钉置入（图21B）。因此，手术选择$\mathrm{C}_{2}$ 椎板螺钉与$\mathrm{C}_{1}$ 侧块螺钉固定技术（图21C）。术后复查CT，显示椎板内交叉螺钉位置良好（图21D，图21E）。  
注：A、B. 术前CT；C. 术中照片；D、E. 术后椎板螺钉  
（3）$\mathrm{C}_{0}–\mathrm{C}_{2}$ 椎弓根螺钉内固定术  
# 适应证  
颅颈交界区畸形多合并出现寰枕融合，对于此类患者，由于$\mathrm{C}_{1}$  侧块与枕髁融为一体，寰枢侧块植钉难度及风险大。若枕骨条 件允许，可在此时选择枕骨$\mathrm{-C}_{2}$ 椎弓根螺钉进行固定。  
# 病例  
女性，43 岁，进行性双上肢麻木1 年，考虑诊断：颅底凹陷、寰枢椎脱位、脊髓空洞症。术前颅颈交界区MRI 提示枕大孔区狭窄，小脑扁桃体下疝伴脊髓空洞（图22A，图22B）。术前CT 提示寰枕融合、寰枢椎脱位、颅底凹陷（图22C）。由于患者存在寰枕融合，手术所采用的内固定方式为枕骨$\mathrm{-C}_{2}$ 椎弓根螺钉内固定技术。术中照片可见枢椎椎弓根螺钉通过枕骨板与枕骨连接固定（图22D）。术后3 个月复查MRI（图22E，图22F），显示枕大孔区减压充分，脊髓空洞明显缩小。复查CT（图22G），可见寰枢椎脱位复位满意，枕骨与枢椎间植骨生长良好。CT 三维重建显示枕颈内固定装置整体结构（图22H）。

# 4 ） $\mathrm{C}_{0}–\mathrm{C}_{2}$  椎弓根 $\mathrm-\mathrm C_{3}$  侧块螺钉内固定术  
适应证  
如果一侧$\mathrm{C}_{2}$ 椎弓根较细，除可选择$\mathrm{C}_{2}$ 椎板交叉螺钉外，还可以选择单侧$\mathrm{C}_{2}$ 椎弓根，并延长到双侧$\mathrm{C}_{3}$ 侧块的螺钉固定方式。  
# 病例  
女性，41 岁，主诉行走不稳30 余年，双上肢无力7 年，加重2 年。术前MRI 见齿状突向后严重压迫延髓（图23A）。正中矢状位CT 可见寰枢椎脱位，颅底凹陷，寰枕融合（图23B），侧方矢状位CT 可见左侧$\mathrm{C}_{2}$ 椎弓根发育良好（图23C），但右侧$\mathrm{C}_{2}$ 椎弓根细小，置钉损伤椎动脉风险极高（图23D）。CTA 三维  
重建可见双侧椎动脉走行异常，右侧椎动脉高跨（图23E）。因此手术进行寰枢椎复位后采用右侧 $\mathrm{C}_{0}–\mathrm{C}_{3}$  侧块、左侧 $\mathrm{C}_{0}–\mathrm{C}_{2}$  椎弓 根$\mathrm{-C}_{3}$ 侧块螺钉内固定术。术后X 线检查可见螺钉位置良好（图23F，图23G），正中矢状位CT 见复位满意（图23H），MRI 见延髓充分减压（图23I）。  
注：A. 术前MRI；B、C、D. 术前CT；E. 术前CTA 三维重建；F、G. 术后X 线；H. 术后CT；I. 术后MRI  
颅颈交界区后路螺钉固定技术较为复杂，尤其对于先天性畸形所致寰枢椎脱位的患者，应根据其具体情况选择个性化的固 定方式。术前X 线片、CT 扫描及二维、三维重建影像是准确安全进行颅颈交界区后路螺钉固定的关键。只有在深入理解解剖结构、细致分析患者病情、仔细规划手术方案的基础上，配合谨慎的术中操作，才能顺利完成手术并达到满意的效果。
