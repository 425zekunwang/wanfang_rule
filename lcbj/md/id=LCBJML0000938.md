# 后巩膜加固手术  
#  后巩膜加固手术的历史与沿革  
高度近视包括单纯高度近视和病理性近视（pathological myopia）。病理性近视的发病机制有遗传因素和环境因素，因其巩膜代谢异常出现的后极部巩膜进行性扩张、变薄，眼轴不断延长，屈光度持续增加，眼球后极部的血液循环发生障碍，引起后极部视网膜脉络膜发生退行性病变，进而损伤视功能，成为永久 性的致盲性眼病之一。病理性近视的治疗主要有控制眼轴延长，矫正屈光异常及治疗眼底病变三个方面。若在病理性近视患者出现视网膜脱离、黄斑出血等严重眼底并发症时再给予治疗，视功 能已出现不可逆性损害。因此，控制眼轴增长的治疗则是病理性近视主动的治疗方法。  
后巩膜加固术，又称巩膜后兜带术、后巩膜支撑术或后巩膜加强术，是针对病理性近视巩膜胶原病理变化的预防为主、防治结合的一种安全有效的手术方式。它是应用异体或自体的生物材料或人工合成材料加固眼球后极部巩膜薄弱的部分，来阻止或缓解眼轴增长，减缓近视发展，防止并发症发生和加重的一种手术方式。后巩膜加固术在临床上应用已有60 多年的历史，其发展过程可分为四个阶段。  
# （1）初步应用  
1954 年，苏联学者Malbran 首次报道了后巩膜加固术在临床中的应用。使用自体或异体阔筋膜，植入眼球后极部，绕过视神经，加固材料两端分别缝合在上、下直肌肌腱旁。该报道包含治疗的21 例近视，其中7 例还进行了巩膜切除术。1957 年，Curtin 应用人的阔筋膜制呈“X”形植入眼球后以加固巩膜强度，延缓高度近视眼的进展，并认为可能改善了脉络膜循环。1962年，Borley 使用异体巩膜按照Curtin 的方法对儿童27 眼，成人36 眼高度近视进行巩膜加固，术后$95\%$ 患者屈光度减低3.0D 以上，$68\%$ 的患者术后视力提高，由此得出结论巩膜加固可以防止近视眼患者在黄斑出现Fuchs 斑和视网膜脱离。1970 年，Nesterov 改良了Curtin 的方法，采用Y 形阔筋膜条带加固后巩膜67 例84眼，术后近视度数较未手术者稳定，且术后视力绝大部分得到了提高。1970 年，苏联学者Zarkova 及英国学者Whitwell 报道了帽状后巩膜加固术，可有效预防和控制黄斑部葡萄肿的发生和发展，但上述手术中，术者为了更清晰的视野，常将眼肌切断，并将加固条带放于上斜肌下，这两种手术操作增加了术后眼前段缺血和眼肌平衡失调的危险性。因此，尽管这些手术在防治高度近  
视的发展上有一定的效果，但其操作复杂，并发症多，在临床上未得到广泛的认可。  
# （2）多量开展  
1972 年，Snyder-Thompson 介绍了一种新的巩膜加固术。将不分叉的单条带同种异体巩膜置于下斜肌与视神经之间，一端固定在上直肌颞侧，一端固定于下直肌鼻侧。该方法能在不切断眼外肌的情况下较好地暴露眼球后极部，并且使用冷凝封闭周边的网膜变性区和裂孔。经过长期随访观察，该术式的安全性和有效性得到了证实。20 世纪80 年代以来，俄罗斯在非手术巩膜加固术治疗高度近视方面做出了一些新探索，即采用非生物材料作为加固巩膜的材料，称为亚手术方法。1981 年，Avetisov 提出在球后筋膜囊下注射聚合物的混合物，在巩膜表面形成弹性壳，通过结缔组织增生达到加固巩膜的目的。1980 年，后巩膜加固术传入中国。20 世纪中后期临床上有较多开展。  
3）边缘化  
由于当时后巩膜加固术没有理想的加固材料及加固材料的形状设计缺陷，导致手术疗效不确切，引起了严重并发症，而逐渐被边缘化。2000 年前后，其他大量新技术受到更多的关注。  
（4）逐渐热门  
目前，随着对病理性近视认识的逐渐加深，越来越多的国家开展后巩膜加固术。Snyder-Thompson 术是由前向后、经过黄斑区的后巩膜加固术，能尽量少的损伤巩膜外组织，是目前临床常用的方法。美国加州的Dr. Brian Ward，俄罗斯莫斯科的Elena P.Tarutta 和中国温州的薛安全是目前开展后巩膜加固术较多的学者。  
由上可知，后巩膜加固术的术式有条带法、片式法和注射法。条带法包括X 型、Y 型和单条带加固法，单条带加固法较为流行。单条带加固法又有一些改良做法，如黄斑加压型后巩膜兜带术、加宽型后巩膜兜带术、后极部扣带加固术。片式法和注射法虽较为简单，但难以达到手术机制的要求。条带法较符合手术机制的要求，但操作较复杂，单条带法相对X 型、Y 型条带法简便（图19）。加固的材料包括生物材料（自体材料、同种异体材料和异种材料）和非生物材料（硅胶、涤纶布、某些高分子材料的凝胶）。自体材料（如阔筋膜），需在身体上留下伤口，患者不易接受，采取困难。同种异体材料（如异体巩膜和硬脑膜），有较好的组织相容性，在移植片与巩膜之间可观察到营养的血管和神经，未见到瘢痕和移位现象。但来源紧缺，易降解，抗张力差。非生物材料来源广泛，植入后不易降解，有较高的张力和韧性，但组织相容性差，易发生慢性无菌性炎症或排斥反应。随着组织工程学和材料学研究的发展，交联技术得到了新的应用。中国温州薛安全使用京尼平进行加固条带的交联。以京尼平为交联剂处理的生物膜材料，呈深蓝色，颜色醒目，可视性好，制成的条带在植入过程中，不容易迷失。交联过的加固条带韧性增高，机械强度增加，组织相容性好，抗降解能力增加。加固的条带有片状和条状。目前常用的是条带梭形设计，为中间宽向两端逐渐变窄，长度$30\sim60\mathrm{mm}$，两端宽度$2\sim3\mathrm{mm}$，中间宽度$10\sim16\mathrm{mm}$，能够避免涡静脉受压和损伤，扩大加固范围。条带边部斜面设计，其边部斜面呈$30\sim45$ 度，植入穿行过程中，以斜面向外或向内作为条带正反面标记，不容易引起扭曲翻转。

#  后巩膜加固术的适应证  
后巩膜加固术的适应证包括：（1）控制近视和预防眼底病变  
儿童和青少年平均每年经睫状肌麻痹验光加深－1.0D 或以上、眼轴$25.5\mathrm{mm}$ 以上，有高度近视家族史；成人原有近视程度在－6.0D 以上，眼轴在26mm 以上，连续3 年平均每年经睫状肌麻痹验光加深－1.0D 或以上；后巩膜葡萄肿患者（图20）。  
（2）治疗巩膜扩张引起的眼底（黄斑）病变  
荧光素眼底血管造影或吲哚青绿脉络膜造影显示视网膜、脉络膜呈现退行性病变、黄斑裂孔、黄斑劈裂（图21）、黄斑脱离（有无裂孔和牵引）、黄斑出血（单纯，CNV）、黄斑萎缩（图22）等。高度近视眼合并视网膜脱离，在进行视网膜复位手术的同时联合后巩膜加固术。  
3）内路术后不良  
解除硅油依赖（高眼压、未复位、低眼压）、复发脱离、眼底萎缩。  
由于病理性近视会随着年龄的增长而眼轴继续增长直至生命的终结，并出现因眼轴过度增长而造成的进行性后极部视网膜脉络膜的病变。因此，后巩膜加固术应尽早进行，眼底损害出现之前进行预防性治疗是手术的最佳时机。在病理性近视中，若病程仍在进展，眼轴继续延伸，屈光度加深或眼底损害逐渐加重者为适应证。  
#  后巩膜加固术的临床效果  
自1972 年Snyder 和Thompson 首次运用单纯性后巩膜加固术治疗近视之后，该方法已有40 多年历史。大量临床研究证实，后巩膜加固术能够稳定眼轴，控制近视度数的加深；改善眼底血液循环，以阻止或暂缓并发症的发生，从而达到保护视功能的目的。刘修铎等对16 例病理性近视患者32 眼进行后巩膜加固术，随访$10\sim13$ 年后发现后巩膜加固术远期能够安全有效地延缓高度近视眼眼轴增长，稳定视力，且并发脉络膜新生血管的比率较低（$\left(3.1\%\right)$），但需要更大样本的对照研究及相关基础实验研究进一步论证。Chen 等对41 例（64 眼）中国病理性近视儿童行改良后Snyder-Thompson 术，对照组11 例（17 眼）配戴接触镜，分别随访（$\left(5.0\pm1.3\right)$）年、（$(4.5\pm1.3\)$）年，对比分析眼轴、SE、UCVA 、 BCVA 、眼底及并发症差异后得出结论。改良后 Snyder- Thompson 术治疗中国儿童近视是安全且有效的。Ward B 等对病理性近视患者（59 眼）进行单眼后巩膜加固术，对侧眼作为对照；术后随访5 年，观察眼轴长度、视力、黄斑情况及其他并发症，认为后巩膜加固术可有效控制病理性近视眼轴的增长，无明显并发症发生。有许多文献也报道了后巩膜加固术术后患者视力得到了提高，但具体机制尚不清楚。有学者认为，加固条带为异体组织，对巩膜、脉络膜、视网膜有局部刺激作用，引起视细胞的兴奋，提高了视敏度。有实验发现，加固条带中有新生血管长入，术后荧光素眼底血管造影显示视网膜血管循环时间延长，增强了视网膜细胞、脉络膜细胞的新陈代谢，提高了视敏度。  
高度近视容易引起脉络膜新生血管，约占高度近视眼患者中的$10\%\sim20\%$，严重危害视功能。有研究通过后巩膜加固术术后10 年以上的随访，发现术后很少出现CNV（$(3.1\%)$），远低于国内外高度近视CNV 的发生率，且高度近视眼底情况基本保持稳定，提示后巩膜加固术对保护高度近视眼底，预防 CNV  有一 定作用。  
总而言之，后巩膜加固术的作用机制是将加固材料植入后极部巩膜，随着时间的进展，加固的异体巩膜和受体巩膜之间形成纤维组织粘连，变成一层较厚的巩膜，起到机械性加强后部巩膜的作用，因而抗张能力增加，阻止眼球进行性扩张，减轻视网膜、脉络膜被牵拉引起的病变。两层巩膜之间形成巩膜新的血管网，脉络膜视网膜血循环得到改善，加强了眼部的血液循环。另外，加固材料对巩膜局部起刺激作用，相当于生物学组织疗法，达到加固巩膜的目的，从而达到控制近视度数发展的目的。  
巩膜生物力学实验研究表明，近视同正常眼相比，眼球更容易发生变形，承载能力下降，延展性增加。实行后巩膜加固术后，提高了巩膜的韧性，增加了巩膜抵抗变形的能力，改善了后极部血液供应。术后巩膜胶原重建，巩膜结构重塑恢复正常。  
除此之外，为了增强加固效果，术中要进行前房放液，调整并缝和加固条带，从而使增长的眼轴缩短。此原理也可用于高度近视性黄斑劈裂和黄斑区视网膜脱离的治疗。  
由于受技术条件的限制，后巩膜加固术的术式和材料存在缺陷，缺乏临床标准，难以达到手术机制的要求，故严重影响了手术的临床效果。有少数研究者认为后巩膜加固术疗效不确切，并发症多，严重的并发症如视网膜脱离。但是，尚未发现后巩膜加固术本身与术后视网膜脱离之间有直接关系。高度近视眼常由于并发周边视网膜裂孔、变性而容易导致视网膜脱离。因此，行后巩膜加固术前详细检查眼底非常必要，及时做预防处理，能降低术后视网膜脱离的发生。同时还与术者操作熟练程度和技巧有关。后巩膜加固术疗效出现争议，是不被临床广泛认同和开展的主要原因。  
后巩膜加固术的技术要点在于：$\textcircled{1}$熟悉眼球周围解剖结构，巩膜条带应贴附于下斜肌止端与视神经之间； $\textcircled{2}$ 手术者应操作轻 柔，特别是游离下斜肌放置巩膜条带时，术中应注意保护涡静脉、睫状动脉及视神经，加压条带勿过紧或过松，防止加固条带 压迫视神经。  
# 26. 后巩膜加固术与玻璃体切除术治疗高度近视黄斑裂孔性视网膜脱离的效果对比  
黄斑裂孔性视网膜脱离多发生于高度近视患者，有文献显示，$97.6\%$ 的黄斑裂孔性视网膜脱离发生于屈光度$\mathrm{>-~}8.25\mathrm{D}$ 的高度近视患者。高度近视黄斑裂孔性视网膜脱离的发生主要源于黄斑前玻璃体或视网膜表面膜对黄斑垂直、切线方向的牵拉，以及后巩膜葡萄肿形成玻璃体对视网膜垂直方向的牵拉。  
另外，视网膜色素上皮萎缩导致视网膜神经上皮与脉络膜黏附力减弱，内界膜表面细胞成分的收缩加重切线方向的牵引等也可引起高度近视黄斑裂孔性视网膜脱离。目前，国内外有多种方法治疗高度近视黄斑裂孔伴视网膜脱离，单纯玻璃体切除术视网膜解剖复位率较好，但视功能恢复较差。玻璃体切除术联合玻璃体后皮质、内界膜剥除及硅油填充术是目前治疗高度近视黄斑裂孔性视网膜脱离的主要方法，其目的在于撕除玻璃体后皮质及内界膜，从而解除黄斑部玻璃体视网膜牵拉和视网膜前膜收缩的牵引力。硅油填充后的顶压作用也有利于视网膜复位及黄斑裂孔愈合。  
邓春梅等回顾分析高度近视黄斑裂孔患者（伴或不伴黄斑裂孔性视网膜脱离）33 例36 眼的病例资料，得出玻璃体切除联合内界膜剥离术可以显著提高高度近视黄斑裂孔的闭合率和术后视力，但对于视网膜的复位率较不剥膜者无明显区别。姚宜等观察29 例（32 眼）高度近视黄斑裂孔视网膜脱离患者在曲安奈德辅助下玻璃体切除联合内界膜剥除及惰性气体或硅油填充治疗的 疗效。结果发现玻璃体切除联合玻璃体后皮质、内界膜剥除及惰性气体或硅油填充是治疗高度近视黄斑裂孔性视网膜脱离的有效方法。  
后巩膜加固术可以通过缩短眼轴，减少眼部体积，释放玻璃体对视网膜的牵引从而使黄斑部脱离和视网膜劈裂得到有效解决，且对于控制病理性近视的发展也是有效的，这一结论在Zhu等研究中也得到证实。他们使用京尼平交联的巩膜给予21 例24眼黄斑脱离和视网膜劈裂患者行后巩膜加固术，随访至术后至少1 年，发现$87.5\%$ 的患者视网膜重新贴附，黄斑部的脱离明显减轻，患者的眼轴在术后半年趋于稳定至随访结束，且术后患者视力也得到了提高。后巩膜加固术术中缝和加固的条带，并缩短眼轴，加强了巩膜厚度及抵抗力，在远期上起到了延缓或稳定眼轴增长的目的。同时，眼轴的缩短可以缓解由后巩膜葡萄肿及玻璃体皮质对内层视网膜的牵拉，增强 RPE  层与神经上皮层的连 接。朱双倩等对25 例（32 眼）病理性近视黄斑劈裂（其中6 眼伴视网膜局限性浅脱离）患者行后巩膜加固术，观察术前与术后6 个月BCVA、SE、眼轴、OCT、ERG 的变化后得出结论，后巩膜加固术治疗病理性近视黄斑劈裂具有一定的疗效，不仅有助于黄斑劈裂腔的减小，也有助于视网膜光感受器传导功能的改善。二者比较来看，后巩膜加固术是相对病因治疗，能延缓病理性近视的进一步发展，且其操作简单，可降低内眼手术的风险。术中缩短眼轴，在一定程度上改善了屈光状态。玻璃体切割术不能延，只能治标，若联 合硅油填充术还需要二次手术，且手术后需要俯卧位，年龄大、身体条件差者不适宜。玻璃体切除术操作相对复杂，手术风险较高，较易出现如黄斑出血、周边视网膜裂孔及脱离、视网膜撕裂等并发症。  
对于后巩膜加固术后依然存在的视网膜脱离或视网膜前纤维增生，可通过玻璃体切割联合内界膜剥除手术去除。  
# 27. 后巩膜加固术与玻璃体切除术二者联合的必要性大吗  
高度近视黄斑裂孔伴视网膜脱离病情复杂，治疗困难，预后欠佳。单纯行玻璃体切除术治疗，视网膜脱离的复位率可达$90\%\sim100\%$，但黄斑裂孔的闭合率只有$11.8\%$。玻璃体切除联合内界膜撕除只能解决黄斑垂直及切线方向的牵引，对后巩膜葡萄肿向后扩张所致牵引力及进展不起作用。后巩膜加固术对有后巩膜葡萄肿的高度近视劈裂效果好，对黄斑区相应的巩膜能增强巩膜厚度及抵抗力，阻止眼轴进行性延长。但是，后巩膜加固术不能完全解除由高度近视眼后巩膜葡萄肿引起的玻璃体视网膜、脉络膜牵引，以及内界膜切线方向的牵引。玻璃体切除联合内界膜撕除主要解除玻璃体牵引和内界膜的牵引，后巩膜加固术主要解除后巩膜葡萄肿的牵引。因此，对于高度近视黄斑劈裂患者，有必要采用后巩膜加固术联合玻璃体切除内界膜撕除术，内外路同时进行。  
后巩膜加固术与玻璃体切除术具有互补性，前者可为后者创造较稳定的眼轴与眼底条件。有学者认为，玻璃体切除术影响眼内正常的生理状态，不能改善黄斑区视网膜营养。而玻璃体切除术联合后巩膜加固术通过组织的炎症反应产生新生血管可改善后极部血液循环，在一定程度上可以改善黄斑区视网膜的营养。通，起到类似扩大黄斑区兜 带的作用，能有效缩短眼轴，减少眼球容积，减轻牵拉，改善后极部的脉络膜血液循环，部分联合气体或硅油的填充，视网膜在顶压作用下紧贴色素上皮，恢复脉络膜的泵吸作用，使黄斑得到良好复位。  
林会儒等将高度近视性黄斑裂孔伴视网膜脱离患者45 例4528  眼和玻璃 体切割内界膜撕除组 17  眼。结果得出玻璃体切割术联合后巩膜 加固术治疗高度近视眼黄斑裂孔伴视网膜脱离的手术方法安全可行，可更好地改善视力，提高黄斑裂孔的闭合率。  
对于高度近视性黄斑裂孔伴视网膜脱离，玻璃体切割术剥 除黄斑区视网膜前膜及内界膜，能解除玻璃体对黄斑裂孔周围视网膜的牵拉，促进黄斑部视网膜与脉络膜粘连，增加黄斑裂孔伴视网膜脱离的复位率。后巩膜加固术术中固定于后极部巩膜外的异体巩膜组织对局部有炎症刺激，且改善了后极部的微循环，条带起到的加固作用可以减轻玻璃体切线方向的牵拉力，联合玻璃体腔硅油填充，使黄斑裂孔、色素上皮、自体巩膜与异体巩膜有足够的组织反应、修复和愈合时间。另外，行后巩膜加固术可以减少玻璃体切除术中冷冻、激光等的使用，减少了对中心视力的损害。  
玻璃体切除术联合后巩膜加固术为内外路联合手术，对术者的技术要求高。对于高度近视眼底病患者实施二者联合手术意义较大。其他非高度近视患者可根据实际情况选择术式。
